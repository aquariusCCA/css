# 第11章 字體屬性

> 返回章節首頁：[README.md](./README.md)
>
> 本章整理 CSS 常見的字體相關屬性：`font-family`、`font-size`、`font-weight`、`font-style` 和 `font` 簡寫。

![字體屬性總結](./images/image_1776223873062_5ff7a5e3.png)

## 導讀
- `font-family` 決定文字使用哪一組字體。
- `font-size` 決定文字大小。
- `font-weight` 決定文字粗細。
- `font-style` 決定文字是否斜體。
- `font` 是把這些設定合併在一起的簡寫。

## 30 秒複習入口
- 字體名稱有空格時，通常要加引號，例如 `"Microsoft YaHei"`
- 多個字體之間用英文逗號分隔，瀏覽器會由左到右依序嘗試
- `font-size` 一定要寫單位，例如 `16px`
- `font-weight` 常用 `400` 和 `700`
- `font-style` 常見值是 `normal` 和 `italic`
- `font` 簡寫必須包含 `font-size` 和 `font-family`

## 正文

### 1. `font-family`
`font-family` 用來定義文字使用的字體系列。
因為不同電腦安裝的字體不一樣，所以通常會寫一串備援字體，讓瀏覽器由左到右依序嘗試。

```css
div {
  font-family: Arial, "Microsoft YaHei", "微软雅黑", sans-serif;
}
```

注意：

- 多個字體之間要用英文逗號分隔
- 字體名稱如果有空格，通常要加引號
- 建議把常見系統字體放在前面，最後再放通用字體系列作備援

### 2. `font-size`
`font-size` 用來設定文字大小。
實際開發中最常用的是 `px`，而且一定要帶單位，否則無效。

```css
body {
  font-size: 16px;
}

/* 標題標籤通常需要另外指定字體大小 */
h2 {
  font-size: 16px;
}
```

補充：

- 瀏覽器預設文字大小通常是 `16px`
- 不同瀏覽器或裝置的預設值可能不完全一致
- 文字大小最好明確指定，不要完全依賴預設值

```html
<h2>pink的秘密</h2>
<p>那一抹众人中最漂亮的颜色,</p>
<p>优雅,淡然,又那么心中清澈.</p>
<p>前端总是伴随着困难和犯错,</p>
<p>静心,坦然,攻克一个又一个.</p>
<p>拼死也要克服它,</p>
<p>这是pink的秘密也是老师最终的嘱托.</p>
```

### 3. `font-weight`
`font-weight` 用來設定字體粗細。
常見值有 `normal`、`bold`、`bolder`、`lighter` 和數字。

```css
font-weight: normal | bold | bolder | lighter | number;
```

實務上最常用的是：

- `400`：正常
- `700`：加粗

```css
.bold {
  /* 700 不要加單位，等價於 bold */
  font-weight: 700;
}

h2 {
  font-weight: 400;
}
```

注意：

- 不是所有字體都支援完整的粗細階級
- 如果字體本身沒有對應字重，頁面效果可能不明顯

![字體粗細](./images/image_1776223873065_dc6e6f17.png)

```html
<h2>pink的秘密</h2>
<p>那一抹众人中最漂亮的颜色,</p>
<p>优雅,淡然,又那么心中清澈.</p>
<p>前端总是伴随着困难和犯错,</p>
<p>静心,坦然,攻克一个又一个.</p>
<p class="bold">拼死也要克服它,</p>
<p>这是pink的秘密也是老师最终的嘱托.</p>
```

### 4. `font-style`
`font-style` 用來設定文字風格，最常見的是控制是否斜體。

```css
p {
  font-style: normal;
}
```

```css
p {
  font-style: italic;
}

em {
  /* 讓斜體標籤恢復正常字形 */
  font-style: normal;
}
```

一般來說，平常不太需要特別替文字加斜體。
但像 `em`、`i` 這類標籤預設會斜體時，常會視需求改回 `normal`。

```html
<p>同学,上课时候的你</p>
<em>下课时候的你</em>
```

![文字樣式](./images/image_1776223873066_4de87ceb.png)

### 5. `font` 簡寫屬性
`font` 可以把多個字體屬性合併成一行，寫法更簡潔。

順序固定為：

```text
font-style font-weight font-size/line-height font-family
```

注意：

- 順序不能亂
- `font-size` 和 `font-family` 不能省略
- 其他屬性可以省略

```css
div {
  /* 想要 div 文字變斜體、加粗、16px、微軟雅黑 */
  font: italic 700 16px "Microsoft YaHei";
}
```

```html
<div>正在努力學習前端知識中 ...</div>
```

## 常見混淆點
- `font-family` 是字體系列，不是單一字體名稱的固定寫法
- `font-weight: 700` 不需要加單位
- `font-style` 主要是斜體，不是字體大小
- `font` 是簡寫，不是所有字體屬性的集合

## 一句話總結
字體相關設定最常用的是 `font-family`、`font-size`、`font-weight`、`font-style`，需要一起寫時就用 `font` 簡寫，但順序不能亂。

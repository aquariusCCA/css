# 第11章 字體屬性

## 章節說明
本章整理 CSS 常見的字體相關屬性：`font-family`、`font-size`、`font-weight`、`font-style` 與 `font` 簡寫。

這一章的重點不是只背語法，而是理解：

- 瀏覽器如何依照字體清單尋找可用字體
- 文字大小如何用 `px`、`rem`、`em`、`%` 等方式設定
- 字重與斜體為什麼有時候看起來沒有明顯變化
- `font` 簡寫哪些值可以省略、哪些值不能省略
- 實務專案中如何設定全站基礎字體

## 建議閱讀順序
1. 先讀「導讀」與「30 秒複習入口」，建立整體印象。
2. 再讀 `font-family`、`font-size`、`font-weight`、`font-style` 四個基本屬性。
3. 最後讀 `font` 簡寫屬性，理解它和前面四個屬性的關係。
4. 回查時優先看「速查區」、「常見錯誤」與「實務推薦寫法」。

## 回查重點
- 想確認字體備援順序時，看 `font-family`。
- 想確認文字大小單位時，看 `font-size`。
- 想確認 `px`、`rem`、`em`、`%` 差異時，看 `font-size` 的補充說明。
- 想確認斜體或字體粗細怎麼寫時，看 `font-style` 與 `font-weight`。
- 想把多個字體屬性合併時，看 `font` 簡寫。
- 想知道專案裡如何設定全站字體時，看「實務推薦寫法」。

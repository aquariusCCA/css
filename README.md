# CSS 筆記總覽

這份倉庫整理的是 CSS 入門學習筆記，內容以基礎觀念、常見書寫習慣與樣式引入方式為主。

目前倉庫已收錄第 1 到第 13 章內容，其中第一章、第三章、第四章、第六章、第七章、第八章、第九章、第十章、第十一章、第十二章與第十三章已有章節入口；第二章與第五章目前以單篇章節筆記呈現，後續仍可沿用同一套整理方式擴充。

## 這份筆記怎麼使用

- 第一次學習時，請依章節順序閱讀，先建立「HTML 管結構、CSS 管樣式」的分工概念，再往後看實作習慣。
- 複習時，優先從每篇筆記開頭的關鍵字與建議回查情境切入，不必每次都重讀全文。
- 如果你是寫到一半卡住，先從最接近問題的章節入口進入，再跳到對應單篇筆記。

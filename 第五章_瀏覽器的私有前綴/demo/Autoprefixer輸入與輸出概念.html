開發者通常先寫標準 CSS

.box {
  display: flex;
  user-select: none;
}

現代專案通常先寫這種可讀性較高的標準語法。

工具可能輸出的結果

.box {
  display: -webkit-box;
  display: -ms-flexbox;
  display: flex;
  -webkit-user-select: none;
  -ms-user-select: none;
  user-select: none;
}
Flex Item A Flex Item B

這個頁面是在示範「輸入與輸出概念」，不是要你手動背完整工具產物。

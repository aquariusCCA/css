# 第五章 瀏覽器的私有前綴

## 章節導讀

這一章整理 CSS 私有前綴的用途、歷史背景，以及它在現代前端開發中的實際定位。

重點不是背所有前綴，而是理解它為什麼出現、什麼情況還會遇到，以及現在多半如何交給工具處理。

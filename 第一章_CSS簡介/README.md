# 第一章 CSS簡介

本章是整份 CSS 筆記的起點，主要建立 HTML 與 CSS 的分工觀念，並先熟悉 CSS 的基本語法與常見程式碼風格。

如果你是第一次接觸 CSS，建議先完整讀過本章；如果你只是忘了基本語法或想快速確認觀念，也可以直接跳到單篇筆記的對應段落回查。

## 本章和整體學習地圖的關係

- 這一章負責建立最基礎的觀念：HTML 負責結構，CSS 負責樣式。
- 讀完這一章後，再進入屬性書寫順序或樣式引入方式會比較不容易混淆。
- 這一章適合作為後續所有 CSS 學習內容的共同前置知識。

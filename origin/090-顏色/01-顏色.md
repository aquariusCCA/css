# **3. 顏色值**

- 顏色值有三種表示方式：
    1. HEX 值
    2. RGB 值
    3. HSL 值

## **3.1 HEX值**

> 用 HEX 值的特定格式來表示顏色值是顏色定義中最常見的方法了。

- HEX 值由符號 # 和十六進制組合而成，其表示形式為：`#RRGGBB`，其中，R表示紅色，G表示綠色，B表示藍色，其值介於 00~ff 之間，用於指定顏色的強度。
    
    ![HEX值.png](./assets/images/colors-img-003-8da288.png)
    

<aside>
⚠️

**注意，當 #RRGGBB 滿足兩個 R、兩個 G 和兩個 B 同時相等時，可以簡寫為 #RGB，比如 #aabbcc 可以簡寫為 #abc。**

</aside>

## **3.2 RGB值**

> 在 CSS 中，也可以通過 RGB 值來指定顏色。

- 其中 RGB 分別表示紅綠藍，書寫格式為：`rgb(141, 0, 0)`，括號裡用三個數值分別表示種顏色的強度，並且數值的範圍是：0~255
    
    ![RGB值1.png](./assets/images/colors-img-004-09f9b5.png)
    
- 另外，有一個知識點是透明度。也就是可以為 RGB 值添加 alpha 值用於定義透明度，具體做法為，在 `rgb(0, 0, 0)` 中添加第四個值，範圍是 0~1，其中 0 是最小值，表示完全透明，1 是最大值，表示不透明。
    
    ![RGB值2.png](./assets/images/colors-img-005-8c78cc.png)
    

## **3.3 HSL值**

> 👉 在 CSS 中，可以使用 HSL 值來表示顏色值，這個顏色表示與前兩種不怎麼像，HSL 值分別表示色調、飽和度和亮度。

### **3.3.1 色調（hue）**

> 色調是色輪的度數，其值的範圍是 0 到 360，其中 0 表示紅色，120 表示綠色，240 表示藍色。
> 
> ![色調示意圖1.png](./assets/images/colors-img-006-2233e1.png)
>
> ![色調示意圖2.png](./assets/images/colors-img-007-c8a93b.png)
> 

### **3.3.2 飽和度（saturation）**

> 飽和度，表示色彩的純度。飽和值越高，色彩越純，越低則色彩越灰。並且注意到，飽和值是用百分號來表示，0% 表示灰色陰影，100% 表示全色。
> 
> ![飽和度示意圖.png](./assets/images/colors-img-008-f53968.png)

### **3.3.3 亮度（lightness）**

> 亮度也可以稱明亮度。值越高越亮，也就越接近白色。值越小就越暗，越接近黑色，跟飽和度一樣，亮度值也是用百分比來表示。其中 0% 是黑色，50％ 表示不明不暗，100％是白色。
> 
> ![亮度示意圖1.png](./assets/images/colors-img-009-68b12e.png)
> 
> ![亮度示意圖2.png](./assets/images/colors-img-010-4517be.png)

- 和 RGB 值一樣，HSL 值也可以添加 alpha 值來指定顏色的不透明度。其語法為：`hsla(0, 0, 0, 0.0)`，最後一個值表示 alpha，取值範圍也是 0.0 到 1.0。
    
    ![亮度示意圖3.png](./assets/images/colors-img-011-3d4910.png)

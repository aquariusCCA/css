💡

**從後向前匹配，一旦匹配成功則終止。**

</aside>

- **示例 1**
    
    ```html
    <!DOCTYPE html>
    <html lang="en">
    
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Document</title>
      <style>
        /* 规则1 */
        @media (min-width: 320px) {
          html {
            background: red;
          }
        }
    
        /* 规则2 */
        @media (min-width: 800px) {
          html {
            background: green;
          }
        }
    
        /* 规则3 */
        @media (min-width: 1024px) {
          html {
            background: blue;
          }
        }
      </style>
    </head>
    
    <body>
    
    </body>
    
    </html>
    ```

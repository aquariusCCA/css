# and

> 可以將多個媒體特性連接到一起，相當於且的意思
> 
- 可以使用關鍵詞 "and" 將多個媒體特性結合在一起。也就是說，一個媒體查詢中可以包含 0 到多個表達式，表達式又可以包含 0 到多個關鍵字，以及一種媒體類型。
- **範例程式碼 1**
    
    ```html
    <!DOCTYPE html>
    <html lang="en">
    
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Document</title>
      <style>
        * {
          margin: 0;
          padding: 0;
        }
    
        h1 {
          height: 200px;
          background-color: gray;
          text-align: center;
          line-height: 200px;
          font-size: 100px;
        }
    
        /* 且運算 */
        @media (min-width: 700px) and (max-width: 800px) {
          h1 {
            background-color: orange;
          }
        }
      </style>
    </head>
    
    <body>
      <h1>你好啊!!</h1>
    </body>
    
    </html>
    ```
    
- **範例程式碼 2**
    
    ```html
    <!DOCTYPE html>
    <html lang="en">
    
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Document</title>
      <style>
        * {

        }

          height: 200px;
          background-color: gray;
          text-align: center;
          line-height: 200px;
          font-size: 100px;
        }
    
        /* 且運算 */
        @media screen and (min-width: 700px) and (max-width: 800px) {
          h1 {
            background-color: orange;
          }
        }
      </style>
    </head>
    
    <body>
      <h1>你好啊!!</h1>
    </body>
    
    </html>
    ```
    

# **or**

> 對於所有的連接選項只要匹配成功一個就能應用規則
> 
- 將多個媒體查詢以逗號分隔放在一起；只要其中任何一個為真，整個媒體語句就返回真，相當於 or 操作符。
- **範例程式碼 1**
    
    ```html

    <html lang="en">

      <meta charset="UTF-8">

        h1 {
          height: 200px;
          background-color: gray;
          text-align: center;
          line-height: 200px;
          font-size: 100px;
        }
    
        /* 或運算 */
        @media (max-width: 700px),
        (min-width: 800px) {
          h1 {
            background-color: orange;
          }
        }
      </style>
    </head>

    </html>
    ```

# **not**

> 排除某個媒體類型或媒體條件，相當於非的意思。
> 
- not 操作符用來對媒體查詢或媒體條件的結果進行取反。

<aside>
⚠️

**注意 !!! `@media not screen` 這類寫法會否定整個媒體查詢；Media Queries Level 4 之後，也可以用 `not (條件)` 否定括號內的媒體條件。**

</aside>

```html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    * {
      margin: 0;
      padding: 0;
    }

    h1 {
      height: 200px;
      background-color: gray;
      text-align: center;
      line-height: 200px;
      font-size: 100px;
    }

    /* 否定運算符 */
    @media not screen {
      h1 {
        background-color: orange;
      }
    }
  </style>
</head>

<body>
  <h1>你好啊!!</h1>
</body>

</html>

```

現代條件語法也可以用來否定括號內的媒體條件：

```css
@media not (hover) {
  h1 {
    background-color: orange;
  }
}
```

# **only**

- only 操作符表示僅在媒體查詢匹配成功時應用指定樣式。
- 💡 **主要可以通過它讓選中的樣式在老式瀏覽器中不被應用。**

```css
/* 肯定運算符 */
@media only screen and (width: 800px){
  h1 {
    background-color: orange;
  }
}
```

上面這行代碼，在老式瀏覽器中被解析為 `media="only"`，因為沒有一個叫 only 的設備，所以實際上老式瀏覽器不會應用樣式。

```css
@media screen and (width: 800px){
  h1 {
    background-color: orange;
  }
}
```

上面這行代碼，在老式瀏覽器中被解析為 `media="screen"`，它把後面的邏輯表達式忽略了。所以老式瀏覽器會應用樣式。所以，在使用媒體查詢時，only 最好不要忽略。

> ✍️ CSS3 允許元素設置多個背景圖片。

```css
.box {
  width: 2500px;
  height: 2500px;
  border: 1px solid black;
  background: url(./images/bg.jpg) no-repeat,
              url(./images/ldh.jpg) no-repeat right top,
              url(./images/logo.png) no-repeat left bottom;
}
```

```html
<div class="box"></div>
```

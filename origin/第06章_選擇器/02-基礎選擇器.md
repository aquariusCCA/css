# **標籤選擇器**

> 是指用 HTML 標籤名稱作為選擇器，按標籤名稱分類，為頁面中某一類標籤指定統一的 CSS 樣式。
> 
- 作用: 標籤選擇器可以把某一類標籤全部選出來，比如 : 所有的 `<div>` 標籤和所有的 `<span>` 標籤。
- 優點: 能快速的為頁面中同類型的標籤統一設置樣式。
- 缺點: 不能設計差異化樣式，只能選擇全部的當前標籤。

```css
标签名 {
    属性名：属性值;
}
```

```css
p {
	color: green;
}

div {
	color: pink;
}
```

```html
<p>男生</p>
<p>男生</p>
<p>男生</p>
<div>女生</div>
<div>女生</div>
<div>女生</div>
```

# **類選擇器**

> 如果想要差異化選擇不同的標籤，單獨選一個或者某幾個標籤，可以使用類選擇器。
> 
- ✍️ 合法的類名：數字、字母、下劃線、中劃線
- 類選擇器在 HTML 中以 class 屬性表示，在 CSS 中，類選擇器以一個點 `.` 號顯示。

<aside>
⚠️

**注意點**

- 類選擇器使用`.` 進行標識，後續緊跟類名。
- 可以理解為給這個標籤起一個名字來表示。
- 長名稱或詞組可以使用中橫線來為選擇器命名。
- 不要使用純數字、中文等命名，盡量使用英文字母表示。
- 💡 命名要有意義，盡量使別人一眼就知道這個類名的目的。
</aside>

```css
.类名{
    属性名：属性值;
		...
}
```

```css
.red {
	 color: red;
}
.star-sing {
	color: green;
}
.memeda {
	color: pink;
}
```

```html
<ul>
  <li class="red">冰雨</li>
  <li class="red">来生缘</li>
  <li>李香兰</li>
  <li class="memeda">生僻字</li>
  <li class="star-sing">江南style</li>
</ul>
<div class="red">我也想变红色</div>
```

### **多類名**

> 我們可以給一個標籤指定多個類名，從而達到更多的選擇目的，這些類名都可以選出這個標籤，簡單理解就是一個標籤有多個名字。
> 
- 一個元素可以有多個類名，空格隔開。

<aside>
👉

**使用場景**

- 可以把一些標籤元素相同的樣式放到一個類裡面。
- 這些標籤都可以調用這個公共的類，然後再調用自己獨有的類。
- 從而節省 CSS 代碼，統一修改也非常方便。
</aside>

<aside>
⚠️

**注意點**

- 在標籤 class 屬性中寫多個類名。
- 多個類名中間必須用空格隔開。
- 這個標籤就可以分別具有這些類名的樣式。
</aside>

```css
.red {
  color: red;
}

.font35 {
  font-size: 35px;
}
```

```html
<div class="red font35">Kevin</div>
```

# **id選擇器**

> id 選擇器可以為標有特定 id 的 HTML 元素指定特定的樣式。
> 
- ✍️ HTML 元素以 id 屬性來設置 id 選擇器，CSS 中 id 選擇器以 `#` 來定義。
- 💡 id 屬性只能在每個 HTML 文檔中出現一次。

<aside>
👉

**id 選擇器和類選擇器的區別 ?**

- 類選擇器好比人的名字，一個人可以有多個名字，同時一個名字也可以被多個人使用。
- id 選擇器好比人的身分證號，全國唯一的、不重複的。
- id 選擇器和類選擇器最大的不同在於使用次數上。
- 類選擇器在修改樣式中用的最多，`id 選擇器一般用於頁面唯一性的元素上`，經常和 JavaScript 搭配使用。
</aside>

```css
#pink {
  color: pink;
}
```

```html
<div id="pink">蕭士麟</div>
```

# **通配符選擇器**

> 在 CSS 中，通配符選擇器使用 * 定義，它表示 選取頁面中所有元素 ( 標籤 )。
> 
- 通配符選擇器不需要調用，自動就給所有的元素使用樣式。

```css
* {
    属性1：属性值1;
    ....
}
```

```css
* {
	color: red;
}
```

```html
<div>我的</div>
<span>我的</span>
<ul>
  <li>还是我的</li>
</ul>
```

- **一般用於統一設置頁面樣式，如: 清除所有的元素標籤的內外邊距。**
    
    ```css
    /* 清除内外边距 */
    * {
      margin: 0;
      padding: 0;
      box-sizing: border-box;
    }
    ```

> 網頁元素很多都帶有默認的內外邊距，而且不同瀏覽器默認的也不一致。因此我們在佈局前，首先要清除下網頁元素的內外邊距。

<aside>
⚠️

**注意：行內元素為了照顧兼容性，盡量只設置左右內外邊距，不要設置上下內外邊距。但是轉換為塊級和行內塊元素就可以了。**

</aside>

- 京東做法
    
    ```css
    * {
      margin: 0;
      padding: 0;
    }
    ```
    
- 淘寶
    
    ```css
    blockquote,
    body,
    button,
    dd,
    dl,
    dt,
    fieldset,
    form,
    h1,
    h2,
    h3,
    h4,
    h5,
    h6,
    hr,
    input,
    legend,
    li,
    ol,
    p,
    pre,
    td,
    textarea,
    th,
    ul {
      margin: 0;
      padding: 0;
    }
    ```

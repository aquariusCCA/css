**外邊距可以讓塊級盒子水平居中，但是必須滿足兩個條件:**

- 盒子必須制定了寬度(width)。
    - 因為如果大盒子沒有設置寬度，此時會默認佔滿父元素的寬度。
- 盒子左右的外邊距都設置為 auto，左右的外邊距都設置為`auto` 有三種寫法 (如下) :
    
    ```css
    1.
    margin-left: auto;
    margin-right: auto;
    
    2.
    margin: auto;
    
    3.
    margin: 0 auto;
    ```
    
</aside>

```css
.header {
	width: 900px;
	height: 200px;
	background-color: pink;
	margin: 100px auto;
}
```

```html
<div class="header"></div>
```

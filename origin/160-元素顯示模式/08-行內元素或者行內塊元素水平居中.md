> 行內元素或者行內塊元素水平居中給其父元素添加 text-align: center 即可。
> 

```css
.header {
	width: 900px;
	height: 200px;
	background-color: pink;
	margin: 100px auto;
	text-align: center;
}
```

```html
<div class="header">
    <span>里面的文字</span>
</div>

<div class="header">
    <img src="./images/down.jpg" alt="">
</div>
```

> 常見的行內元素有<a>、<strong>、<b>、<em>、<i>、<del>、<s>、<ins>、<u>、<span>等，其中<span>標籤是最典型的行內元素，有的地方也將行內元素稱為內聯元素。
> 

<aside>
💡

**行內元素的特點**

- 相鄰行內元素在一行上，一行可以顯示多個。
- **高，寬直接設置是無效的**。
- 寬度和高度默認由內容撐開。
- 行內元素只能容納文本或其他行內元素。
</aside>

```css
span {
  background-color: hotpink;
}
```

```html
<span>pink老师你怎么穿着品如的衣服呢</span>
<strong>品如的衣服</strong>
<span>pink老师</span>
<strong>品如的衣服</strong>
```

# **特殊情況:超鏈接**

- 鏈接裡面不能再放鏈接。
    
    ```html
    <a href="http://www.baidu.com">
      <a href="#">空鏈接</a>
    </a>
    ```
    
- 特殊情況鏈接`<a>`裡面可以放塊級元素，但是給`<a>`轉換一下塊級模式最安全。
    
    ```css
    a {
      display: block;
    }
    ```
    
    ```html
    <a class="a" href="">
    	<div></div>
    </a>
    ```

> 原因: 行內元素、行內塊元素彼此之間的換行會被瀏覽器解析為一個空白字符。
> 
> - 💡 解決方案 → 給父元素設置 `font-size: 0`; 再給需要顯示文字的元素單獨設置字體大小。

```css
div {
  width: 400px;
  height: 400px;
  background-color: gray;
  font-size: 0;
}

.s1 {
  background-color: red;
  font-size: 16px;
}

.s2 {
  background-color: green;
  font-size: 16px;
}

.s3 {
  background-color: blue;
  font-size: 16px;
}
```

```html
<div>
  <span class="s1">xxx</span>
  <span class="s2">aaa</span>
  <span class="s3">zzz</span>
</div>
```

- 轉換`transform` : 我們簡單理解就是變形，有 2D 和 3D 之分。
- 我們暫且學了三個分別是`位移`、`旋轉`、`縮放`。
- 2D 移動 `translate(x, y)` 最大的優勢是不影響其他盒子， 裡面參數用`%`，是相對於自身寬度和高度來計算的。
- 可以分開寫比如 `translateX(x)` 和 `translateY(y)`。
- 2D 旋轉 `rotate` (度數) 可以實現旋轉元素，度數的單位是`deg`。
- 2D 縮放 `sacle(x, y)` 裡面參數是數字 不跟單位 可以是小數 最大的優勢 不影響其他盒子。
- 設置轉換中心點 `transform-origin : x y;` 參數可以百分比、像素或者是方位名詞。
- 同時使用多個轉換，其格式為: `transform:translate() rotate() scale()` 移動-旋轉-縮放。

- 絕對定位、固定定位會壓住下面標準流所有的內容
    
    ```css
    .box {
    	/* 1. 浮動的元素不會壓住下面標準流的文字 */
    	/* float: left; */
    	
    	/* 2. 絕對定位（固定定位） 會壓住下面標準流所有的內容。 */
    	position: absolute;
      width: 150px;
      height: 150px;
      background-color: pink;
    }
    ```
    
    ```html
    <body>
        <div class="box"></div>
        <p>阁下何不同风起，扶摇直上九万里</p>
    </body>
    ```
    

<aside>
💡

**浮動之所以不會壓住文字，因為浮動產生的目的最初是為了做文字環繞效果的。文字會圍繞浮動元素。**

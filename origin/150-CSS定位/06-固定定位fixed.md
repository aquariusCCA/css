> **固定定位是元素固定於瀏覽器的可視區的位置。**
> 
> - 👉 主要使用場景： 可以在瀏覽器頁面滾動時元素的位置不會改變。

<aside>
💡

**固定定位的參考點在哪裡 ?**

- 以瀏覽器的可視窗口為參照點移動元素。
- 參考它的視口
- 什麼是視口 → 對於 PC 瀏覽器來說，視口就是我們看網頁的那扇 "窗戶"。
</aside>

<aside>
💡

**特點:**

- 跟父元素沒有任何關係。
- 不隨滾動條滾動。
- 固定定位不再佔有原先的位置 ( 脫標 )；脫離文檔流會對後面的兄弟元素、父元素有影響。
- left 不能和 right 一起設置，top 和 bottom 不能一起設置。
- 固定定位、浮動不能同時設置，如果同時設置，浮動失效，以定位為主。
- 固定定位的元素，也能通過 margin 調整位置，但不推薦這樣做。
- 無論是什麼元素 (行內、行內塊、塊級) 設置為固定定位之後，都變成了定位元素。
    - 何謂定位元素 → 默認寬、高都被內容所撐開，且能自由設置寬高。
- 固定定位也是脫標的，其實固定定位也可以看做是一種特殊的絕對定位。
</aside>

```css
选择器 {
	position: fixed;
}
```

# **固定定位小技巧**

> 固定定位小技巧： 固定在版心右側位置。
> 
- 小算法 :
    - 讓固定定位的盒子 `left: 50%`. 走到瀏覽器可視區（也可以看做版心） 的一半位置。
    - 讓固定定位的盒子 `margin-left` 版心寬度的一半距離。多走版心寬度的一半位置。

```css
.w {
  width: 800px;
  height: 1400px;
  background-color: pink;
  margin: 0 auto;
}

.fixed {
  position: fixed;
	/* 1. 走浏览器宽度的一半 */
	left: 50%;

	/* 2. 利用margin 走版心盒子宽度的一半距离 */
	margin-left: 400px;

  width: 50px;
  height: 150px;
  background-color: skyblue;
}
```

```html
<body>
    <div class="fixed"></div>
    <div class="w">版心盒子 800像素</div>
</body>
```

# **字體系列font-family**

> CSS 使用 font-family 屬性定義文本的字體系列。

<aside>
⚠️

**注意點 :**

- 各種字體之間必須使用英文狀態下的逗號隔開。
- 一般情況下，如果有空格隔開的多個單詞組成的字體，加引號。
- 盡量使用系統默認自帶自體，保證在任何用戶的瀏覽器中都能正確顯示。
</aside>

<aside>
💡

**渲染規則 :**

- 從左往右按照順序查找，如果電腦中未安裝該字體，則顯示下一個字體。
- 如果都不支持，此時會根據操作系統，顯示最後字體系列的默認字體。
</aside>

```css
div {
	font-family: Arial, "Microsoft Yahe", "微软雅黑";
}
```

# **字體大小font-size**

> CSS 使用 font-size 屬性定義字體大小。

<aside>
💡

**重點 :**

- 單位需要設置，否則無效。
- px ( 象素 ) 大小是我們網頁的最常用的單位。
- 谷歌瀏覽器默認的文字大小為 16px。
- 不同瀏覽器可能默認顯示的字號大小不一致，我們盡量給一個明確值大小，不要使用默認大小。
</aside>

```css
body {
  font-size: 16px;
}

/* 標題標籤比較特殊,需要單獨指定文字大小 */
h2 {
  font-size: 16px;
}
```

```html
<h2>pink的秘密</h2>
<p>那一抹众人中最漂亮的颜色,</p>
<p>优雅,淡然,又那么心中清澈.</p>
<p>前端总是伴随着困难和犯错,</p>
<p>静心,坦然,攻克一个又一个.</p>
<p>拼死也要克服它,</p>
<p>这是pink的秘密也是老师最终的嘱托.</p>

```

- 可以給 body 指定整個頁面文字的大小。
    
    ```css
    body {
    	font-size: 16px;
    }
    ```
    

# **字體粗細font-weight**

> CSS 使用 font-weight 屬性設置文本字體的粗細。
> 
> ![字體粗細.png](./assets/images/font-properties-img-002-261a82.png)

```css
font-weight:  normal | bold |  bolder  | lighter | number
```

<aside>
💡

**重點 :**

- 不是所有字體都提供了九種粗細，因此部分取值頁面中無變化。
- 實際開發中以 400(正常)、700(加粗) 兩種取值最常使用。
- 實際開發時，我們更喜歡用數字表示粗細。
</aside>

```css
.bold {
	/* 這個 700 的後面不要跟單位，等價於 bold 都是加粗的效果 */
	/* 實際開發中，我們跟提倡使用數字，表示加粗或者變細 */
	font-weight: 700;
}

h2 {
  font-weight: 400;
}
```

```html
<h2>pink的秘密</h2>
<p>那一抹众人中最漂亮的颜色,</p>
<p>优雅,淡然,又那么心中清澈.</p>
<p>前端总是伴随着困难和犯错,</p>
<p>静心,坦然,攻克一个又一个.</p>
<p class="bold">拼死也要克服它,</p>
<p>这是pink的秘密也是老师最终的嘱托.</p>

```

# **4. 文字樣式font-style**

> CSS 使用 font-style 屬性設置文本的風格。
> 
> ![文字樣式.png](./assets/images/font-properties-img-003-7bf350.png)
> 
> - 平時我們很少給文字加斜體，反而要給斜體標籤 ( em、i ) 改為不傾斜字體。

```css
p {
	font-style: normal;
}
```

```css
p {
  font-style: italic;
}

em {
	/* 讓傾斜的字體不傾斜   就是趕緊脈動回來 */
	font-style: normal;
}
```

```html
<p>同学,上课时候的你</p>
<em>下课时候的你</em>
```

# **字體複合屬性**

> 字體屬性可以把以上文字樣式綜合來寫，這樣可以更節約代碼。

- 💡 順序：文字樣式 文字粗細 文字大小/行高 文字字體。

<aside>
⚠️

**注意點 :**

- 使用 font 屬性時，必須按照上面語法格式中的順序書寫，不能更換順序，並且個個屬性間以空格隔開。
- 不需要設置的屬性可以省略，但必須保留 `font-size` 和 `font-family` 屬性，否則 font 屬性將不起作用。
    - 只能省略前兩個，如果省略了相當設置了默認值。
</aside>

```css
div {
  /* 想要 div 文字變傾斜 加粗 字號設置為16像素 並且是微軟雅黑 */
  /* 複合屬性：簡寫的方式  節約代碼 */
  /* 顺序：font-style font-weight  font-size/line-height  font-family */
  font: italic 700 16px 'Microsoft yahe';
}
```

```html
<div>正在努力學習前端知識中 ... </div>
```

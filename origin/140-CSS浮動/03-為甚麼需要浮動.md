- 浮動最典型的應用：可以讓多個塊級元素一行內排列顯示。
- 網頁佈局第一準則：多個塊級元素縱向排列找標準流，多個塊級元素橫向排列找浮動。
- 總結： 有很多的佈局效果，標準流沒有辦法完成，此時就可以利用浮動完成佈局。因為浮動可以改變元素標籤默認的排列方式。

# **提問：如何讓多個塊級盒子(div)水平排列成一行？**

![三個 div 透過浮動排列在同一行的示意圖](./assets/images/why-do-we-need-float-img-001-432417.png)

> 答：比較難，雖然轉換為行內塊元素可以實現一行顯示，但是他們之間會有大的空白縫隙，很難控制。

```css
div {
  display: inline-block;
	/* float: left; */
	width: 150px;
  height: 200px;
  background-color: pink;
}
```

```html
<div>1</div>
<div>2</div>
<div>3</div>
```

# **提問：如何實現兩個盒子的左右對齊？**

![左浮動和右浮動讓兩個盒子分別貼齊父盒子兩側的示意圖](./assets/images/why-do-we-need-float-img-002-7020ef.png)

```css
.left,
.right {
  width: 200px;
  height: 200px;
  background-color: pink;
}

.left {
  float: left;
}

.right {
  float: right;
}
```

```html
<div class="left">左青龙</div>
<div class="right">右白虎</div>
```

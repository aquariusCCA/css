
> 在 CSS3 中，我們可以使用 text-shadow 屬性將陰影應用於文本。
> 
> ![文字陰影.png](%E6%96%87%E5%AD%97%E9%99%B0%E5%BD%B1text-shadow/%25E6%2596%2587%25E5%25AD%2597%25E9%2599%25B0%25E5%25BD%25B1.png)

```css
div {
	font-size: 50px;
	color: orangered;
	font-weight: 700;
	text-shadow: 5px 5px 6px rgba(0, 0, 0, .3);
}
```

```html
<div>
    你是阴影,我是火影
</div>
```

# **1. HTML的侷限性**

說起 HTML，這其實是個非常單純的傢伙，它只關注內容的語意，比如 `<h1>` 表明這是一個大標題，`<p>` 表明這是一個段落。

很早的時候，世界上的網站雖然很多，但是他們都有一個共同的特點 → 醜。

雖然 HTML 可以做簡單的樣式，但是帶來的是無盡的臃腫和繁瑣...

# **2. CSS網頁的美容師**

> 💡 CSS 也是一種標記語言。
> 
- CSS 是層疊樣式表的簡稱，有時我們也會稱為 CSS 樣式表或級聯樣式表。
- CSS 主要用於設置 HTML 頁面中的文本內容、圖片的外型以及版面的布局和外觀顯示樣式。
- CSS 讓我們的網頁更加豐富多彩，布局更加靈活；簡單理解 CSS 可以美化 HTML，讓 HTML 更漂亮，讓布局更簡單。

# **3. CSS語法規範**

- 使用 HTML 時，需要遵守一定的規範，CSS 也是如此，要想熟練地使用 CSS 對網頁進行修飾，首先需要了解 CSS 的樣式規則。
- CSS 規則由兩個主要的部份構成，選擇器以及一條或多條聲明
    
    ![CSS語法規範.png](./assets/images/css-introduction-img-001-3ac98e.png)
    

```css
选择器 {
    属性名: 属性值
		...
}
```

- 重點如下 :
    - 屬性和屬性值以 "鍵值對" 的形式出現。
    - 屬性是對指定的對象設置的樣式屬性，例如 : 字體大小、文本顏色等。
    - 屬性和屬性值之間用英文 **:** 分開。
    - 多個 "鍵值對" 之間用英文 `;` 進行區分。

# **4. CSS代碼風格**

> 💡 項目上線時，我們會通過工具將展開風格的代碼，變成緊湊風格，這樣可以減小文件體積，節約網路流量，同時也能讓用戶打開網頁時速度更快。
> 
- 展開風格 → 開發時推薦，便於維護和測試。
    
    ```css
    h1 {
    	color: red;
    	font-size: 40px;
    }
    ```
    
- 緊湊風格 → 項目上線時推薦，可減小文件體積。
    
    ```css
    h1{color:red;font-size:40px;}
    ```
    

# **5. 總結**

- HTML 主要做結構，顯示元素內容。
- CSS 美化 HTML ，布局網頁。
- CSS 最大價值 : 由 HTML 專注去做結構呈現，樣式交給 CSS ，即結構與樣式分離。

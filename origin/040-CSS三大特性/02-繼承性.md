> **有些樣式會繼承，元素如果本身設置了某個樣式，就使用本身設置的樣式；但如果本身沒有設置某個樣式，會從父元素開始一級一級的繼承。**
> 
> - 優先繼承離得近的祖先元素。

- 會繼承的 CSS 屬性
    - 字體屬性、文本屬性 (除了 vertical-align )、文字顏色 …. 等。

- 不會繼承的 CSS 屬性
    - 邊框、背景、內邊距、外邊距、寬高、溢出方式 …. 等。

- CSS 中的繼承: 子標籤會繼承父標籤的某些樣式，如文本顏色和字號，恰當的使用繼承可以簡化代碼，降低 CSS 樣式的複雜性。

<aside>
💡

一個規律  ⇒  能繼承的屬性，都是不影響布局的，簡單的說都是和盒子模型沒有關係的。

</aside>

```css
body {
	/* 1.5 的意思：讓裡面所有的子元素的行高是當前文字大小的 1.5 倍。 */
	font: 12px/1.5   Microsoft YaHei
}
```

# **子元素可以繼承父元素的樣式**

- text- 這些開頭的可以繼承。
- font- 這些開頭的可以繼承。
- line- 這些開頭的可以繼承。
- color。

```css
body {
  font: 12px/1.5 Microsoft YaHei
}

div {
  color: pink;
  font-size: 50px;
}
```

```html
<div>
  <p>龙生龙，凤生凤，老鼠生的孩子会打洞</p>
</div>
```

# **行高的繼承**

```css
body {
  color: pink;
  /*
    行高可以跟單位也可以不跟單位
    沒有單位: font: 12px/1.5 'Microsoft YaHei';
    有單位: font: 12px/24px 'Microsoft YaHei';
  */
  font: 12px/1.5 'Microsoft YaHei';
}

div {
  /* 子元素繼承了父元素 body 的行高 1.5 */
  /* 這個 1.5 就是當前元素文字大小 font-size 的 1.5 倍   所以當前 div 的行高就是 21 像素 */
  font-size: 14px;
}

p {
  /* 1.5 * 16 =  24 当前的行高 */
  font-size: 16px;
}

/*
  li 沒有手動指定文字大小  則會繼承父親的 文字大小  body 12px 所以 li 的文字大小為 12px
  當前 li 的行高就是  12 * 1.5  =  18
*/
```

```html
<div>粉红色的回忆</div>
<p>粉红色的回忆</p>
<ul>
    <li>我没有指定文字大小</li>
</ul>
```

# **繼承失效的特殊情況**

![CSS 繼承失效示意圖，標示 a 標籤 color 與 h 系列 font-size 不繼承父層](./assets/images/css-inheritance-img-001-c2fc84.png)

- a 標籤的 **color** 會繼承失效。
- h 系列標籤的 **font-size** 會繼承失效。

```css
div {
  color: red;
  font-size: 12px;
}
```

```html
<div>
  <a href="#">超鏈接</a>
  <h1>一級標題</h1>
</div>
```

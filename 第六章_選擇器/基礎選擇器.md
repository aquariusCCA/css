## 本節導讀

基礎選擇器是最常見、最基礎的 CSS 選擇器。  
它們的共同點是：都只靠單一條件來選出元素，適合用來建立最基本的樣式規則。

## 關鍵字

- 基礎選擇器

- 類選擇器
- id 選擇器
- 通配符選擇器
- class
- selector

## 什麼是基礎選擇器

基礎選擇器是由單個選擇器組成的。

常見的基礎選擇器包括：

- 標籤選擇器
- 類選擇器
- id 選擇器
- 通配符選擇器

## 標籤選擇器

標籤選擇器是用 HTML 標籤名稱作為選擇器，根據標籤名稱統一指定樣式。

### 作用

- 可以一次選中同一種標籤
- 適合替頁面中同類元素設定共通樣式

### 優點

- 能快速統一同類標籤的樣式

### 缺點

- 不能做細部差異化設定
- 一旦選中某種標籤，會影響同類型的所有元素

### 寫法

```css
標籤名 {
  屬性名: 屬性值;
}
```

### 例子

```css
p {
  color: green;
}

div {
  color: pink;
}
```

```html
<p>男生</p>
<p>男生</p>
<p>男生</p>
<div>女生</div>
<div>女生</div>
<div>女生</div>
```

## 類選擇器

如果想針對某一類元素做差異化選擇，可以使用類選擇器。

類選擇器在 HTML 中用 `class` 屬性表示，在 CSS 中用 `.` 來表示。

### 作用

- 可以選出某一個元素
- 也可以選出某幾個元素
- 適合做重複使用的樣式分類

### 命名注意

- 類名可以由數字、字母、下劃線、中劃線組成
- 長名稱或詞組可以用中橫線連接
- 不要用純數字或中文命名
- 類名最好有意義，看到就能理解用途

### 寫法

```css
.類名 {
  屬性名: 屬性值;
}
```

### 例子

```css
.red {
  color: red;
}

.star-sing {
  color: green;
}

.memeda {
  color: pink;
}
```

```html
<ul>
  <li class="red">冰雨</li>
  <li class="red">來生緣</li>
  <li>李香蘭</li>
  <li class="memeda">生僻字</li>
  <li class="star-sing">江南style</li>
</ul>

<div class="red">我也想變紅色</div>
```

## 多類名

一個標籤可以同時擁有多個類名。

### 作用

- 把共通樣式抽成一個類
- 再搭配元素自己的類名
- 方便重用，也方便統一修改

### 注意點

- 多個類名寫在同一個 `class` 屬性中
- 類名之間要用空格隔開
- 這個元素會同時套用這些類名對應的樣式

### 例子

```css
.red {
  color: red;
}

.font35 {
  font-size: 35px;
}
```

```html
<div class="red font35">Kevin</div>
```

## id 選擇器

id 選擇器是用 HTML 元素的 `id` 屬性來指定樣式。

### 作用

- 為特定的唯一元素指定樣式
- 常用在頁面中只會出現一次的元素上

### 命名與使用

- HTML 用 `id` 屬性設定
- CSS 中以 `#` 表示
- 一個 id 在同一份 HTML 文件中應該只出現一次

### 和類選擇器的差別

- 類選擇器像名字，一個元素可以有多個類名
- id 選擇器像身分證，一個頁面中應該唯一
- 類選擇器通常更常用
- id 選擇器常和 JavaScript 搭配使用

### 例子

```css
#pink {
  color: pink;
}
```

```html
<div id="pink">蕭士麟</div>
```

## 通配符選擇器

通配符選擇器使用 `*` 表示，會選中頁面中的所有元素。

### 作用

- 套用到所有標籤
- 不需要手動逐一呼叫

### 寫法

```css
* {
  屬性1: 屬性值1;
}
```

### 例子

```css
* {
  color: red;
}
```

```html
<div>我的</div>
<span>我的</span>
<ul>
  <li>還是我的</li>
</ul>
```

### 常見用途

- 統一設定頁面樣式
- 清除元素預設的內外邊距

```css
/* 清除內外邊距 */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
```

## 一句話總結

基礎選擇器就是用最單純的方式把元素選出來，再對它們套用樣式。

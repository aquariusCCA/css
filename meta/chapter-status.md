# CSS 筆記包章節處理狀態表

> 用途：追蹤 CSS 筆記章節從 `origin` 到 `atomic`、`notes`、`demos`、`practice`、`review`、`appendix` 的處理進度。

## 前提

- 目前此筆記包仍在從既有筆記包骨架清理為 CSS 筆記包。
- CSS 原始筆記會由人工整理後放入 `origin/`。
- 本檔案不預先建立 CSS 章節大綱。
- 章節狀態需等 `origin/` 內容確認後再新增。

## 狀態選項

| 狀態 | 說明 |
| --- | --- |
| 未開始 | 尚未處理 |
| 進行中 | 正在處理 |
| 待確認 | 需要人工確認或決策 |
| 已完成 | 已完成並可進入下一階段 |
| 暫停 | 暫時停止處理 |
| 不適用 | 該章節不需要此步驟 |

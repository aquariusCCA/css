# Demo 目錄索引

本目錄用來集中放置第 13 章「背景屬性」各小節的 demo。

## 命名規則

```text
demo/{css-property-name}/index.html
demo/{css-property-name}/style.css
demo/{css-property-name}/README.md
```

## 未來可新增

```text
demo/background-image/
demo/background-repeat/
demo/background-position/
demo/background-size/
demo/background-attachment/
demo/background-origin/
demo/background-clip/
```

每一個主題都可以保留自己的 `index.html`、`style.css`、`README.md`，避免所有 demo 檔案混在同一層。

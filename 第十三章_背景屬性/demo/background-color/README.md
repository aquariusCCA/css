# background-color demo 使用說明

## 如何開啟

直接用瀏覽器開啟：

```text
demo/background-color/index.html
```

不需要安裝 Node、不需要啟動 Vite，也不需要額外套件。

## Demo 包含內容

1. 預設透明背景 vs 指定背景色
2. 背景色與盒模型範圍
3. 半透明背景 vs `opacity`
4. 常見 CSS 色彩語法
5. `background-color` 作為 `background-image` 的底色
6. `background-color` vs `background` 簡寫
7. CSS 變數管理背景色

## 建議觀看方式

打開瀏覽器 DevTools，依序觀察每個範例的：

- `background-color`
- `background-image`
- `background`
- `opacity`
- `background-clip`
- 盒模型中的 `content`、`padding`、`border`、`margin`

CSS 第 13 章：背景屬性

背景顏色 background-color Demo

這個 demo 用視覺方式對照 background-color 的核心概念：透明背景、盒模型範圍、半透明背景、opacity 差異、背景圖片底色與 CSS 變數。

01

預設透明背景 vs 指定背景色

沒有設定 background-color 時，元素背景預設是 transparent，因此會露出父層或頁面的背景。

transparent

這張卡片沒有背景色，只是有邊框，所以可以看到棋盤格背景。

background-color: transparent;

指定背景色

這張卡片設定了明確背景色，因此會蓋住後方棋盤格。

background-color: #ffffff;
02

背景色與盒模型範圍

背景色預設會畫到 border-box，也就是內容、padding、border 背後；但不包含 margin。

margin 區域：背景色不會畫到這裡
content + padding + border 背後有背景色
.box {
  margin: 32px;
  padding: 32px;
  border: 10px dashed rgb(15 23 42 / 45%);
  background-color: #fde68a;
}
03

半透明背景 vs opacity

只想讓背景半透明，使用 rgb(... / alpha) 或 rgba()；使用 opacity 會讓文字與子元素一起變透明。

背景半透明

文字仍然清楚，只有背景有透明感。

推薦：只想淡化背景時使用 background-color: rgb(255 255 255 / 72%);

整個元素透明

文字、背景、子元素一起變淡。

適合：整個元素淡出時使用 opacity: .45;
04

常見色彩語法

background-color 可以接受 CSS <color>，例如關鍵字、HEX、RGB、HSL、currentColor、CSS 變數。

05

background-color 是 background-image 的底色

背景圖片或漸層在上層，背景色在下層；當圖片有透明區域或載入失敗時，底色會出現。

圖片 / 漸層 + 底色

你看到的斜向亮面是 background-image，深藍底是 background-color。

background-color + background-image

只有底色

拿掉 background-image 後，仍然有 background-color 作為基礎色。

background-color only
06

background-color vs background 簡寫

只想改顏色時，用 background-color；background 是簡寫，可能重設其他背景設定。

保留背景圖案

只改 background-color，原本的 background-image 還在。

background-color: #fff7ed;

簡寫重設背景

使用 background: #fff7ed; 會重設背景圖片與其他背景設定。

background: #fff7ed;
07

CSS 變數管理背景色

實務專案通常會用語意化變數管理背景色，方便主題切換與組件維護。

Dashboard Card

語意化背景色

這張卡片不是硬寫白色，而是使用 var(--card-bg)。

:root {
  --page-bg: #f8fafc;
  --card-bg: #ffffff;
}

.card {
  background-color: var(--card-bg);
}

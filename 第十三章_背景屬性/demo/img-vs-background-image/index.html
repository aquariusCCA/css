第 13 章｜背景屬性

img 標籤和背景圖片的區別

這個 demo 展示內容圖片與裝飾背景的差異：語意、alt、版面佔位、object-fit、background-size、功能 icon 與圖表情境。

01

內容圖片：使用 <img>

商品圖、頭像、文章圖、圖表這類圖片本身就是內容，應該放進 HTML，並提供合適的 alt。

商品圖片

這張圖片傳達商品外觀，拿掉後內容不完整，所以用 <img>。

使用者頭像

頭像通常代表人物或帳號資訊，也是內容的一部分。

02

裝飾圖片：使用 background-image

如果圖片只是讓畫面更漂亮，拿掉後不影響內容理解，就適合用 CSS 背景。

卡片底紋

這裡的圖案只是裝飾，不需要被輔助工具讀出。

Hero 氛圍背景

背景圖負責視覺氛圍；真正內容是標題與文字。

03

版面佔位：img 會佔空間，背景圖不會撐開盒子

<img> 會顯示自身尺寸

圖片是內容，會參與文件流。

背景圖不會撐開元素

上方 div 有背景圖，但沒有高度，因此幾乎看不到。

給盒子尺寸後才看得到

背景圖片需要元素本身有可見空間。

04

裁切與縮放：object-fit vs background-size

如果圖片是內容，不要只為了 cover 效果就改成背景圖；可以用 object-fit。

<img> + object-fit: cover

保留內容語意，同時達到裁切填滿效果。

background-size: cover

適合裝飾背景，不適合承載重要內容圖片。

05

功能 icon：描述功能，不是描述圖案

只有 icon 的按鈕必須有可及性名稱；背景圖沒有 alt，所以語意要由 button 提供。

使用 <img> 當 icon

alt="搜尋" 描述功能，而不是寫「放大鏡」。

使用背景圖當 icon

背景圖沒有 alt，所以 button 使用 aria-label="搜尋"。

06

快速判斷表

用 <img>

  • 商品圖
  • 文章圖
  • 頭像
  • Logo / 品牌識別
  • 圖表、截圖、流程圖

用 background-image

  • 背景紋理
  • 裝飾圖案
  • Hero 氛圍圖
  • 漸層遮罩
  • 卡片右下角裝飾

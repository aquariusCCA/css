第 13 章｜背景屬性

背景複合寫法 background

這個 demo 展示 background 簡寫、position / size 的斜線規則、 shorthand 重設陷阱、多背景圖，以及 origin / clip 的 box 值。

01

Longhand vs Shorthand

兩張卡片視覺相近：左邊拆開寫，右邊使用 background 複合寫法。

Longhand

分別寫 background-image、position、size、repeat。

Shorthand

使用一行 background: url(...) right bottom / size no-repeat color。

02

position / size：斜線不能省略

center / cover 中，center 是位置，cover 是尺寸。

position / size

background: url(...) center / cover no-repeat

這是 Hero 背景最常見的 shorthand 寫法。

03

Shorthand 重設陷阱

只想換背景圖時，如果誤用 background: url(...)，原本的 position、size、repeat 可能全部被重設。

原本狀態

背景圖在右下角、不重複、有尺寸。

錯誤：background: url(...)

重設成預設位置、預設尺寸、預設重複方式。

安全：background-image

只換圖片來源，保留原本 position、size、repeat。

04

多背景圖：逗號分層

第一個背景在最上層；背景色只能放在最後一層。

multiple backgrounds

漸層遮罩 + 風景圖 + 底色

linear-gradient(...), url(...) center / cover no-repeat #0f172a

05

origin / clip 的 box 值

一個 box 值同時設定 origin 和 clip；兩個 box 值時，第一個是 origin，第二個是 clip。

一個 box 值：padding-box

大致等於 origin 和 clip 都是 padding-box。

兩個 box 值：padding-box content-box

第一個控制 origin，第二個控制 clip。

06

複雜背景：拆開寫也很好

當 shorthand 太長時，拆回 longhand 反而更適合維護。

.hero {
  background:
    linear-gradient(rgb(15 23 42 / 72%), rgb(15 23 42 / 72%)),
    url("./assets/hero-scene.svg") center / cover no-repeat fixed #0f172a;
}
.hero {
  background-color: #0f172a;
  background-image:
    linear-gradient(rgb(15 23 42 / 72%), rgb(15 23 42 / 72%)),
    url("./assets/hero-scene.svg");
  background-position: center;
  background-size: cover;
  background-repeat: no-repeat;
  background-attachment: fixed;
}

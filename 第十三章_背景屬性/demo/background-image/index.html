第 13 章｜背景屬性

背景圖片 background-image

這個 demo 展示 url()、漸層、多背景圖、預設平鋪、遮罩、fallback 背景色， 以及 background-image 不會撐開盒子的特性。

01

基本背景圖：background-image 只設定圖片來源

有設定尺寸，所以背景圖看得見

這張卡片使用 background-image: url("./assets/hero-pattern.svg")， 並搭配 background-size 和 background-position 控制顯示效果。

背景色是圖片的保底色

如果圖片透明、載入較慢或失敗，background-color 仍然會提供基本視覺底色。

02

背景圖片不會撐開盒子

只有背景圖，沒有高度

左邊這個元素有 background-image，但沒有內容高度、沒有 padding、沒有 min-height。 它本身幾乎沒有可見面積，所以背景圖不會像 <img> 一樣把盒子撐開。

上方元素有背景圖，但沒有高度，所以幾乎看不到。

這個元素補上 min-height 後，背景圖才有地方顯示。

03

漸層也是背景圖片

linear-gradient()

線性漸層可以直接寫在 background-image。

radial-gradient()

放射狀漸層也屬於背景圖像。

漸層 + 圖案

可以把漸層和圖片疊在一起做出層次。

04

預設會平鋪：background-repeat: repeat

預設 repeat

小圖案會重複鋪滿整個盒子。

no-repeat

只顯示一次，位置和大小要另外控制。

05

多背景圖：第一個值在最上層

Layered Background

圖片上加一層半透明漸層遮罩

這裡的第一層是 linear-gradient() 遮罩，第二層是 url() 背景圖， 最底層是 background-color。

06

background-image vs <img>

裝飾圖：適合 background-image

右下角圖案只是裝飾，拿掉後內容仍然完整，所以適合用 CSS 背景圖。

內容圖：使用 <img>

如果圖片本身是內容的一部分，就應該放進 HTML，並提供適當的 alt。

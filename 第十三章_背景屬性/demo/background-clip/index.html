第 13 章｜背景屬性

背景裁減 background-clip

這個 demo 展示 border-box、padding-box、content-box、 text，以及 background-clip 和 background-origin 的差異。

01

三種盒模型裁切範圍

三張卡片有相同的 padding、dashed border、背景色與背景圖。 差別只在 background-clip。

border-box

背景畫到 border 外緣下方，所以可透過虛線邊框空隙看到背景。

padding-box

背景畫到 padding 外緣，不會畫到 border 下方。

content-box

背景只畫在 content 區域，padding 區也不會有背景。

02

background-origin vs background-clip

origin 控制背景圖片從哪裡開始定位； clip 控制背景最後畫到哪裡為止。

origin: content-box
clip: border-box

背景圖從 content-box 開始定位，但背景可畫到 border-box。

origin: border-box
clip: content-box

背景圖從 border-box 開始定位，但最後只顯示在 content-box。

03

漸層文字：background-clip: text

背景被裁切到文字形狀內，並搭配透明文字色讓漸層露出來。

background-clip: text

Gradient Text

這個效果使用 background-clip: text、 color: transparent 和 WebKit 前綴。

04

多背景圖逐層裁切

第一層角落圖案裁切到 content-box，第二層底紋裁切到 border-box。

Layer Mapping

content-box, border-box

第一個 background-clip 對應第一張背景圖； 第二個對應第二張背景圖。

05

進階值：border-area

如果瀏覽器支援 background-clip: border-area， 下方卡片會示範背景裁切到 border 繪製區域；否則會使用 fallback。

border-area support demo

這個值屬於較新的語法，實務使用前建議先確認瀏覽器支援。

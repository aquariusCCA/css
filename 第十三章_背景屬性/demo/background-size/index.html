第 13 章｜背景屬性

背景圖片大小 background-size

這個 demo 展示單值 / 雙值、auto、cover、contain、 平鋪 tile 大小，以及多背景圖逐層設定。

01

單值與雙值

一個值通常代表寬度，高度為 auto； 兩個值則是明確指定寬度與高度。

120px

寬度 120px，高度 auto。

120px 80px

寬度 120px，高度 80px，可能變形。

auto 120px

高度 120px，寬度依比例自動計算。

02

cover vs contain

cover 寧可裁切也要鋪滿；contain 寧可留白也要完整。

cover

保持比例並鋪滿容器，可能裁掉圖片的一部分。

contain

保持比例並完整顯示圖片，可能在周圍留下空白。

03

百分比與平鋪 tile 大小

百分比是相對背景定位區域計算；若背景會平鋪，background-size 也會改變每個 tile 的大小。

50% auto

寬度使用背景定位區域的 50%。

repeat + 48px

平鋪 tile 較小，圖案密度更高。

repeat + 96px

平鋪 tile 較大，圖案密度更低。

04

多背景圖逐層設定

第一層徽章用固定尺寸，第二層底紋用另一個尺寸，並各自對應自己的背景圖層。

Layer Mapping

96px 96px, 72px 72px

第一個 background-size 對應第一張背景圖； 第二個則對應第二張背景圖。

05

Shorthand 中的 position / size

在 background 簡寫中，background-size 必須寫在 background-position 後面，並用 / 分隔。 

.hero {
  background:
    url("./assets/hero-scene.svg")
    center / cover
    no-repeat
    #0f172a;
}

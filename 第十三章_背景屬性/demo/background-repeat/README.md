# background-repeat demo

本 demo 對應筆記：

```text
background-repeat_9_5_notes.md
```

## 開啟方式

直接用瀏覽器開啟：

```text
demo/background-repeat/index.html
```

## 檔案結構

```text
demo/background-repeat/
├─ index.html
├─ style.css
├─ README.md
└─ assets/
   ├─ badge.svg
   ├─ tile-dot.svg
   ├─ stripe-horizontal.svg
   └─ stripe-vertical.svg
```

## 展示重點

1. `background-repeat: repeat`
2. `background-repeat: no-repeat`
3. `background-repeat: repeat-x`
4. `background-repeat: repeat-y`
5. `background-repeat: space`
6. `background-repeat: round`
7. 雙值語法：`水平 | 垂直`
8. 多背景圖與 `background-repeat` 逐層對應
9. `background-repeat` 不控制背景顏色
10. 實務常見的水平線、垂直線裝飾

## 注意

本 demo 使用本地 SVG 圖檔，不依賴外部圖片網址。

第 13 章｜背景屬性

背景平鋪 background-repeat

這個 demo 展示 repeat、no-repeat、repeat-x、 repeat-y、space、round、雙值語法與多背景圖對應。

01

基礎值：repeat / no-repeat / repeat-x / repeat-y

同一張背景圖，因為重複方向不同，會產生完全不同的視覺結果。

repeat

水平與垂直都重複，也是預設值。

no-repeat

不重複，只顯示一張；通常要搭配 position。

repeat-x

只沿水平方向重複。

repeat-y

只沿垂直方向重複。

02

進階值：space / round

space 會保留圖片比例並分配間距； round 會調整圖片尺寸，讓圖案剛好填滿。

space

圖片完整顯示，剩餘空間變成圖案之間的間距。

round

圖案會被縮放，盡量剛好鋪滿，不留邊緣空白。

03

雙值語法：水平 | 垂直

background-repeat: repeat no-repeat; 的第一個值控制水平方向， 第二個值控制垂直方向。

repeat no-repeat

水平方向重複，垂直方向不重複；等同 repeat-x。

no-repeat repeat

水平方向不重複，垂直方向重複；等同 repeat-y。

space repeat

水平分配間距，垂直方向重複。

04

多背景圖：逐層對應 background-repeat

第一個 background-repeat 對應第一個 background-image； 第二個值對應第二張背景圖。

Layer Mapping

上層 badge 不重複，下層 tile 重複

這張卡片使用兩層背景圖： badge.svg 使用 no-repeat， tile-dot.svg 使用 repeat。

05

repeat 不控制背景顏色

背景色本來就會鋪滿元素；background-repeat 只控制背景圖片如何重複。

只有 background-color

沒有背景圖片，所以 repeat 值不會產生可見效果。

background-color + background-image

背景色在下方，重複圖片在上方。

06

實務組合：水平線與垂直線

repeat-x 做標題底線

小圖案沿 X 軸重複，常用於水平裝飾帶。

repeat-y 做側邊線

小圖案沿 Y 軸重複，常用於時間軸或側邊裝飾。

# background-position demo

本 demo 對應筆記：

```text
background-position_9_5_notes.md
```

## 開啟方式

直接用瀏覽器開啟：

```text
demo/background-position/index.html
```

## 檔案結構

```text
demo/background-position/
├─ index.html
├─ style.css
├─ README.md
└─ assets/
   ├─ marker.svg
   ├─ dot-grid.svg
   └─ corner-badge.svg
```

## 展示重點

1. 九宮格關鍵字定位
2. 單值省略規則
3. 數值定位：`24px 40px`
4. 百分比定位：`50% 50%`、`100% 100%`、`25% 75%`
5. 三值 / 四值語法
6. 多背景圖與 `background-position` 逐層對應
7. `background-position`、`background-size`、`background-repeat` 的分工

## 注意

本 demo 使用本地 SVG 圖檔，不依賴外部圖片網址。

第 13 章｜背景屬性

多背景圖

這個 demo 展示多背景圖的圖層順序、逐層對應、 Hero 遮罩背景、卡片裝飾圖 + 底紋，以及 shorthand 寫法。

01

圖層順序：第一個在最上層

左右兩張卡片只是交換圖層順序。第一個背景會壓在第二個背景上面。

badge 在上、halo 在下

第一個背景圖是徽章，所以它在最上層。

halo 在上、badge 在下

交換順序後，光暈會壓在徽章上方。

02

Hero：漸層遮罩 + 背景圖 + 底色

這是多背景圖最常見的實務場景之一：用第一層漸層讓文字更容易閱讀。

linear-gradient + image + color

Hero Overlay

背景層順序為：上層遮罩、下層圖片、最底部背景色。

03

逐層對應：position / size / repeat

第一層徽章不重複，第二層格紋重複，並使用不同的位置與尺寸。

Per-layer Mapping

badge + pattern + gradient

你可以把每一層背景當成一個獨立圖層，逐層設定自己的 repeat、position、size。

04

每層也能有自己的 origin / clip

這張卡片讓徽章只在 content-box 內定位與裁切， 底紋則鋪滿到 border-box。

content-box, border-box

第一層：徽章
第二層：底紋

05

Shorthand 多背景圖

每一層都可以寫成一組 shorthand，然後用逗號分隔。 

.card {
  background:
    url("./assets/badge.svg")
    right 24px bottom 24px / 96px 96px
    no-repeat,

    url("./assets/pattern-grid.svg")
    left top / 72px 72px
    repeat,

    linear-gradient(135deg, #eff6ff, #ffffff);
}

# 背景顏色

> 返回章節首頁：[README.md](./README.md)
>
> `background-color` 用來設定元素的背景顏色。

## 導讀
- `background-color` 主要用來控制盒子的底色。
- 預設值是 `transparent`，也就是透明。
- 背景色不會改變盒子的大小或位置，但很適合在開發時幫助確認版面範圍。

## 關鍵字
- background-color
- 背景顏色
- transparent
- rgba
- 透明度

## 30 秒複習入口
- `background-color` 用來設定背景顏色
- 預設值是 `transparent`
- 半透明背景可以用 `rgba()` 或帶 alpha 的 `rgb()`
- 背景色只影響視覺，不會改變盒模型尺寸

## 速查區

| 寫法 | 說明 |
| --- | --- |
| `background-color: transparent;` | 透明背景 |
| `background-color: pink;` | 指定單一背景色 |
| `background-color: rgba(0, 0, 0, .3);` | 半透明黑色背景 |

## 正文
`background-color` 屬性定義元素的背景顏色。

一般情況下，背景預設是 `transparent`，也就是透明。如果需要，也可以手動把背景設成透明色。

```css
background-color: transparent;
```

```css
div {
  width: 200px;
  height: 200px;
  /* background-color: transparent; */
  /* background-color: red; */
  background-color: pink;
}
```

```html
<div></div>
```

背景顏色不會改變盒子的尺寸或位置，但常常會在開發時先加上，方便觀察盒子的範圍與排版。

## 背景色半透明
CSS3 提供了背景色半透明效果。

```css
background-color: rgba(0, 0, 0, .3);
```

最後一個參數是 alpha 透明度，取值範圍是 `0` 到 `1`。

如果想表示不透明，可以寫 `1`；如果想表示完全透明，可以寫 `0`。

```css
div {
  width: 300px;
  height: 300px;
  background-color: rgba(0, 0, 0, .3);
  /* background-color: rgb(0 0 0 / 30%); */
}
```

```html
<div>隱形的翅膀</div>
```

注意：背景半透明指的是盒子的背景半透明，盒子裡面的內容不會一起變透明。

## 一句話總結
`background-color` 用來設定背景顏色，預設是透明，半透明背景可用 `rgba()` 或帶 alpha 的 `rgb()`。

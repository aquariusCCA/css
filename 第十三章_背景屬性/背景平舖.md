# 背景平舖

> 返回章節首頁：[README.md](./README.md)
>
> `background-repeat` 用來控制背景圖片是否平鋪，以及要沿哪個方向平鋪。

## 導讀
- 背景圖片預設會重複平鋪。
- 如果不想讓圖片重複，可以改成不平鋪。
- 也可以只沿著水平或垂直方向平鋪。

## 關鍵字
- background-repeat
- 平鋪
- repeat
- no-repeat
- repeat-x
- repeat-y

## 30 秒複習入口
- `background-repeat` 控制背景圖是否重複
- `repeat` 是預設值
- `no-repeat` 表示不平鋪
- `repeat-x` 只沿水平方向平鋪
- `repeat-y` 只沿垂直方向平鋪

## 速查區

| 寫法 | 說明 |
| --- | --- |
| `repeat` | 預設值，水平和垂直都平鋪 |
| `no-repeat` | 不平鋪 |
| `repeat-x` | 只沿 x 軸平鋪 |
| `repeat-y` | 只沿 y 軸平鋪 |

## 正文
如果需要在 HTML 頁面上對背景圖像進行平鋪，可以使用 `background-repeat` 屬性。

```css
background-repeat: repeat | no-repeat | repeat-x | repeat-y;
```

```css
div {
  width: 900px;
  height: 900px;
  background-color: pink;
  background-image: url(./images/logo.png);

  /* 1. 背景圖片不平鋪 */
  background-repeat: no-repeat;

  /* 2. 默認情況下，背景圖片是平鋪的 */
  /* background-repeat: repeat; */

  /* 3. 沿著 x 軸平鋪 */
  /* background-repeat: repeat-x; */

  /* 4. 沿著 y 軸平鋪 */
  /* background-repeat: repeat-y; */
}
```

```html
<div></div>
```

頁面元素既可以添加背景顏色，也可以添加背景圖片，只不過背景圖片會蓋在背景顏色上面。

## 一句話總結
`background-repeat` 用來控制背景圖片是否平鋪，以及要沿著哪個方向平鋪。

# 第13章 背景屬性

## 章節說明
本章整理 CSS 的背景相關屬性，目前收錄背景顏色 `background-color`、背景圖片 `background-image` 與背景平舖 `background-repeat`。

## 回查重點
- 想確認背景顏色怎麼寫時，看 `background-color`
- 想確認透明背景怎麼寫時，看 `transparent`
- 想確認半透明背景怎麼寫時，看 `rgba()` 或帶 alpha 的 `rgb()`
- 想確認背景圖片怎麼寫時，看 `background-image`
- 想確認背景圖預設是否平鋪時，看 `background-image` 的預設行為
- 想確認背景圖片是否平鋪時，看 `background-repeat`

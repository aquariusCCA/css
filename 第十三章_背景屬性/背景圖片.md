# 背景圖片

> 返回章節首頁：[README.md](./README.md)
>
> `background-image` 用來為元素設定背景圖片。

## 導讀
- `background-image` 常用在 logo、裝飾圖案、頁面大背景等場景。
- 背景圖片和背景顏色一樣，主要是裝飾用途，不會撐開盒子。
- 預設情況下，背景圖片會平鋪顯示。

## 正文
`background-image` 屬性描述了元素的背景圖像。

實際開發中，常見於 logo、裝飾性小圖片，或是大型背景圖片。它的優點是方便控制位置，也方便搭配其他背景屬性一起使用。

```css
background-image: url(圖片路徑);
```

### 注意點
- `url()` 中通常可以省略引號。
- 背景圖片預設會在水平和垂直方向平鋪。
- 背景圖片只會作為盒子的裝飾，不會撐開盒子。

```css
div {
  width: 300px;
  height: 300px;
  background-image: url(./images/logo.png);
}
```

```html
<div></div>
```

## 一句話總結
`background-image` 用來設定背景圖片，預設會平鋪，而且只負責裝飾，不會影響盒子的大小。

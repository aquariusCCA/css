# 第10章 顏色

## 章節說明
本章整理 CSS 常用顏色表示法：命名色、HEX、RGB/RGBA、HSL/HSLA，並補充現代 CSS 顏色語法、透明度寫法、實戰選型與常見混淆點。

這份筆記的定位不是只背語法，而是讓你之後看到設計稿色碼、調整 UI 深淺、處理透明邊框或背景時，能快速判斷該用哪一種顏色表示法。

## 建議閱讀順序
1. 先讀「30 秒複習入口」，快速建立整體印象。
2. 再看「速查區」，掌握每種顏色表示法的寫法。
3. 接著讀正文第 1～6 節，理解命名色、HEX、RGB/RGBA、HSL/HSLA 與透明度。
4. 最後讀「實戰選型建議」與「常見混淆點」，建立工程實務判斷能力。

## 回查重點
- 想快速確認顏色寫法時，看「速查區」。
- 想知道 `#abc`、`#aabbcc`、`#aabbcc80` 的差異時，看「HEX」。
- 想處理透明度時，看「RGB/RGBA」、「HSL/HSLA」與「HEX alpha」。
- 想比較 RGB 和 HSL 的差異時，看正文第 4、5 節。
- 想知道實務上該用哪一種表示法時，看「實戰選型建議」。
- 想排查觀念錯誤時，看「常見混淆點」。

## 本章學完後要能回答
- CSS 有哪些常見顏色表示法？
- HEX、RGB、HSL 的核心差異是什麼？
- `alpha` 是什麼？`0` 和 `1` 分別代表什麼？
- 為什麼 HSL 適合調整同一顏色的深淺？
- 實務上什麼時候用 HEX，什麼時候用 RGB/HSL？

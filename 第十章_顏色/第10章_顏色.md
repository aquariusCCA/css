> 本章整理 CSS 常用顏色表示法：命名色、HEX、RGB/RGBA、HSL/HSLA。

## 導讀
- CSS 可以用不同方式指定顏色。
- 最常見的是命名色、HEX、RGB/RGBA 與 HSL/HSLA。
- `RGBA` 和 `HSLA` 都多了 `alpha`，用來控制透明度。
- 如果要快速調整顏色深淺，HSL 通常比較直觀。

## 正文

### 1. CSS 顏色簡介
CSS 的 `color` 相關屬性可以用來設定文字、背景、邊框等元素的顏色。
常見的表示法有 `命名色`、`HEX`、`RGB` 和 `HSL`。
其中 `RGBA`、`HSLA` 是在前兩種表示法上再加入透明度。

![CSS 顏色概覽](./images/image_1776221801211_d0109e30.png)

### 2. 命名色
命名色是直接使用顏色英文名稱，例如 `red`、`blue`、`green`。
這種寫法最直觀，適合快速指定基本顏色，但可調整的彈性較少。

![CSS 命名色](./images/image_1776221801216_4ad1b0a7.png)

### 3. HEX
HEX 是最常見的色碼表示法，格式通常是 `#RRGGBB`。

- `RR` 代表紅色通道
- `GG` 代表綠色通道
- `BB` 代表藍色通道
- 每個通道的範圍是 `00` 到 `ff`

例如 `#ff0000` 代表純紅色。

如果每一組數值都由重複字元組成，可以縮寫成 3 位形式。
例如 `#aabbcc` 可以寫成 `#abc`。

![HEX 色碼](./images/image_1776221801217_7be6ceb1.png)

### 4. RGB 與 RGBA
`rgb()` 用紅、綠、藍三個數值來表示顏色。
每個通道的範圍是 `0` 到 `255`。

例如：

```css
rgb(141, 0, 0)
```

`rgba()` 則在 RGB 後面多一個 `alpha`，用來控制透明度。
`alpha` 的範圍是 `0` 到 `1`。

- `0` 代表完全透明
- `1` 代表完全不透明

例如：

```css
rgba(255, 0, 0, 0.5)
```

![RGB 色碼](./images/image_1776221801217_01ac12e6.png)
![RGBA 色碼](./images/image_1776221801217_c6e504db.png)

### 5. HSL 與 HSLA
`hsl()` 用三個概念來描述顏色：

- `hue`：色相，範圍通常是 `0` 到 `360`
- `saturation`：飽和度，百分比
- `lightness`：亮度，百分比

`hsla()` 則是在 HSL 後面加上 `alpha`，用法和 `rgba()` 類似。

例如：

```css
hsl(0, 100%, 50%)
hsla(0, 100%, 50%, 0.5)
```

HSL 的優點是比較容易從「顏色的明暗與鮮豔程度」去思考。
如果要調整同一顏色的深淺，HSL 通常比 RGB 更直觀。

### 6. 常見混淆點
- `HEX` 本質上也是 RGB 的另一種寫法。
- `RGBA` 和 `HSLA` 的差別，不在透明度，而在前面的顏色描述方式。
- `HSL` 的 `S` 是飽和度，`L` 是亮度，不是光線強度。
- `alpha` 越小越透明。

## 一句話總結
CSS 顏色可以用命名色、HEX、RGB 或 HSL 表示；如果需要透明度，就用 `RGBA` 或 `HSLA`。

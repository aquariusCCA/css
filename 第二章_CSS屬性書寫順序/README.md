# 第二章 CSS 屬性書寫順序

## 章節導讀

這一章整理的是 CSS 屬性在實務中常見的書寫順序觀念。

重點不只是背一份固定清單，而是理解為什麼要把相關屬性放在一起，讓樣式更容易閱讀、維護與協作。

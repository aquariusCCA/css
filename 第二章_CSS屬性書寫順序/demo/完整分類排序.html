完整分類排序

這個範例把單一規則拆成五個類別：布局與定位、盒模型與尺寸、文字與字體、背景與外觀、互動與動畫。

排序後的卡片

當樣式數量變多時，依類別分組可以更快找到要改的屬性。

.example { /* 1. 布局與定位 */ display: flex; position: relative; /* 2. 盒模型與尺寸 */ max-width: 640px; padding: 24px; border: 1px solid #dbe4ee; border-radius: 18px; /* 3. 文字與字體 */ color: #334155; font-size: 16px; line-height: 1.7; /* 4. 背景與外觀 */ background: linear-gradient(135deg, #fff, #eff6ff); box-shadow: 0 14px 28px rgb(15 23 42 / 10%); /* 5. 互動、動畫與其他 */ cursor: pointer; transition: transform 0.2s ease; }

# 行內與行內塊元素在父元素中的居中

## 學習目標

讀完這篇筆記，你應該能夠：

- 使用 `text-align: center` 讓行內或行內塊內容水平居中。
- 理解行內與行內塊元素可以被父元素當作文字處理。
- 使用 `line-height` 與 `vertical-align: middle` 處理固定高度單行場景的垂直居中。
- 說明這種居中方式的適用條件與限制。

## 問題情境

塊元素水平居中常用 `margin: 0 auto`，但如果要居中的對象是文字、圖片、`span` 或 `input`，做法就不一樣。

例如：

```html
<div class="header">
  <span>裡面的文字</span>
</div>
```

`span` 是行內元素，它會參與文字流排版。這時要控制它在父元素中的水平位置，通常是對父元素設定文字對齊。

## 一句話理解

行內與行內塊元素可以像文字一樣被父元素對齊，因此水平居中常寫在父元素的 `text-align` 上。

## 水平居中

如果子元素是行內元素或行內塊元素，可以給父元素設定：

```css
text-align: center;
```

範例：

```css
.header {
  width: 900px;
  height: 200px;
  background-color: pink;
  margin: 100px auto;
  text-align: center;
}
```

```html
<div class="header">
  <span>裡面的文字</span>
</div>
```

這裡 `.header` 本身用 `margin: 100px auto` 水平置中；`.header` 裡面的 `span` 則透過父元素的 `text-align: center` 水平置中。這兩層居中控制的是不同對象。

圖片如果是行內或行內替換元素，也可以被 `text-align: center` 影響：

```html
<div class="header">
  <img src="./images/down.jpg" alt="">
</div>
```

上面的圖片路徑只是原始示意碼，重點是說明圖片可以被父元素當作行內內容對齊。

## 垂直居中：固定高度單行場景

行內或行內塊元素如果要在固定高度父元素中垂直居中，可以搭配：

- 父元素 `height` 等於 `line-height`。
- 子元素設定 `vertical-align: middle`。
- 父元素視情況設定 `font-size: 0`，減少基線與空白字元造成的干擾。

範例：

```css
.outer {
  width: 400px;
  height: 400px;
  background-color: gray;
  text-align: center;
  line-height: 400px;
  font-size: 0;
}

input {
  vertical-align: middle;
}

span {
  font-size: 20px;
  vertical-align: middle;
  background-color: orange;
}
```

```html
<div class="outer">
  <span>出來玩啊</span>
  <input type="text">
</div>
```

## 範例拆解

- `.outer { text-align: center; }`：讓內部行內與行內塊內容水平居中。
- `.outer { line-height: 400px; }`：讓單行文字的行高等於父元素高度，形成垂直居中的基礎。
- `span` 和 `input` 設定 `vertical-align: middle`：讓不同類型的行內內容以中線對齊。
- `.outer { font-size: 0; }`：減少空白字元與字體基線造成的細微偏差。
- `span { font-size: 20px; }`：恢復 `span` 自己的文字大小。

這種方法適合固定高度、單行內容、inline / inline-block 場景。

## 常見錯誤

### 把 text-align 寫在子元素上

```css
span {
  text-align: center;
}
```

如果你想讓 `span` 這個行內元素在父元素裡水平居中，應該設定父元素的 `text-align: center`。寫在 `span` 自己身上，通常只會影響它內部文字的對齊，而不是它在父元素中的位置。

### 忘記恢復 font-size

父元素設定 `font-size: 0` 後，子元素文字也可能繼承到 0。只要子元素要顯示文字，就應該另外設定字體大小。

### 用 line-height 處理多行內容

`line-height` 等於父元素高度的技巧主要適合單行內容。若文字會換行，或內容高度不固定，這種方法就容易出問題，應改用更彈性的布局方式。

## 實務判斷

- 行內或行內塊元素水平居中：父元素 `text-align: center`。
- 單行文字在固定高度盒子中垂直居中：可用 `line-height`。
- 多個 inline-block 元素一起垂直對齊：可搭配 `vertical-align: middle`。
- 複雜的水平與垂直居中：優先評估 Flexbox 或 Grid。

## 重點整理

- 行內與行內塊元素可被父元素當作文字處理。
- 水平居中通常寫在父元素：`text-align: center`。
- 固定高度單行垂直居中可用 `line-height` 搭配 `vertical-align`。
- `font-size: 0` 可減少空白與基線干擾，但要恢復子元素字體大小。

# 塊元素在父元素中的居中

## 學習目標

讀完這篇筆記，你應該能夠：

- 使用 `margin: 0 auto` 讓固定寬度塊元素水平居中。
- 說明塊元素水平居中的必要條件。
- 用固定高度場景的計算方式處理垂直居中。
- 理解傳統 `margin-top` 垂直居中方法的限制。

## 問題情境

你常會需要把一個固定寬度的盒子放在父容器中間，例如：

- 讓頁面主體 `.container` 在瀏覽器中水平置中。
- 讓固定寬度的 header 區塊置中。
- 讓一個小盒子在固定大小的外層盒子中水平或垂直置中。

對塊元素來說，水平置中最常見的傳統寫法是 `margin: 0 auto`。

## 一句話理解

塊元素水平居中的核心是：子元素有明確寬度，左右外邊距交給瀏覽器平均分配。

## 水平居中條件

讓塊級盒子水平居中，通常需要兩個條件：

1. 子元素必須有明確的 `width`。
2. 子元素左右外邊距都設為 `auto`。

常見寫法有三種：

```css
/* 1 */
margin-left: auto;
margin-right: auto;

/* 2 */
margin: auto;

/* 3 */
margin: 0 auto;
```

實務上最常看到的是 `margin: 0 auto`，意思是上下外距為 `0`，左右外距自動分配。

## 水平居中範例

```css
.header {
  width: 900px;
  height: 200px;
  background-color: pink;
  margin: 100px auto;
}
```

```html
<div class="header"></div>
```

這裡 `.header` 有明確寬度 `900px`，左右外距是 `auto`，瀏覽器會把父容器剩下的水平空間平均分給左右兩側，因此盒子會水平居中。

## 垂直居中：固定高度的傳統作法

如果父元素和子元素的高度都已知，可以用 `margin-top` 做垂直居中：

```text
(父元素 content 高度 - 子元素總高) / 2
```

例如父元素高 `400px`，子元素高 `100px`：

```text
(400 - 100) / 2 = 150
```

所以子元素可以設定 `margin-top: 150px`。

## 完整範例拆解

```css
.outer {
  width: 400px;
  height: 400px;
  background-color: gray;
  overflow: hidden;
}

.inner {
  width: 200px;
  height: 100px;
  background-color: orange;
  margin: 0 auto;
  margin-top: 150px;
  text-align: center;
  line-height: 100px;
}
```

```html
<div class="outer">
  <div class="inner">inner</div>
</div>
```

逐段看：

- `.outer` 是固定大小的父容器。
- `.inner` 有明確寬度 `200px`，所以 `margin: 0 auto` 可以讓它水平居中。
- `.inner` 高度是 `100px`，父容器高度是 `400px`，所以 `margin-top: 150px` 讓它在垂直方向看起來居中。
- `.outer { overflow: hidden; }` 可避免子元素上外距塌陷跑到父元素外面，影響觀察結果。
- `text-align: center` 和 `line-height: 100px` 是讓 `.inner` 裡的文字居中，和 `.inner` 盒子本身居中是兩件事。

## 常見錯誤

### 沒有設定 width

```css
.box {
  margin: 0 auto;
}
```

如果正常流中的塊元素沒有設定寬度，它通常會撐滿父容器可用寬度。這時左右已經沒有剩餘空間可分配，看起來就不會有「居中」效果。

### 以為 margin: auto 會自動垂直居中

在一般正常流中，`margin: auto` 常用於水平居中；它不會在普通塊布局中自動完成垂直居中。垂直居中需要看具體布局方式，固定高度場景可用計算後的 `margin-top`，更通用的場景通常改用 Flexbox 或 Grid。

### 忽略外邊距塌陷

父子塊元素之間可能出現垂直外邊距塌陷。若你用 `margin-top` 做垂直定位，卻發現父元素也跟著被推開，需要檢查是否受到外邊距塌陷影響。

## 實務判斷

- 固定寬度大容器水平居中：`width` + `margin: 0 auto`。
- 固定高度父子盒子垂直居中：可用計算後的 `margin-top`，但要注意限制。
- 內容高度不固定或需求更彈性：優先考慮 Flexbox 或 Grid。
- 只想讓文字在盒子內水平置中：那是 `text-align: center`，不是 `margin: 0 auto`。

## 重點整理

- 塊元素水平居中需要明確 `width` 與左右 `auto` 外距。
- `margin: 0 auto` 的 `auto` 是讓瀏覽器分配左右剩餘空間。
- 固定高度垂直居中可用 `(父元素高度 - 子元素高度) / 2` 計算 `margin-top`。
- 傳統垂直居中技巧有條件限制，複雜布局應改用更合適的布局方式。

# 背景屬性入門：顏色、圖片與 img 的選擇

## 學習目標

讀完這篇筆記，你應該能夠：

- 理解 CSS 背景屬性主要用來控制元素的裝飾層。
- 使用 `background-color` 設定盒子的背景顏色。
- 使用 `rgba()` 製作半透明背景，並知道它不會讓內容一起透明。
- 使用 `background-image` 加入背景圖片。
- 分辨什麼情況該用 `img`，什麼情況該用 CSS 背景圖。

## 問題情境

做網頁版面時，你常會遇到幾種需求：想看清楚一個盒子到底多大、想讓卡片有底色、想放一張裝飾性的 logo、想做大面積的頁面背景，或想在文章中呈現一張真正重要的圖片。

這些需求看起來都和「圖片」或「顏色」有關，但不一定都該用同一種做法。CSS 背景屬性適合處理元素的視覺裝飾；真正屬於內容的一張圖片，通常應該使用 HTML 的 `img` 標籤。

## 一句話理解

背景是元素盒子背後的裝飾層，可以放顏色或圖片，但背景圖片不會撐開元素，也不應該承載重要內容。

## 背景屬性可以控制什麼

常見背景屬性包含：

- 背景顏色：`background-color`
- 背景圖片：`background-image`
- 背景是否平鋪：`background-repeat`
- 背景圖片位置：`background-position`
- 背景是否固定：`background-attachment`
- 背景圖片尺寸：`background-size`

本篇先建立入門概念：背景顏色、半透明背景、背景圖片，以及背景圖和 `img` 的差異。後續筆記再進一步處理平鋪、定位、裁剪與大小。

## background-color：設定盒子的背景顏色

`background-color` 用來設定元素的背景顏色。

```css
.box {
  width: 200px;
  height: 200px;
  background-color: pink;
}
```

```html
<div class="box"></div>
```

如果沒有手動指定背景色，元素的背景預設是 `transparent`，也就是透明。

```css
.box {
  background-color: transparent;
}
```

在練習排版時，常會先給盒子加上背景色，因為背景色可以幫你看清楚盒子的大小與位置。它不會改變盒模型大小，只是把元素背景區域塗上顏色。

## rgba()：只讓背景半透明

如果想要背景顏色半透明，可以使用 `rgba()`：

```css
.panel {
  background-color: rgba(0, 0, 0, 0.3);
}
```

最後一個參數是 alpha 透明度，範圍是 `0` 到 `1`：

- `0`：完全透明
- `1`：完全不透明
- `0.3`：30% 不透明

也可以把小數前面的 `0` 省略：

```css
.panel {
  background-color: rgba(0, 0, 0, .3);
}
```

要特別注意：`rgba()` 用在背景色時，半透明的是盒子的背景，盒子裡面的文字或其他內容不會一起變透明。

```css
.panel {
  width: 300px;
  height: 180px;
  background-color: rgba(0, 0, 0, .3);
  color: #fff;
}
```

這和直接對整個元素設定 `opacity` 不同。`opacity` 會讓元素和內容一起透明；`rgba()` 可以只控制背景色。

## background-image：設定背景圖片

`background-image` 用來設定元素的背景圖像。

```css
.logo-box {
  width: 300px;
  height: 300px;
  background-image: url(./images/logo.png);
}
```

背景圖片常見於：

- logo 裝飾
- 小圖示裝飾
- 大尺寸頁面背景
- 不需要被搜尋引擎或輔助工具當成內容的視覺圖案

寫 `url()` 時，引號通常可以省略：

```css
.logo-box {
  background-image: url(./images/logo.png);
}
```

背景圖片有幾個入門時很重要的特性：

1. 預設會在水平和垂直方向平鋪。
2. 只是盒子的背景裝飾，不會撐開盒子。
3. 可以和背景顏色同時存在。
4. 當背景圖片覆蓋在背景顏色上方時，會遮住下面的背景色。

所以只寫背景圖片但沒有設定元素尺寸時，可能看不到預期效果，因為背景圖不會像 `img` 一樣把元素撐出大小。

## img 標籤和背景圖片怎麼選

同樣是顯示圖片，`img` 和 `background-image` 的角色不同。

| 做法 | 特性 | 適合情境 |
| --- | --- | --- |
| `img` | 是 HTML 內容；不設定寬高也會依原始尺寸顯示 | 商品圖、文章圖片、人物照片、內容中重要圖片 |
| `background-image` | 是 CSS 裝飾；需要元素本身有尺寸 | 裝飾圖、背景紋理、可替換的視覺效果、大背景圖 |

如果圖片是頁面真正要表達的內容，通常使用 `img`：

```html
<img src="./images/product.jpg" alt="商品照片">
```

如果圖片只是裝飾，可以使用背景圖：

```css
.hero {
  width: 100%;
  height: 360px;
  background-image: url(./images/hero-bg.jpg);
}
```

## 常見錯誤

### 以為背景圖片會撐開盒子

背景圖片只是裝飾，不會決定元素尺寸。下面這段如果 `.banner` 沒有內容，也沒有寬高，背景圖片可能看不到：

```css
.banner {
  background-image: url(./images/banner.jpg);
}
```

應該給元素明確尺寸，或讓內容把盒子撐開：

```css
.banner {
  height: 300px;
  background-image: url(./images/banner.jpg);
}
```

### 用背景圖放重要內容圖片

如果一張圖片本身是內容，例如商品照片、教學示意圖、人物照片，應該優先使用 `img`，並提供適當的 `alt`。背景圖比較適合裝飾，或可以被 CSS 控制位置、平鋪、尺寸的視覺素材。

### 混淆背景半透明與整個元素透明

`background-color: rgba(...)` 只影響背景色；`opacity` 會影響整個元素和內容。若只是想做半透明底色，不要直接把整個元素設成半透明。

## 實務判斷準則

- 想看清楚盒子範圍：先加 `background-color`。
- 想做半透明底色但保留文字清楚：用 `rgba()` 背景色。
- 想加裝飾圖片或大背景：用 `background-image`。
- 圖片是內容的一部分：用 `img`。
- 使用背景圖時，要記得元素本身需要尺寸。

## 重點整理

- CSS 背景屬性用來控制元素的背景裝飾。
- `background-color` 設定背景顏色，預設是 `transparent`。
- `rgba()` 可讓背景半透明，內容不會一起透明。
- `background-image` 是裝飾層，不會撐開盒子。
- 重要圖片用 `img`，裝飾圖片用背景圖。

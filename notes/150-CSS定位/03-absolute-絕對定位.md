# absolute：絕對定位

## 學習目標

讀完這篇筆記，你應該能夠：

- 說明絕對定位會脫離文檔流，不再佔原本位置。
- 判斷絕對定位元素以哪個祖先元素作為參照。
- 理解「最近定位祖先」與「包含塊」在絕對定位中的作用。
- 避免把絕對定位誤認為永遠相對瀏覽器視口。

## 使用情境

當你想把元素放到父盒子的角落、做遮罩、放置角標、圖示、關閉按鈕，或讓元素覆蓋在其他內容上方時，常會使用絕對定位。

絕對定位很適合做局部精準擺放，但它會脫離文檔流，因此也容易影響父元素高度與兄弟元素排列。

## 一句話理解

絕對定位會讓元素脫離文檔流，並相對最近的定位祖先進行偏移。

## 基本語法

```css
.son {
  position: absolute;
  top: 10px;
  left: 10px;
}
```

`top` 和 `left` 的參照對象不是永遠固定的。它取決於 `.son` 的包含塊，最常見的判斷方式是尋找最近一層有定位的祖先元素。

## 絕對定位的參照點

絕對定位的參照點可以這樣判斷：

1. 從自己往上找祖先元素。
2. 找最近一個有定位的祖先，也就是 `position` 不是 `static` 的祖先。
3. 如果找不到定位祖先，通常相對初始包含塊定位，可先理解為文檔根部，而不是固定在瀏覽器可視區。

這也是為什麼絕對定位常搭配父級 `position: relative` 使用。

## 父級沒有定位的範例

```css
.father {
  width: 500px;
  height: 500px;
  background-color: skyblue;
}

.son {
  position: absolute;
  top: 10px;
  left: 10px;
  width: 200px;
  height: 200px;
  background-color: pink;
}
```

```html
<body>
  <div class="father">
    <div class="son"></div>
  </div>
</body>
```

這裡 `.father` 沒有定位，`.son` 不會以 `.father` 為定位參照。它會繼續往外尋找定位祖先；如果都沒有，通常會以初始包含塊為參照。

## 祖先有定位的範例

```css
.yeye {
  position: relative;
  width: 800px;
  height: 800px;
  background-color: hotpink;
  padding: 50px;
}

.father {
  width: 500px;
  height: 500px;
  background-color: skyblue;
}

.son {
  position: absolute;
  left: 30px;
  bottom: 10px;
  width: 200px;
  height: 200px;
  background-color: pink;
}
```

```html
<body>
  <div class="yeye">
    <div class="father">
      <div class="son"></div>
    </div>
  </div>
</body>
```

拆解：

- `.son` 是絕對定位元素。
- `.father` 沒有定位，不作為 `.son` 的定位參照。
- `.yeye` 有 `position: relative`，是最近的定位祖先。
- 因此 `.son` 會以 `.yeye` 作為參照來計算 `left: 30px` 與 `bottom: 10px`。

## 絕對定位的特點

絕對定位要抓住這些行為：

- 元素不再佔用原本位置，也就是脫標。
- 脫離文檔流後，會影響後面的兄弟元素與父元素高度計算。
- 行內、行內塊、塊級元素設定絕對定位後，都會變成定位元素。
- 定位元素可以設定寬高；若不設定寬高，預設大小常由內容撐開。
- 絕對定位和浮動同時設定時，定位規則會主導，浮動不再是主要布局方式。

## 邊偏移注意事項

初學時建議同一個軸向避免同時設定兩邊：

- 水平軸：先避免同時寫 `left` 和 `right`。
- 垂直軸：先避免同時寫 `top` 和 `bottom`。

這不是說 CSS 規格中完全不能同時設定，而是同時設定時會涉及元素尺寸、`auto` 值與約束計算。入門階段若只是要指定位置，選一邊會更清楚。

## 常見錯誤

### 誤以為 absolute 永遠相對瀏覽器視口

絕對定位會找最近定位祖先。只有在沒有定位祖先時，才會退到較外層的包含塊。若要讓它相對父盒子，父盒子就需要加定位。

### 忘記它會脫標

絕對定位元素不再佔原本位置。若父元素裡只有絕對定位子元素，父元素可能不會被子元素撐高。

### 用 absolute 取代正常排版

絕對定位適合做局部疊放或精準位置，不適合把整頁主要布局都用固定座標擺出來。

## 實務判斷準則

- 子元素要放到父盒子內某個角落：常用 absolute。
- 父盒子要成為參照：父盒子加 `position: relative`。
- 需要元素仍影響文檔流：不要用 absolute。
- 使用 absolute 前先確認它脫標後是否會影響父元素高度與其他內容。

## 重點整理

- `absolute` 會脫離文檔流，不佔原本位置。
- 它通常相對最近定位祖先定位。
- 沒有定位祖先時，通常相對初始包含塊。
- 父級加 `position: relative` 是控制絕對定位參照點的常見方法。
- 同軸同時設定兩邊偏移時要理解約束規則，初學可先避免。

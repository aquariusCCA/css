# 外部樣式表：把 CSS 獨立成檔案

## 學習目標

讀完這篇筆記，你應該能夠：

- 說明外部樣式表的基本概念。
- 使用 `.css` 檔案存放樣式。
- 使用 `<link>` 標籤在 HTML 中引入 CSS 檔案。
- 理解為什麼實際開發通常使用外部樣式表。

## 一句話理解

外部樣式表是把 CSS 單獨寫在 `.css` 檔案中，再透過 HTML 的 `<link>` 標籤引入使用。

這種方式讓 HTML 和 CSS 分成不同檔案，是實際開發中最常見的 CSS 引入方式。

## 第一步：建立 CSS 檔案

使用外部樣式表時，會先建立一個副檔名為 `.css` 的檔案。

例如建立 `style.css`：

```css
/* style.css */
div {
  color: pink;
}
```

這個檔案裡只寫 CSS，不寫 HTML 標籤結構。

也就是說，`style.css` 裡應該放這種內容：

```css
p {
  font-size: 20px;
}
```

不應該把完整 HTML 寫進 CSS 檔案中。

## 第二步：在 HTML 中引入 CSS

建立 CSS 檔案後，需要在 HTML 文件中使用 `<link>` 標籤引入它。

```html
<link rel="stylesheet" href="./style.css">
```

其中：

| 屬性 | 說明 |
| --- | --- |
| `rel="stylesheet"` | 表示引入的資源是一份樣式表 |
| `href="./style.css"` | 指向 CSS 檔案的位置 |

`<link>` 標籤通常放在 HTML 的 `<head>` 裡。

## 完整範例

假設專案中有兩個檔案：

```text
index.html
style.css
```

`style.css`：

```css
div {
  color: pink;
}
```

`index.html`：

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <link rel="stylesheet" href="./style.css">
</head>
<body>
  <div>來呀~快活呀，反正有大把時間...</div>
</body>
</html>
```

瀏覽器讀取 `index.html` 時，會根據 `<link>` 找到 `style.css`，再把 CSS 套用到頁面中。

## 外部樣式表的作用範圍

外部樣式表可以被多個 HTML 頁面引入。

例如：

```html
<link rel="stylesheet" href="./style.css">
```

如果 `index.html`、`about.html` 和 `contact.html` 都引入同一份 `style.css`，這三個頁面就可以共用同一套樣式。

這是外部樣式表很重要的價值：樣式可以集中管理，也可以重複使用。

## 為什麼實際開發常用外部樣式表

外部樣式表適合樣式比較多、頁面比較多、需要長期維護的專案。

它的優點包含：

- HTML 和 CSS 分成不同檔案，結構更清楚。
- 同一份 CSS 可以被多個頁面共用。
- 修改共用樣式時，只需要改 CSS 檔案。
- CSS 檔案可以被瀏覽器快取，減少重複下載。
- 更符合「結構與樣式分離」的開發思路。

當你開始製作真正的網站時，外部樣式表通常會比行內樣式表和內部樣式表更合適。

## 路徑要寫正確

使用外部樣式表時，常見錯誤是 `href` 路徑寫錯。

例如：

```html
<link rel="stylesheet" href="./style.css">
```

這表示 CSS 檔案和目前 HTML 檔案在同一層資料夾。

如果 CSS 放在 `css` 資料夾中，可能要寫成：

```html
<link rel="stylesheet" href="./css/style.css">
```

路徑錯誤時，HTML 仍然會顯示內容，但 CSS 不會套用，看起來就像樣式失效。

## 常見誤解

### 誤解：建立了 CSS 檔案，樣式就會自動套用

CSS 檔案不會自動連到 HTML。必須在 HTML 中使用 `<link>` 引入，瀏覽器才知道要載入哪一份樣式表。

### 誤解：CSS 檔案裡可以寫 HTML 結構

CSS 檔案中應該只放 CSS 規則。HTML 結構仍然放在 `.html` 檔案中。

### 誤解：外部樣式表只適合大型網站

外部樣式表在大型網站中尤其有價值，但即使是小型練習，只要想練習正式的檔案分工，也可以使用外部樣式表。

## 重點整理

- 外部樣式表把 CSS 寫在獨立的 `.css` 檔案中。
- HTML 透過 `<link rel="stylesheet" href="...">` 引入 CSS。
- `<link>` 標籤通常放在 `<head>` 裡。
- 外部樣式表可以讓多個頁面共用同一份 CSS。
- 實際開發中常用外部樣式表，因為它更利於維護、複用和結構分離。

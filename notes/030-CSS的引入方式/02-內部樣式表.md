# 內部樣式表：把 CSS 集中寫在同一份 HTML 裡

## 學習目標

讀完這篇筆記，你應該能夠：

- 說明內部樣式表的寫法與放置位置。
- 理解 `<style>` 標籤的用途。
- 判斷內部樣式表適合的使用情境。
- 說明內部樣式表和結構與樣式分離之間的關係。

## 一句話理解

內部樣式表是把 CSS 寫在 HTML 文件內部的 `<style>` 標籤中，通常放在 `<head>` 裡，用來控制目前這一個頁面的元素樣式。

基本形式如下：

```html
<head>
  <style>
    div {
      color: pink;
    }
  </style>
</head>
```

這段 CSS 會套用到目前 HTML 頁面中的 `div` 元素。

## `<style>` 標籤的用途

`<style>` 標籤用來在 HTML 文件中放置 CSS 程式碼。

雖然部分瀏覽器可能容忍 `<style>` 出現在其他位置，但在教學與正式寫法中，建議把它放在 `<head>` 裡。

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>內部樣式表</title>
  <style>
    div {
      color: pink;
    }
  </style>
</head>
<body>
  <div>這段文字會套用內部樣式表中的樣式。</div>
</body>
</html>
```

在這個例子中，CSS 沒有寫在 `<div>` 標籤的 `style` 屬性裡，而是集中寫在 `<style>` 標籤中。

## 作用範圍

內部樣式表的作用範圍通常是「目前這一個 HTML 頁面」。

例如：

```html
<style>
  p {
    color: pink;
  }
</style>
```

這段 CSS 會影響目前頁面中符合選擇器 `p` 的段落元素。只要在同一份 HTML 裡，所有符合條件的元素都可以套用同一組樣式。

這比行內樣式更容易維護，因為同一類元素的樣式可以集中管理。

## 適合的使用情境

內部樣式表常用在練習、示範或單一頁面的樣式設定。

適合使用的情境包含：

- 初學 CSS 時，在同一份 HTML 中快速練習選擇器和屬性。
- 製作單頁範例，不需要和其他頁面共用樣式。
- 測試某段 CSS 對目前頁面的影響。

如果網站有多個頁面，而且多個頁面需要共用同一套樣式，通常就應該改用外部樣式表。

## 優點與限制

內部樣式表的優點是結構比行內樣式清楚。

例如，所有 CSS 都集中在 `<style>` 裡：

```html
<style>
  h1 {
    color: pink;
  }

  p {
    font-size: 20px;
  }
</style>
```

讀者可以一次看到目前頁面的樣式規則，不需要在每個 HTML 標籤裡尋找 `style` 屬性。

不過，內部樣式表仍然沒有做到完全的結構與樣式分離，因為 CSS 仍然放在 HTML 文件裡。如果樣式很多，HTML 檔案會變長；如果其他頁面也需要同樣樣式，還是得重複複製。

## 和行內樣式的差異

| 比較項目 | 行內樣式表 | 內部樣式表 |
| --- | --- | --- |
| 寫在哪裡 | HTML 標籤的 `style` 屬性 | HTML 文件的 `<style>` 標籤 |
| 作用範圍 | 目前標籤 | 目前頁面中符合選擇器的元素 |
| 複用性 | 低 | 比行內樣式高 |
| 適合情境 | 少量單一元素樣式 | 練習、示範、單頁樣式 |

內部樣式表比行內樣式更適合練習 CSS，因為它會讓你開始使用選擇器管理樣式。

## 常見誤解

### 誤解：`<style>` 可以隨便放，能動就好

為了讓文件結構清楚，建議把 `<style>` 放在 `<head>` 裡。這也是教學和正式文件中比較穩定的寫法。

### 誤解：內部樣式表已經完全分離結構與樣式

內部樣式表只是把 CSS 從各個標籤中集中到 `<style>` 裡，並沒有把 CSS 從 HTML 檔案中獨立出去。真正更完整的分離通常要使用外部樣式表。

### 誤解：內部樣式表可以讓多個 HTML 頁面共用樣式

內部樣式表只存在於目前這份 HTML 文件中。其他頁面不能直接共用這段樣式，除非複製貼上，但那會增加維護成本。

## 重點整理

- 內部樣式表把 CSS 寫在 HTML 文件中的 `<style>` 標籤裡。
- `<style>` 標籤建議放在 `<head>` 裡。
- 內部樣式表通常用來控制目前頁面的元素樣式。
- 它比行內樣式更清楚，也較容易集中管理。
- 它仍然沒有讓 CSS 完全獨立於 HTML，樣式多或多頁共用時應考慮外部樣式表。

# font-size：設定文字大小

## 學習目標

讀完這篇筆記，你應該能夠：

- 說明 `font-size` 控制的是文字顯示大小。
- 正確使用 `px`、`rem`、百分比等帶單位的字號值。
- 理解為什麼不應完全依賴瀏覽器預設字號。
- 知道設定 `body` 字號與單獨設定標題字號的差異。

## 使用情境

當你希望正文容易閱讀、標題有清楚層級、按鈕文字不會太小，或整個頁面有一致的基礎字號，就會使用 `font-size`。

字號不是單純的視覺裝飾，它會影響閱讀舒適度、版面密度與資訊層級。

## 一句話理解

`font-size` 用來設定文字的大小；非 `0` 的長度值必須帶單位。

## 基本語法

```css
body {
  font-size: 16px;
}
```

`16px` 是網頁中很常見的正文基礎字號。Chrome 等瀏覽器的預設文字大小通常也是 `16px`，但不同瀏覽器、使用者設定或元素預設樣式可能不同，因此正式頁面常會明確指定基礎字號。

除了 `px`，`font-size` 也可以使用其他型別的值：

```css
.intro {
  font-size: 1rem;
}

.note {
  font-size: 90%;
}
```

## 單位注意事項

使用數值長度時，非 `0` 值需要帶單位：

```css
/* 正確 */
p {
  font-size: 16px;
}

/* 錯誤 */
p {
  font-size: 16;
}
```

`font-size: 16;` 沒有單位，瀏覽器會把這條宣告視為無效。

## 設定整頁字號

可以在 `body` 設定整個頁面的基礎文字大小：

```css
body {
  font-size: 16px;
}
```

大多數普通文字會繼承 `body` 的字號。不過要注意，標題元素如 `h1`、`h2`、`h3` 本身有瀏覽器預設樣式，通常會比正文大。如果你希望標題也符合某個設計規格，需要單獨指定。

```css
body {
  font-size: 16px;
}

h2 {
  font-size: 16px;
}
```

這裡的 `h2` 會被明確設定為 `16px`，而不是使用瀏覽器預設的標題大小。

## 範例拆解

```html
<h2>pink的秘密</h2>
<p>那一抹众人中最漂亮的颜色,</p>
<p>优雅,淡然,又那么心中清澈.</p>
<p>前端总是伴随着困难和犯错,</p>
<p>静心,坦然,攻克一个又一个.</p>
<p>拼死也要克服它,</p>
<p>这是pink的秘密也是老师最终的嘱托.</p>
```

```css
body {
  font-size: 16px;
}

h2 {
  font-size: 16px;
}
```

- `body { font-size: 16px; }`：設定頁面基礎文字大小。
- `<p>`：通常會繼承 `body` 的 `16px`。
- `h2 { font-size: 16px; }`：覆蓋瀏覽器對 `h2` 的預設標題大小。

這個範例的重點是：設定 `body` 可以建立基礎，但不是所有元素的最終顯示都只由 `body` 決定。

## 常見錯誤

### 非 0 數值漏掉單位

```css
p {
  font-size: 16;
}
```

CSS 不知道這個 `16` 是像素、rem 還是其他單位，因此宣告無效。應改成：

```css
p {
  font-size: 16px;
}
```

### 完全依賴瀏覽器預設字號

不同瀏覽器、使用者設定或元素預設樣式可能造成顯示差異。正式版面通常應設定明確的基礎字號，再依需求調整標題與局部文字。

### 誤以為設定 `body` 後標題一定同大小

標題標籤有自己的預設樣式，若設計要求標題與正文同大小，必須另外指定標題的 `font-size`。

## 實務判斷準則

- 正文基礎大小常見起點是 `16px`。
- 頁面整體基準可放在 `body`。
- 標題、註解、小字、按鈕等需要根據設計目的另外設定。
- 不要只用字號大小表達所有層級，也要搭配字重、間距與版面結構。

## 重點整理

- `font-size` 控制文字大小。
- 非 `0` 的長度值需要單位，例如 `16px`、`1rem`。
- Chrome 等瀏覽器常見預設字號是 `16px`，但不要完全依賴預設。
- `body` 可以設定整頁基礎字號。
- 標題標籤通常需要依設計單獨指定大小。

# calc() 函數：在 CSS 屬性值中進行計算

## 學習目標

讀完這篇筆記後，你應該能夠：

- 說明 `calc()` 在 CSS 中的用途。
- 使用 `calc()` 混合百分比與固定長度進行尺寸計算。
- 理解 `+`、`-` 前後需要保留空白的語法要求。
- 判斷 `calc()` 的計算結果是否符合屬性可接受的值型別。

## 問題情境

有些版面尺寸不能只用固定 `px`，也不能只用單純的 `%`。

例如：你希望子盒子的寬度永遠跟父盒子有關，但又要比父盒子少 `30px`。如果只寫 `width: 100%`，子盒子會跟父盒子一樣寬；如果只寫固定寬度，又無法隨父盒子變化。

這種「相對尺寸再扣掉固定距離」的需求，就很適合使用 `calc()`。

## 一句話理解

`calc()` 讓你在 CSS 屬性值中直接寫計算式，讓瀏覽器幫你算出最後的值。

## calc() 是什麼

`calc()` 是 CSS 函數，可以在宣告 CSS 屬性值時執行一些計算。

```css
width: calc(100% - 80px);
```

這個值的意思是：寬度等於可用寬度的 `100%`，再減去 `80px`。

`calc()` 常見用途是混合不同單位，例如：

- `%` 搭配 `px`。
- `100vw` 搭配固定邊距。
- `rem` 搭配 `px`。

它的重點不是取代所有尺寸設定，而是用來描述「需要計算才能得到的 CSS 值」。

## 基本語法

```css
selector {
  width: calc(100% - 30px);
}
```

`calc()` 可以使用下列運算符：

| 運算符 | 用途 |
| --- | --- |
| `+` | 加法 |
| `-` | 減法 |
| `*` | 乘法 |
| `/` | 除法 |

使用時要注意：`+` 和 `-` 前後需要保留空白。

```css
/* 正確 */
width: calc(100% - 30px);

/* 錯誤 */
width: calc(100%-30px);
```

`calc(100%-30px)` 少了空白，瀏覽器可能無法正確解析這個計算式。

## 範例需求

需求：子盒子寬度永遠比父盒子小 `30px`。

HTML 結構如下：

```html
<body>
  <div class="father">
    <div class="son"></div>
  </div>
</body>
```

CSS 可以這樣寫：

```css
.father {
  width: 300px;
  height: 200px;
  background-color: pink;
}

.son {
  width: calc(100% - 30px);
  margin: 0 auto;
  height: 30px;
  background-color: skyblue;
}
```

## 範例拆解

```css
.father {
  width: 300px;
}
```

父盒子的寬度是 `300px`。

```css
.son {
  width: calc(100% - 30px);
}
```

`.son` 的 `100%` 會以父盒子的內容寬度為基準，也就是以 `300px` 為基準，再減去 `30px`。

所以這裡的計算結果是：

```text
300px - 30px = 270px
```

最後 `.son` 的寬度會是 `270px`。

```css
.son {
  margin: 0 auto;
}
```

`margin: 0 auto` 讓這個有明確寬度的 block 元素在父盒子中水平置中。因此畫面效果會是：子盒子比父盒子窄 `30px`，並且左右置中。

## 值型別要符合屬性需求

`calc()` 的結果仍然必須符合該 CSS 屬性可接受的值型別。

例如 `width` 需要的是長度、百分比或相關尺寸值，所以這樣是合理的：

```css
width: calc(100% - 30px);
```

但如果某個屬性不接受計算後的型別，即使語法看起來像計算式，最後也不會是有效宣告。

理解 `calc()` 時可以記住：它只是幫你算值，不會讓屬性接受原本不支援的值。

## 常見錯誤

### 加減號前後漏掉空白

```css
width: calc(100%-30px);
```

這是很常見的錯誤。`+` 和 `-` 前後要保留空白，應改成：

```css
width: calc(100% - 30px);
```

### 誤解 100% 的參考對象

在 `width: calc(100% - 30px);` 中，`100%` 通常不是指整個瀏覽器視窗，而是依該屬性的百分比規則去參考對應的包含區塊。

在本篇範例裡，`.son` 的 `100%` 是以父盒子 `.father` 的內容寬度為基準。

### 以為 calc() 可以用在任何值上

`calc()` 的結果仍要符合屬性本身支援的值型別。不要把它理解成「任何屬性都可以任意計算」。

例如尺寸相關屬性很常搭配 `calc()`，但使用前仍要確認該屬性接受什麼型別的值。

## 實務判斷準則

- 需要混合相對單位與固定單位時，可以考慮 `calc()`。
- 常見情境是「滿版寬度扣掉固定間距」或「百分比寬度扣掉固定尺寸」。
- 寫 `+`、`-` 時，前後保留空白。
- 計算結果要符合屬性可接受的值型別。
- 若只是單純固定尺寸或單純百分比，沒有必要為了使用函數而使用 `calc()`。

## 重點整理

- `calc()` 可以在 CSS 屬性值中進行計算。
- `calc()` 可以使用 `+`、`-`、`*`、`/`。
- `+` 和 `-` 前後需要保留空白。
- `width: calc(100% - 30px);` 可表示「以父層可用寬度為基準，再減去 `30px`」。
- `calc()` 的結果仍必須符合屬性可接受的值型別。

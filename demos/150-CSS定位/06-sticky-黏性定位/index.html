position: sticky

先像 relative，再像 fixed

sticky 需要捲動容器與 top / left / right / bottom 門檻；沒有門檻時不會黏住。

往下捲動這個框
章節導覽 sticky

內容段落 1：sticky 一開始仍在文件流中。

內容段落 2：當它碰到指定門檻，就會黏在捲動容器內。

內容段落 3：它不是永遠固定，仍受父層與捲動範圍限制。

內容段落 4：最常見用法是表頭、側邊目錄、區塊標題。

內容段落 5：若忘記 top 等偏移值，sticky 不會啟動。

設定 top: 0

.nav {
  position: sticky;
  top: 0;
}

捲動到導覽列碰到容器頂端時，它會暫時黏住。

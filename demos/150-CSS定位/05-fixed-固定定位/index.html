position: fixed

相對視窗固定

fixed 元素不隨內容捲動，通常用於浮動工具、固定導覽或貼在內容版心旁的元件。

內容區塊 1：捲動畫面時，fixed 工具仍貼著視窗。

內容區塊 2：它不佔文件流空間，所以內容可能被覆蓋。

內容區塊 3：需要時要替內容補 padding 或 margin。

內容區塊 4：常見在返回頂部、客服、固定選單。

內容區塊 5：版心旁固定可用 left: 50% 加 margin-left。

FIXED

右下角固定工具

.tool {
  position: fixed;
  right: 24px;
  bottom: 24px;
}

right 與 bottom 很適合建立貼在視窗角落的固定工具。

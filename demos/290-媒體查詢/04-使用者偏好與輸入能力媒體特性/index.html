Preference and input features

媒體查詢也可以尊重使用者與輸入設備

此頁會依系統深淺色、減少動畫偏好、hover 能力與 pointer 精確度調整呈現。

主題偏好

`prefers-color-scheme` 讓顏色跟著系統偏好切換。

減少動畫

`prefers-reduced-motion: reduce` 時，動畫會停用或大幅簡化。

在 `pointer: coarse` 的環境中，按鈕點擊區域會變大；支援 hover 時才提供 hover 視覺回饋。

未匹配 `prefers-color-scheme: dark`
未匹配 `prefers-reduced-motion: reduce`
未匹配 `hover: hover`
未匹配 `any-hover: hover`
未匹配 `pointer: fine`
未匹配 `pointer: coarse`

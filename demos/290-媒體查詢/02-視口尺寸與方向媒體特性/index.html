Viewport media features

媒體查詢看的是視口，不是設備名稱

縮放視窗或改變比例，觀察寬度、高度與方向條件如何影響版面。

寬度 -- px
高度 -- px
方向 --
工具列 橫向且寬度不超過 900px 時固定在上方

主要內容

在窄視口中維持單欄；空間足夠時改為雙欄。這是用 `min-width` 判斷可用寬度，而不是猜測設備。

輔助內容

高度不足時，區塊會縮小垂直間距，讓內容仍然能快速掃描。

未匹配 `@media (min-width: 900px)`
未匹配 `@media (max-height: 560px)`
未匹配 `@media (orientation: landscape)`
未匹配 `@media (orientation: landscape) and (max-width: 900px)`

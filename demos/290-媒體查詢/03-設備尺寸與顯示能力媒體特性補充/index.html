`aspect-ratio`

目前比例讀取中。

寬螢幕時改成橫向組合；偏窄比例時改成直向堆疊。這比 `device-aspect-ratio` 更貼近目前可用空間。

未匹配 `@media (min-aspect-ratio: 16/10)`

寬比例布局。

未匹配 `@media (max-aspect-ratio: 1/1)`

窄比例或直向布局。

未匹配 `@media (min-resolution: 2dppx)`

常用於高解析度資源。

未匹配 `@media (color)`

設備支援顏色顯示。

未匹配 `@media (monochrome)`

單色或灰階顯示設備。

未匹配 `@media (grid)`

這不是 CSS Grid Layout。

`device-*` 特性

`device-width`、`device-height`、`device-aspect-ratio` 主要用於理解舊程式碼；新專案版面斷點優先使用視口相關特性。

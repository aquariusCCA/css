CSS animation demo

animation 概念與 keyframes 基礎

觀察 transition 的觸發式變化，和 animation 依照 keyframes 自動播放多個節點的差異。

transition：狀態改變後才過渡

hover / focus

移入或聚焦時，方塊只在起點與終點之間變化。

animation：keyframes 決定播放序列

0% 25% 50% 75% 100%

定義 keyframes 後，還要呼叫動畫

只有 @keyframes

@keyframes move { ... }

沒有套用到元素，所以畫面不會播放。

缺少 duration

animation-name: move;

duration 預設是 0s，看不到播放過程。

正確呼叫

animation: move 2s;

名稱與非 0s 時長都存在，動畫才可見。

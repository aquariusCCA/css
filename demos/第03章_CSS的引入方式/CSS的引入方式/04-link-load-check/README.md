# Demo 04：外部 CSS 載入檢查

來源筆記：`notes/第03章_CSS的引入方式/CSS的引入方式.md`

對應知識點：外部樣式表多了一個載入步驟；若畫面沒有變，除了檢查 selector 與 declaration，也要檢查 `<link>`、`rel` 與 `href`。

開啟方式：直接用瀏覽器開啟 `index.html`。

操作方式：不需要操作，觀察左右兩個載入狀態。

觀察重點：左側呈現 CSS 沒有成功進來時的樣子；右側呈現外部 CSS 成功套用後的樣子。頁面底部列出外部樣式表的基本檢查順序。

常見誤解：外部 CSS 沒效果不一定是 CSS 規則寫錯，也可能是 HTML 沒有正確載入 CSS 檔案。

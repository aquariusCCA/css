Demo 04

外部 CSS 沒效果時，先檢查 link

左側模擬 CSS 沒有載入時的結果；右側模擬 rel 與 href 正確時的結果。這個 demo 不真的引用不存在的檔案。

載入失敗的結果

外部樣式沒有進來

畫面只剩一般文字，像是 CSS 規則完全沒有套用。

<link rel="stylesheet" href="./wrong-path.css">

載入成功的結果

外部樣式成功載入

同樣的內容，因為 CSS 有進來，所以出現清楚的視覺樣式。

<link rel="stylesheet" href="./style.css">

檢查順序

  1. HTML 裡是否有 <link>。
  2. rel 是否是 stylesheet。
  3. href 是否指向真正存在的 CSS 檔案。

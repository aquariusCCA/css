Demo 02

內部樣式表集中控制本頁元素

<style> 寫在這份 HTML 的 <head> 裡。只要元素符合 selector，就會套用同一條內部 CSS 規則。

沒有符合內部 selector 的區塊

第一個 div：沒有被本 demo 的內部規則選到。
第二個 div：仍維持一般文字樣式。

同一頁中符合 selector 的元素

第一個 div：被內部樣式表選到。
第二個 div：同樣被內部樣式表選到。

寫在 index.html 的內部樣式表

<style>
    .internal-style-demo .target {
        color: #be185d;
        font-size: 20px;
        font-weight: 700;
    }
</style>

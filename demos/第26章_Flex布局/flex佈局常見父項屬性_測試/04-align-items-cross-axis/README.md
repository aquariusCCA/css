# align-items 控制單行側軸對齊

來源筆記：`notes/第26章_Flex布局/flex佈局常見父項屬性_測試.md`

目標知識點：`align-items` 控制單行 flex items 在側軸上的對齊。

## 開啟方式

直接用瀏覽器開啟 `index.html`。

## 觀察重點

- `flex-start` 讓 items 靠側軸起點。
- `center` 讓 items 在側軸置中。
- `stretch` 在子元素沒有固定側軸尺寸時才明顯。

## 常見誤解

`align-items` 適合處理單行 items。若畫面已經是多行，且要控制多行整體分配，應看 `align-content`。

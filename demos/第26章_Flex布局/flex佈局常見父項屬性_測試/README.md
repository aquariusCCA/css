# Flex 佈局常見父項屬性 Demo

來源筆記：`notes/第26章_Flex布局/flex佈局常見父項屬性_測試.md`

目標範圍：`flex佈局常見父項屬性`

本目錄根據來源筆記拆成多個小型 demo。每個 demo 聚焦一個主要觀念，避免把整章內容塞進同一個範例。

## Demo 清單

| Demo | 主要觀念 |
|---|---|
| `01-flex-container-role/` | `display: flex` 必須寫在父元素上，直接子元素才會成為 flex items。 |
| `02-flex-direction-axis/` | `flex-direction` 會改變主軸方向，進而影響排列方向。 |
| `03-justify-content-main-axis/` | `justify-content` 控制主軸上的剩餘空間。 |
| `04-align-items-cross-axis/` | `align-items` 控制單行 items 在側軸上的對齊。 |
| `05-flex-wrap-lines/` | `flex-wrap` 決定空間不足時是否允許換行。 |
| `06-align-content-multiple-lines/` | `align-content` 控制多行 flex lines 在側軸上的分配。 |
| `07-flex-flow-shorthand/` | `flex-flow` 同時設定 `flex-direction` 與 `flex-wrap`。 |

## 開啟方式

直接用瀏覽器開啟各 demo 目錄下的 `index.html`。

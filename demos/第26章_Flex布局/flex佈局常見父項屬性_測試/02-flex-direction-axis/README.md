# flex-direction 改變主軸

來源筆記：`notes/第26章_Flex布局/flex佈局常見父項屬性_測試.md`

目標知識點：`flex-direction` 決定主軸方向。

## 開啟方式

直接用瀏覽器開啟 `index.html`。

## 觀察重點

- `row` 讓子元素沿水平方向排列。
- `column` 讓子元素沿垂直方向排列。
- 主軸改變後，`justify-content` 與 `align-items` 的方向判斷也會改變。

## 常見誤解

不要把 `justify-content` 固定理解成水平對齊，也不要把 `align-items` 固定理解成垂直對齊。先看主軸。

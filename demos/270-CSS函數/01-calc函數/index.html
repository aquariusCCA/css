calc() 函數：用 CSS 算出尺寸

調整父盒寬度與扣除值，觀察 width: calc(100% - 固定長度) 如何讓子盒跟著父盒變化。

混合單位

100% 先看父層，再扣掉固定長度

width: calc(100% - 40px); 360px - 40px = 320px
父盒 360px
子盒 320px

語法空白

加減號前後少空白，宣告可能失效

正確寫法

.son {
  width: calc(100% - 30px);
}
270px

父盒是 300px，子盒寬度會被算成 270px，左右各留出 15px。

錯誤寫法

.son {
  width: calc(100%-30px);
}
無效後回到 auto

少了空白時，這個寬度宣告無法正確解析，block 子盒會退回自動寬度。

值型別

calc() 只負責算值，不改變屬性規則

有效：屬性接受長度或百分比

width: calc(100% - 30px);
有效寬度

無效：屬性不接受這種結果

background-color: calc(100% - 30px);
宣告被忽略

基礎選擇器

類選擇器只改有指定 class 的元素

同樣是一組 li，標籤選擇器會全選，類選擇器可以只選有 class="important" 的項目。

選太廣：li

  • 冰雨 class="important"
  • 來生緣 class="important"
  • 李香蘭
  • 江南 style
li {
  color: red;
}

選精準：.important

  • 冰雨 class="important"
  • 來生緣 class="important"
  • 李香蘭
  • 江南 style
.important {
  color: red;
}

主要 CSS 規則

.important 前面的句點表示 CSS 要找的是 class 名稱。沒有這個 class 的清單項目不會被選到。

# 基礎選擇器 Demos

來源筆記：`notes/第06章_選擇器/01-基礎選擇器.md`

本目錄包含 5 個基礎選擇器 demo。每個 demo 都可以直接用瀏覽器開啟該子目錄中的 `index.html`。

| Demo | 主要知識點 | 入口 |
|---|---|---|
| 標籤選擇器 | `p` 會選到所有 `<p>` 元素 | `tag-selector/index.html` |
| 類選擇器 | `.important` 只選有指定 class 的元素 | `class-selector/index.html` |
| 多類名 | `class` 中多個類名要用空格分隔 | `multiple-classes/index.html` |
| id 選擇器 | `#main-title` 對應 `id="main-title"` | `id-selector/index.html` |
| 通配符選擇器 | `*` 會影響所有元素，容易過度選取 | `universal-selector/index.html` |

每個子目錄都有自己的 `README.md`，說明觀察重點與常見誤解。

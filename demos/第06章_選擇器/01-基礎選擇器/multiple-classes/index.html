基礎選擇器

多個 class 要用空格分隔

觀察同一個元素如何同時被 .red 與 .large-text 選到，以及逗號為什麼會造成漏選。

錯誤：逗號分隔

重要提醒

<p class="red, large-text">重要提醒</p>

正確：空格分隔

重要提醒

<p class="red large-text">重要提醒</p>

主要 CSS 規則

.red {
  color: red;
}

.large-text {
  font-size: 32px;
}

正確寫法會讓同一個元素同時符合兩個 class；逗號會變成 class 名稱的一部分，導致 .red 選不到。

# 交集選擇器 Demo

來源筆記：`notes/第06章_選擇器/02-複合選擇器.md`

對應知識點：交集選擇器 `p.box` 選同一個元素同時符合 `p` 與 `.box`。

開啟方式：直接用瀏覽器開啟 `index.html`。

觀察重點：

- 左側 `p .box` 有空格，選的是 `p` 裡面的 `.box` 後代。
- 右側 `p.box` 沒空格，選的是同一個元素同時是 `p` 且有 `box` class。
- `<div class="box">` 不會被 `p.box` 選到，因為它不是 `p`。

常見誤解：交集選擇器中間不能加空格；加了空格就變成後代選擇器。

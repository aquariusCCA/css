transition

讓狀態切換變成一段平滑變化

`transition` 不決定終點樣式；它決定元素從舊狀態走到新狀態時，要花多久、用什麼節奏完成。

狀態變化

沒有過渡會跳變，有過渡會平滑變化

沒有 transition

立即改變
.box:hover {
  width: 220px;
  background-color: tomato;
}

有 transition

0.5s 過渡
.box {
  transition: width 0.5s, background-color 0.5s;
}

語法欄位

調整時間、運動曲線與延遲

.box {
  transition: left 500ms ease 0s,
              background-color 500ms ease 0s;
}

運動曲線

開始延遲

常見錯誤

`transition` 通常寫在一般狀態

只寫在變化後狀態

離開時跳回
.box.active {
  width: 220px;
  transition: width 0.5s;
}

寫在一般狀態

來回都平滑
.box {
  transition: width 0.5s;
}
.box.active {
  width: 220px;
}

指定屬性

`all` 方便，但明確列出屬性更可控

transition: all

額外變化也被過渡
transition: all 0.5s;

指定 width 與 background-color

只過渡指定屬性
transition: width 0.5s,
            background-color 0.5s;

linear-gradient 漸層與圖片遮罩

觀察 `linear-gradient()` 如何成為背景圖像，並用在圖片下方遮罩來提升白色文字的可讀性。

background-image

漸層本身就是一張 CSS 圖像

預設方向

linear-gradient(transparent, rgba(0, 0, 0, .6))

沒有指定方向時，漸層由上往下變化，適合讓圖片下方逐漸變暗。

指定角度

linear-gradient(135deg, #fff7ad, #4f8f8b)

第一個參數可以指定角度，後面的色標決定顏色如何平滑過渡。

010-CSS簡介

CSS 與 HTML 的分工

HTML 結構

<article>
    <h1>我的第一個網頁</h1>
    <p>HTML 描述內容結構。</p>
    <ul>
        <li>標題</li>
        <li>段落</li>
        <li>清單</li>
    </ul>
</article>

呈現結果

我的第一個網頁

HTML 描述內容結構，CSS 決定這些內容如何顯示。

  • 標題保留語意角色
  • 段落保留文字內容
  • 清單保留資料關係

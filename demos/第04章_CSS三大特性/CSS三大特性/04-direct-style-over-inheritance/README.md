# Demo 04：直接樣式優先於繼承值

來源筆記：`notes/第04章_CSS三大特性/CSS三大特性.md`

對應知識點：`a` 的顏色與 `h1` 的字級看起來沒有使用父元素樣式，通常不是繼承機制失效，而是元素自己已有直接套用的預設樣式。

開啟方式：直接用瀏覽器開啟 `index.html`。

操作方式：不需要操作，觀察左右兩組內容。

觀察重點：左側只設定父元素，連結仍可能維持瀏覽器預設連結色，標題也仍然比 12px 大；右側直接選到 `a` 與 `h1` 後，才明確使用紅色與 12px。

常見誤解：不要把所有「看起來沒有繼承」都判斷成元素不能繼承；應先檢查目標元素是否已有直接套用的樣式。

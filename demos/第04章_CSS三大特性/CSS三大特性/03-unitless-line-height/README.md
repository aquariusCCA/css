# Demo 03：無單位 line-height 的繼承

來源筆記：`notes/第04章_CSS三大特性/CSS三大特性.md`

對應知識點：無單位 `line-height` 繼承時保留比例，子元素會用自己的 `font-size` 重新計算實際行高。

開啟方式：直接用瀏覽器開啟 `index.html`。

操作方式：不需要操作，觀察兩段不同字級文字的行距。

觀察重點：父層只設定 `line-height: 1.5;`。小字段落以 `14px * 1.5` 形成行高，大字段落以 `24px * 1.5` 形成行高，因此兩者都維持相同閱讀比例。

常見誤解：無單位 `line-height` 不是固定像素值；它在子元素上會跟著子元素自己的字級重新計算。

Demo 03

無單位 line-height 繼承的是比例

父層只設定一次 line-height: 1.5;，不同字級的子元素會用自己的字級計算行高。

畫面結果

14px 文字

第一行文字用來觀察行距。

第二行仍然使用同一個比例。

24px 文字

第一行文字用來觀察行距。

第二行也會依 24px 重新計算。

實際 CSS

.type-parent {
    line-height: 1.5;
}

.small-copy {
    font-size: 14px;
}

.large-copy {
    font-size: 24px;
}
14px * 1.5 = 21px 24px * 1.5 = 36px

# Demo 01：同權重衝突與來源順序

來源筆記：`notes/第04章_CSS三大特性/CSS三大特性.md`

對應知識點：層疊性處理同一元素、同一屬性發生衝突時的取捨；權重相同時，後寫的衝突宣告會勝出。

開啟方式：直接用瀏覽器開啟 `index.html`。

操作方式：不需要操作，觀察畫面上的文字顏色與字級，再對照 CSS。

觀察重點：文字最後是粉紅色，因為第二段相同 selector 的 `color` 寫在後面；但 `font-size: 12px;` 沒有被重新設定，所以仍然保留。

常見誤解：後面的整段規則不會把前面的整段規則清掉，只有同一元素、同一屬性的衝突值需要決定勝負。

# Demo 06：複合選擇器權重疊加但不進位

來源筆記：`notes/第04章_CSS三大特性/CSS三大特性.md`

對應知識點：複合選擇器的 specificity 會依 selector 類型分組累加，並由左到右比較；標籤選擇器累加再多，也不會進位成類選擇器。

開啟方式：直接用瀏覽器開啟 `index.html`。

操作方式：不需要操作，觀察清單文字顏色與三條 CSS 的權重。

觀察重點：`.nav li`、`ul li`、`li` 都選到同一批清單項目。即使 `.nav li` 寫在最前面，最後仍是粉紅色，因為它包含一個類選擇器。

常見誤解：selector 寫得比較長不一定比較強；要看各類 selector 分別累加後的位置。

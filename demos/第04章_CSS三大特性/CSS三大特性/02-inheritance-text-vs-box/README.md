# Demo 02：文字繼承與盒模型不繼承

來源筆記：`notes/第04章_CSS三大特性/CSS三大特性.md`

對應知識點：子元素沒有自己設定可繼承屬性時，會沿用父元素的文字樣式；`border`、`padding`、`margin` 這類盒模型屬性通常不會自動繼承。

開啟方式：直接用瀏覽器開啟 `index.html`。

操作方式：不需要操作，觀察父元素外框與子元素文字。

觀察重點：子元素文字繼承了父元素的顏色與字級，但子元素本身沒有多一圈藍色邊框，也沒有自動取得父元素的內距。

常見誤解：父元素有某個 CSS 屬性，不代表子元素一定會得到同樣屬性；要先判斷該屬性是否通常可繼承。

字體圖標的用途與特性

用簡化 glyph 觀察字體圖標如何被 CSS 當成文字控制，以及哪些情境適合或不適合使用。

Observation 01

同一個圖標，不同 CSS 狀態

.font-icon {
  color: currentColor;
  font-size: 24px;
  opacity: 0.88;
  text-shadow: none;
  transform: rotate(0deg);
}

Observation 02

適合與不適合的圖像需求

適合字體圖標

簡單、單色、尺寸小、重複使用，且常需要跟著介面狀態調整顏色或大小。

不適合字體圖標

照片、複雜插圖、多色細節與材質紋理，通常應改用圖片、SVG 或更合適的格式。

Observation 03

沒有 glyph 對應時，空元素不會自己變成圖標

錯誤寫法

<span class="icon-cart"></span>

修正寫法

.icon-cart::before {
  content: "□";
}

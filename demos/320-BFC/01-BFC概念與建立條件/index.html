Block Formatting Context

BFC 概念與建立條件

BFC 是一個獨立的區塊格式化環境。這個 demo 用同一組元素觀察「同一個環境」與「建立新環境」後的佈局差異。

觀察一

同一個 BFC 裡，block 會垂直排列，垂直 margin 可能重疊

Block A
margin-bottom: 36px
相鄰垂直 margin 會取較大值，不是簡單相加
Block B
margin-top: 36px
.block-a { margin-bottom: 36px; }
.block-b { margin-top: 36px; }
/* 同一個 BFC 中，相鄰垂直 margin 可能 collapse */

觀察二

切換建立 BFC 的條件

float child

下一個普通區塊

目前狀態

父容器沒有建立新的 BFC

float 子元素不會撐開父容器高度，邊框看起來像貼在頂部。

.container {
  border: 4px solid #263238;
}

觀察三

flow-root 與 overflow: hidden 都能建立 BFC，但副作用不同

display: flow-root

溢出標籤

建立 BFC，但不裁切超出邊界的內容。

.box {
  display: flow-root;
}

overflow: hidden

溢出標籤

也會建立 BFC，但超出邊界的內容會被裁切。

.box {
  overflow: hidden;
}

二倍圖：檔案變大，版面不變

二倍圖提供更多實際像素細節，但仍要用 CSS 把畫面尺寸設回設計稿需要的大小。

目前二倍圖被縮回 50px x 50px，版面尺寸符合設計稿。

原圖

圖片檔：50 x 50

顯示區也是 50px x 50px，普通螢幕足夠，高 DPR 螢幕可能細節不足。

正確使用二倍圖

圖片檔：100 x 100；顯示：50 x 50

圖片有 2 倍實際像素，CSS 顯示尺寸仍維持設計稿大小。

常見錯誤

未設定 CSS 寬高：100 x 100

只換大圖卻沒有設定顯示尺寸，圖片會按原始尺寸撐大版面。

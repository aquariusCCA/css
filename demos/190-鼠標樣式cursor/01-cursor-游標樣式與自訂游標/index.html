CSS cursor

游標樣式與互動語意

把滑鼠移到不同區塊上，觀察游標形狀如何傳達可點擊、可拖曳、可輸入或不可操作等狀態。

系統預設游標

每張卡片都套用不同的 cursor 值。

default 一般指向

預設箭頭游標，適合普通內容。

pointer 可點擊

常見於連結、按鈕等可操作項目。

move 可移動

暗示元素可以拖曳或重新排列。

text 文字操作

用於可選取或輸入文字的區域。

not-allowed 不可操作

提示目前狀態不允許執行動作。

help 取得說明

適合帶有補充說明或提示的項目。

wait 等待中

表示系統正在載入或處理。

自訂游標與 fallback

自訂圖片後面仍要接一個系統預設游標，讓圖片無法使用時有備援行為。

url(...), default 自訂游標區

移入這個區塊時，瀏覽器會嘗試使用 demo 內的游標圖片；若圖片不可用，改用 default。

.custom-zone {
  cursor: url("./assets/images/cursor-style-img-002-81baa8.png"), default;
}

游標不會創造功能

cursor: pointer 只是視覺提示，元素是否真的能操作仍取決於實際功能。

cursor: pointer 假裝可點擊的文字

游標變成手形，但點擊後沒有實際行為，語意容易誤導。

cursor: pointer 狀態會同步改變

目前可操作，所以使用 pointer。

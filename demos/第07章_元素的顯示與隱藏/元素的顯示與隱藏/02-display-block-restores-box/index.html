元素的顯示與隱藏

display: block 讓元素重新以區塊方式顯示

比較 display: none 和 display: block，觀察被隱藏的盒子重新出現後如何推開下一個區塊。

隱藏狀態：display none

提示面板
下一個區塊
.demo-panel {
  display: none;
}

顯示狀態：display block

提示面板
下一個區塊
.demo-panel {
  display: block;
}

主要觀察

display: block 不是單純取消隱藏而已，它會讓元素以 block 盒子參與排版。右側粉色面板出現後，藍色區塊被排到它下方。

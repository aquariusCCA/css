Demo 04

日常先寫標準 CSS，需要時才補前綴

兩個按鈕的畫面目標相同。左側保留標準寫法；右側加入前綴作為舊環境相容補充。將游標移到按鈕上，觀察狀態變化。

日常主線

.standard-button {
    border-radius: 10px;
    transition: background-color .2s, transform .2s;
}

有舊版支援需求時

.compat-button {
    border-radius: 10px;
    -webkit-transition: background-color .2s, transform .2s;
    transition: background-color .2s, transform .2s;
}

檢查順序

  1. 先確認標準屬性是否能滿足目標瀏覽器。
  2. 只有舊版支援或實驗功能需求成立時，再補前綴。
  3. 補前綴時，讓標準屬性保留在最後。

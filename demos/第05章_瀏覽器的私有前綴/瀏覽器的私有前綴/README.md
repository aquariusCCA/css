# 瀏覽器的私有前綴 Demo

來源筆記：`notes/第05章_瀏覽器的私有前綴/瀏覽器的私有前綴.md`

本目錄將「瀏覽器的私有前綴」拆成四個單一概念 demo。每個 demo 都可以直接用瀏覽器開啟該目錄中的 `index.html`。

| Demo | 觀察重點 |
|---|---|
| `01-prefixed-property-name` | 私有前綴是 property name 的一部分，不是 selector 或 value |
| `02-standard-last-order` | 需要前綴時，前綴版本放前面，標準屬性放最後 |
| `03-prefix-reading-map` | 先讀前綴代表的瀏覽器家族，再讀後面的 CSS 功能 |
| `04-standard-first-workflow` | 日常先寫標準 CSS，有舊版支援需求時才補前綴 |

這些 demo 不使用框架、外部 CDN、建置工具或 JavaScript。

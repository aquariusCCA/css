# Demo 01：私有前綴是屬性名稱的一部分

來源筆記：`notes/第05章_瀏覽器的私有前綴/瀏覽器的私有前綴.md`

對應知識點：`-webkit-`、`-moz-`、`-ms-`、`-o-` 這類前綴是 property name 的一部分，用來標示特定瀏覽器廠商曾經提供的前綴版本。

開啟方式：直接用瀏覽器開啟 `index.html`。

操作方式：不需要操作，觀察左右兩張卡片的圓角與下方 CSS 對照。

觀察重點：右側多了前綴版本，但它們仍然在服務同一個圓角功能；最後的 `border-radius` 是標準屬性。

常見誤解：不要把前綴當成 selector 或 value。它是屬性名稱的一部分，不代表新的 CSS 類型。

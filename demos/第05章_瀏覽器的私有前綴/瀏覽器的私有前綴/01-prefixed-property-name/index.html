前綴不是 value，而是 property name 的一部分

左右兩張卡片都在做同一件事：讓盒子出現圓角。右側多出的前綴寫法只是相容候選，標準屬性仍是主線。

標準屬性

border-radius 標準寫法 
.standard-card {
    border-radius: 24px;
}

前綴版本 + 標準屬性

prefix + standard 相容補充 
.prefixed-card {
    -moz-border-radius: 24px;
    -webkit-border-radius: 24px;
    -o-border-radius: 24px;
    border-radius: 24px;
}

拆解觀察

-webkit-border-radius: 24px;

-webkit- 是前綴，border-radius 才是要控制的 CSS 功能。

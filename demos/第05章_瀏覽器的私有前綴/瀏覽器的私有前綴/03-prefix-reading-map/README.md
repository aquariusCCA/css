# Demo 03：讀懂常見私有前綴

來源筆記：`notes/第05章_瀏覽器的私有前綴/瀏覽器的私有前綴.md`

對應知識點：來源筆記列出 `-moz-`、`-ms-`、`-webkit-` 與 `-o-` 的入門對應關係。讀到前綴時，應先看前綴代表的瀏覽器家族，再看後面的 CSS 功能。

開啟方式：直接用瀏覽器開啟 `index.html`。

操作方式：不需要操作，觀察每張卡片中的前綴、property name 與畫面效果。

觀察重點：`-webkit-transform` 的主要功能仍然是 `transform`，`-moz-border-radius` 的主要功能仍然是 `border-radius`。前綴只是相容性標記。

常見誤解：不要看到 `-webkit-` 就以為那是一個全新的 CSS 功能，也不要把前綴清單當成完整瀏覽器歷史。

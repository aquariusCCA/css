CSS 引入方式

比較與選擇

根據作用範圍、複用性與維護性，選出適合目前情境的 CSS 引入方式。

行內樣式表

寫在 HTML 標籤的 style 屬性

作用範圍最小，適合少量、臨時、單一元素樣式。

內部樣式表

寫在 HTML 文件的 <style> 標籤

控制目前頁面，適合練習、示範與單頁樣式。

外部樣式表

寫在獨立的 .css 檔案

可被多個頁面共用，適合實際開發與長期維護。

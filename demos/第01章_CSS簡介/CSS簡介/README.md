# CSS 簡介 Demo

來源筆記：`notes/第01章_CSS簡介/CSS簡介.md`

本目錄將「CSS簡介」拆成四個單一概念 demo。每個 demo 都可以直接用瀏覽器開啟該目錄中的 `index.html`。

| Demo | 觀察重點 |
|---|---|
| `01-html-css-separation` | HTML 結構相同時，CSS 如何改變畫面外觀 |
| `02-selector-target` | selector 沒選到元素時，declaration 不會影響畫面 |
| `03-css-syntax-punctuation` | CSS 需要英文冒號與分號，錯誤標點會讓宣告失效 |
| `04-code-style-readability` | 展開風格與緊湊風格畫面相同，但維護性不同 |

這些 demo 不使用框架、外部 CDN、建置工具或 JavaScript。

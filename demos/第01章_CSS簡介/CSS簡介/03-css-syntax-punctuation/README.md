# CSS 語法標點

來源筆記：`notes/第01章_CSS簡介/CSS簡介.md`

對應知識點：property 和 value 之間使用英文冒號 `:`，一條 declaration 通常使用英文分號 `;` 結束。

開啟方式：直接用瀏覽器開啟 `index.html`。

觀察重點：

- 左側 `.bad-title` 使用全形 `：` 與 `；`，瀏覽器無法把它解析成有效 CSS 宣告。
- 右側 `.good-title` 使用英文 `:` 與 `;`，所以文字會變色並放大。
- 當樣式沒有生效時，除了 selector，也要檢查標點是否是 CSS 語法需要的英文符號。

常見誤解：中文輸入法下輸入的全形標點看起來相似，但不是 CSS 語法中的冒號與分號。

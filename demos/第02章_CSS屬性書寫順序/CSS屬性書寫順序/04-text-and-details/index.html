Demo 04

先確認文字，再看圓角、陰影與游標

同一個按鈕範例先建立盒子與文字，再加上其他細節。這能避免一開始就被裝飾效果干擾。

先看文本屬性

閱讀範例 
.text-only {
    display: inline-block;

    padding: 10px 16px;
    background: #2563eb;

    color: #ffffff;
    font-size: 16px;
    font-weight: 700;
    text-align: center;
    text-decoration: none;
}

最後看其他細節

閱讀範例 
.detailed {
    display: inline-block;

    padding: 10px 16px;
    background: #2563eb;

    color: #ffffff;
    font-size: 16px;
    font-weight: 700;
    text-align: center;
    text-decoration: none;

    border-radius: 6px;
    box-shadow: 0 2px 8px rgba(0, 0, 0, .18);
    cursor: pointer;
}

Demo 03

自身屬性先描述元素盒子

這個 demo 把 `width`、`margin`、`padding`、`border`、`background` 放在同一段觀察。它們一起描述元素自己的大小、內外距與可見範圍。

內距不足

內容貼邊

文字很靠近邊框，閱讀時會覺得擁擠。

.box-tight {
    display: block;

    width: 260px;
    margin: 0;
    padding: 0;
    border: 2px solid #64748b;
    background: #ffffff;
}

盒子條件清楚

內容有空間

外距拉開與外界的距離，內距拉開內容和邊框。

.box-spacious {
    display: block;

    width: 260px;
    margin: 18px;
    padding: 18px;
    border: 2px solid #0f766e;
    background: #ecfdf5;
}

# CSS 屬性書寫順序 Demo

來源筆記：`notes/第02章_CSS屬性書寫順序/CSS屬性書寫順序.md`

本目錄將「CSS 屬性書寫順序」拆成四個單一概念 demo。每個 demo 都可以直接用瀏覽器開啟該目錄中的 `index.html`。

| Demo | 觀察重點 |
|---|---|
| `01-order-readability` | 屬性重新排序通常不改變畫面，但會改善閱讀路線 |
| `02-display-first` | `display` 影響元素顯示模式，適合放在宣告區塊前段 |
| `03-box-properties` | `width`、`margin`、`padding`、`border`、`background` 一起描述元素盒子 |
| `04-text-and-details` | 先讀文字樣式，再讀圓角、陰影、游標等細節效果 |

這些 demo 不使用框架、外部 CDN、建置工具或 JavaScript。

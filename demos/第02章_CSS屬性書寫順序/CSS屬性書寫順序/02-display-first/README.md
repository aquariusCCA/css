# 先看 display

來源筆記：`notes/第02章_CSS屬性書寫順序/CSS屬性書寫順序.md`

對應知識點：`display` 關係到元素顯示模式，因此適合放在布局定位屬性的第一個位置觀察。

開啟方式：直接用瀏覽器開啟 `index.html`。

觀察重點：

- 左側的 `span` 保留預設 inline 行為，即使寫了 `width` 與 `height`，盒子尺寸也不如右側明確。
- 右側先寫 `display: inline-block;`，再寫寬高與背景，畫面上能清楚看到固定大小的盒子。
- 這說明閱讀 CSS 時應先確認顯示模式，再檢查尺寸、間距與外觀。

常見誤解：把「先寫 `display`」理解成排序本身會讓畫面改變。真正造成右側畫面差異的是 `display: inline-block;` 這條宣告；把它放前面是為了讓維護者更快看到顯示模式。

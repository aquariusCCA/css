Demo 02

先看 display，先確認元素的顯示模式

兩個標籤都寫了寬高。左側保留 span 預設 inline 行為，右側先設定 `display: inline-block;`，寬高才變成容易觀察的盒子。

未先設定 display

inline span 旁邊文字 
.tag-inline {
    width: 160px;
    height: 52px;
    padding: 10px;
    background: #fef3c7;
}

先設定 display

inline-block span 旁邊文字 
.tag-box {
    display: inline-block;

    width: 160px;
    height: 52px;
    padding: 10px;
    background: #dcfce7;
}

觀察提示

本 demo 的重點不是完整教 `display`，而是看見它會影響後面寬高、間距等判斷，因此適合放在 declaration block 前段。

元素顯示與隱藏

看不見，不代表版面行為一樣

`display: none` 會移出版面；`visibility: hidden` 會保留原本位置。

隱藏方式

切換第一個方塊，觀察第二個方塊是否補位

box 1 要被隱藏的元素
box 2 觀察我會不會補上來
box 3 下方參考元素

目前 CSS

.box1 {
  display: block;
  visibility: visible;
}
是否可見 可見
是否保留位置 保留

第一個方塊正常參與版面，第二個方塊維持在它下方。

還原顯示模式

重新顯示時，要回到合適的 `display` 值

原本是 inline-flex

標籤一 標籤二 
.tag {
  display: inline-flex;
}

誤還原成 block

標籤一 標籤二 
.tag {
  display: block;
}

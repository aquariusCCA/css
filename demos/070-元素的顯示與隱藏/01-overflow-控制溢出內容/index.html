overflow

內容超出盒子時，畫面要怎麼處理？

`overflow` 控制的是盒子內部超出範圍的內容，不是讓整個元素消失。

四種常用值

切換 `overflow`，觀察同一段內容的變化

固定高度訊息盒

這是一段故意放長的內容。盒子的寬度與高度固定，文字比盒子可以容納的空間還多。 當內容超出盒子時，`overflow` 會決定超出的部分要顯示在外面、被裁切，或透過捲動條查看。

再加一段內容，讓垂直方向明確產生溢出。請觀察盒子邊界和超出文字之間的關係。

目前 CSS

.message {
  overflow: visible;
}

觀察重點

超出的內容仍會顯示在盒子外面，可能干擾周圍版面。

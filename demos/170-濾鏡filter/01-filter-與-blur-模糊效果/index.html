filter 與 blur 模糊效果

調整模糊半徑，觀察同一張圖片在不同 filter: blur(...) 數值下的清晰度變化。

互動觀察

非 0 的 blur 數值需要長度單位

filter: blur(5px); 半徑越大，圖片越模糊。

並排比較

同一張圖片，不同 blur 半徑

沒有模糊

filter: blur(0);

中等模糊

filter: blur(5px);

明顯模糊

filter: blur(12px);

常見錯誤

錯誤寫法會被忽略

正確

img {
  filter: blur(5px);
}

漏掉單位

img {
  filter: blur(5);
}

把 blur 當屬性

img {
  blur: 5px;
}

# 第三章 CSS 的引入方式

本章主要在整理 CSS 的三種常見引入方式：行內樣式、內部樣式表、外部樣式表。

這一章的重點不是死記名稱，而是理解三種方式各自適合什麼場景，尤其是知道正式開發為什麼通常會優先使用外部樣式表。

## 本章和整體學習地圖的關係

- 這一章承接前面的 CSS 基礎觀念，進一步回答「樣式要寫在哪裡」。
- 先理解引入方式，再往後學選擇器、屬性與版面時，會更清楚樣式管理的脈絡。
- 這一章也是從練習寫法過渡到實務寫法的重要起點。

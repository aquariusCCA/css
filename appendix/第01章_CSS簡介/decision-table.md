# 問題對應屬性表

來源筆記：`notes/第01章_CSS簡介/CSS簡介.md`

用途：遇到畫面問題時，快速判斷應優先檢查哪個 CSS 屬性或條件。

| 畫面問題 | 優先檢查 | 建議屬性 | 判斷理由 | 來源 |
|---|---|---|---|---|
| 想讓標題文字變紅 | selector 是否選到標題元素 | `color` | `color` 用來設定文字顏色 | [第一課](../../notes/第01章_CSS簡介/CSS簡介.md#第一課如何讓-html-保持結構讓-css-負責外觀) |
| 想讓標題字體變大 | selector 是否選到標題元素 | `font-size` | `font-size` 用來設定字體大小 | [第一課](../../notes/第01章_CSS簡介/CSS簡介.md#第一課如何讓-html-保持結構讓-css-負責外觀) |
| 寫了樣式但畫面沒變 | selector 是否匹配 HTML 元素 | 待補來源 | 筆記要求先確認 selector 是否真的選到元素 | [第二課](../../notes/第01章_CSS簡介/CSS簡介.md#第二課一條-css-規則如何套用到畫面上) |
| 樣式宣告看起來正確但仍失效 | 冒號、分號是否為英文符號 | 待補來源 | CSS 語法中的 `:` 與 `;` 需使用英文標點 | [常見誤解與修正方式](../../notes/第01章_CSS簡介/CSS簡介.md#常見誤解與修正方式) |
| CSS 難以閱讀或修改 | 是否使用展開風格 | 待補來源 | 展開風格較容易逐行檢查語法與宣告 | [第三課](../../notes/第01章_CSS簡介/CSS簡介.md#第三課為什麼開發時常用展開風格) |

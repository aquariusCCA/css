# 名詞表

來源筆記：`notes/第01章_CSS簡介/CSS簡介.md`

用途：快速查找本章重要名詞與初學者應掌握的理解方式。

| 名詞 | 簡短定義 | 初學者理解方式 | 相關屬性或概念 | 來源 |
|---|---|---|---|---|
| HTML | 描述網頁內容與語意結構 | 說明內容是什麼，例如標題或段落 | HTML 與 CSS 分工 | [先看問題](../../notes/第01章_CSS簡介/CSS簡介.md#先看問題html-有結構但畫面不一定符合需求) |
| CSS | 描述 HTML 元素外觀與版面的樣式語言 | 告訴瀏覽器元素看起來如何 | `color`、`font-size` | [小結](../../notes/第01章_CSS簡介/CSS簡介.md#小結) |
| selector | 指定 CSS 規則作用對象的語法 | 先問「選誰」 | declaration block | [第二課](../../notes/第01章_CSS簡介/CSS簡介.md#第二課一條-css-規則如何套用到畫面上) |
| declaration block | 用 `{}` 包住的一組宣告 | 表示被選到的元素要套用哪些樣式 | declaration、`{}` | [第二課](../../notes/第01章_CSS簡介/CSS簡介.md#第二課一條-css-規則如何套用到畫面上) |
| declaration | 一個 `property: value;` 樣式設定 | 回答「怎麼改」 | property、value | [第二課](../../notes/第01章_CSS簡介/CSS簡介.md#第二課一條-css-規則如何套用到畫面上) |
| property | 要修改的樣式類型 | 例如改顏色或字體大小 | `color`、`font-size` | [基本語法回查](../../notes/第01章_CSS簡介/CSS簡介.md#基本語法回查) |
| value | property 使用的設定值 | 例如 `red` 或 `40px` | `red`、`40px` | [基本語法回查](../../notes/第01章_CSS簡介/CSS簡介.md#基本語法回查) |
| 展開風格 | 將 selector 與每條 declaration 分行書寫 | 開發時較容易閱讀、修改與檢查 | 緊湊風格 | [第三課](../../notes/第01章_CSS簡介/CSS簡介.md#第三課為什麼開發時常用展開風格) |
| 緊湊風格 | 移除多數空白與換行的 CSS 寫法 | 畫面效果相同，但較不利閱讀 | 展開風格 | [第三課](../../notes/第01章_CSS簡介/CSS簡介.md#第三課為什麼開發時常用展開風格) |
| 瀏覽器預設樣式 | 未額外寫 CSS 時瀏覽器給元素的基本呈現 | 不一定符合設計需求 | HTML 與 CSS 分工 | [第一課](../../notes/第01章_CSS簡介/CSS簡介.md#第一課如何讓-html-保持結構讓-css-負責外觀) |

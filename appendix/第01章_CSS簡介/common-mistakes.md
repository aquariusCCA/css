# 常見誤解速查

來源筆記：`notes/第01章_CSS簡介/CSS簡介.md`

用途：快速排查本章常見誤解與錯誤用法。

| 誤解或錯誤寫法 | 錯誤原因 | 修正方式 | 相關概念 | 來源 |
|---|---|---|---|---|
| 文字變大就等於語意上是標題 | CSS 只改外觀，不改 HTML 語意 | 用正確 HTML 標籤建立語意，再用 CSS 調外觀 | HTML、CSS | [常見誤解與修正方式](../../notes/第01章_CSS簡介/CSS簡介.md#常見誤解與修正方式) |
| 寫了 `color` 畫面就一定會變 | selector 可能沒有匹配到元素 | 先檢查 selector 與 HTML 結構是否對得上 | selector、`color` | [常見誤解與修正方式](../../notes/第01章_CSS簡介/CSS簡介.md#常見誤解與修正方式) |
| CSS 標點可混用全形符號 | 全形 `：`、`；` 不是 CSS 語法需要的標點 | 使用英文 `:` 與 `;` | declaration | [常見誤解與修正方式](../../notes/第01章_CSS簡介/CSS簡介.md#常見誤解與修正方式) |
| 緊湊風格較適合日常開發 | 緊湊風格不利閱讀與維護 | 開發時優先使用展開風格 | 展開風格、緊湊風格 | [常見誤解與修正方式](../../notes/第01章_CSS簡介/CSS簡介.md#常見誤解與修正方式) |
| CSS 只是讓網頁變好看 | CSS 也負責文字、圖片外型、間距與版面結果 | 把 CSS 理解成外觀與版面規則 | CSS | [常見誤解與修正方式](../../notes/第01章_CSS簡介/CSS簡介.md#常見誤解與修正方式) |
| 把 `font-size` 寫在大括號外 | 宣告不屬於目標 selector 的規則區塊 | 將宣告放進對應 selector 的 `{}` 中 | declaration block、`font-size` | [第二課](../../notes/第01章_CSS簡介/CSS簡介.md#第二課一條-css-規則如何套用到畫面上) |

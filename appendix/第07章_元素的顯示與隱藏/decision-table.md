# 問題對應屬性表

來源筆記：`notes/第07章_元素的顯示與隱藏/元素的顯示與隱藏.md`

用途：遇到顯示、隱藏或溢出問題時，快速判斷應優先檢查哪個 CSS 屬性或條件。

| 畫面問題 | 優先檢查 | 建議屬性 | 判斷理由 | 來源 |
|---|---|---|---|---|
| 元素要完全消失，後方內容要往上補位 | 是否希望元素退出版面 | `display: none` | `display: none` 不顯示也不佔位 | [第一課](../../notes/第07章_元素的顯示與隱藏/元素的顯示與隱藏.md#第一課想讓元素完全從版面中消失) |
| 元素看不見後仍留下空白 | 是否使用了 `visibility: hidden` | `visibility` | `visibility: hidden` 會保留原本位置 | [第三課](../../notes/第07章_元素的顯示與隱藏/元素的顯示與隱藏.md#第三課想讓元素看不見但保留原本位置) |
| 元素不見但下方內容沒有補位 | 需求是否其實要移除佔位 | `display`、`visibility` | 要補位應使用 `display: none`，不是 `visibility: hidden` | [常見誤解](../../notes/第07章_元素的顯示與隱藏/元素的顯示與隱藏.md#常見誤解與修正方式) |
| 被 `display: none` 的盒子要重新出現 | 是否要以 block 方式參與排版 | `display: block` | `block` 會讓元素重新產生盒子並排進正常流程 | [第二課](../../notes/第07章_元素的顯示與隱藏/元素的顯示與隱藏.md#第二課想把被隱藏的元素重新顯示) |
| 還原顯示後排版方式不對 | 原本元素是否適合 block 排版 | `display` | 筆記提醒不應把 `display: block` 當所有元素的通用還原值 | [第二課](../../notes/第07章_元素的顯示與隱藏/元素的顯示與隱藏.md#第二課想把被隱藏的元素重新顯示) |
| 固定高度盒子的文字跑出邊框 | 問題是否是內容超出盒子 | `overflow` | `overflow` 用來處理內容溢出後的顯示方式 | [第四課](../../notes/第07章_元素的顯示與隱藏/元素的顯示與隱藏.md#第四課想處理超出盒子的內容) |
| 超出盒子的內容不要露出 | 是否要裁切超出範圍 | `overflow: hidden` | `hidden` 會裁切超出盒子範圍的內容 | [第四課](../../notes/第07章_元素的顯示與隱藏/元素的顯示與隱藏.md#第四課想處理超出盒子的內容) |
| 超出內容需要可以查看 | 是否需要捲動查看內容 | `overflow: auto`、`overflow: scroll` | `auto` 溢出時才顯示捲動條，`scroll` 不溢出也可能顯示 | [第四課](../../notes/第07章_元素的顯示與隱藏/元素的顯示與隱藏.md#第四課想處理超出盒子的內容) |
| 看見 `hidden` 不確定是哪一種隱藏 | `hidden` 所屬屬性是什麼 | `visibility`、`overflow` | `visibility: hidden` 隱藏元素可見狀態；`overflow: hidden` 裁切溢出內容 | [第四課](../../notes/第07章_元素的顯示與隱藏/元素的顯示與隱藏.md#第四課想處理超出盒子的內容) |
| 元素看不見但原因不明 | 先確認是元素、佔位還是溢出問題 | `display`、`visibility`、`overflow` | 筆記建議分開判斷元素本身與盒子裡的內容 | [判斷流程](../../notes/第07章_元素的顯示與隱藏/元素的顯示與隱藏.md#顯示與隱藏的判斷流程) |

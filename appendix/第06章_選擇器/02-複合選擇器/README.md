## 核心總覽

| 核心項目 | 角色或作用範圍 | 適合情境 | 注意或限制 | 來源 |
|---|---|---|---|---|
| 複合選擇器 | 組合多個選擇條件，讓 CSS 更精準選到目標 | 單一基礎選擇器不夠精準時 | 重點不是讓 `selector` 變長，而是描述必要條件 | [先看問題](../../../notes/第06章_選擇器/02-複合選擇器.md#先看問題基礎選擇器不一定夠精準) |
| 後代選擇器 | 用空格選外層內所有層級的目標元素 | 限制某區塊內的連結、標題或段落 | 不是只選直接子元素，範圍可能較大 | [第一課](../../../notes/第06章_選擇器/02-複合選擇器.md#第一課想選到某個區塊裡的所有後代元素) |
| 子選擇器 | 用 `>` 選直接子層元素 | 只選第一層結構，不選更深層 | HTML 多包一層就不會選到 | [第二課](../../../notes/第06章_選擇器/02-複合選擇器.md#第二課想只選直接子元素) |
| 兄弟選擇器 | 用 `+` 或 `~` 選同一層後方元素 | 控制標題後段落或後續同層元素 | 只能往後選，且必須是同一父元素下的同層元素 | [第三課](../../../notes/第06章_選擇器/02-複合選擇器.md#第三課想選同一層中後面的元素) |
| 並集選擇器 | 用 `,` 讓多個 `selector` 共用同一組宣告 | 減少重複宣告 | 忘記逗號會改變 `selector` 意義 | [第四課](../../../notes/第06章_選擇器/02-複合選擇器.md#第四課想讓多種元素共用同一組樣式) |
| 交集選擇器 | 讓同一元素同時符合多個選擇條件 | 精準選到某類標籤中的特定 `class` | 條件中間不能加空格 | [第五課](../../../notes/第06章_選擇器/02-複合選擇器.md#第五課想選同時符合多個條件的元素) |

# 本章查表入口

來源筆記：`notes/第06章_選擇器/01-基礎選擇器.md`

用途：快速判斷基礎選擇器應該先查哪份資料。

本章查表路徑：`appendix/第06章_選擇器/01-基礎選擇器/`

## 先從這裡查

| 讀者想查 | 先看檔案 | 適合查什麼 |
|---|---|---|
| 不知道該用哪種基礎選擇器 | `decision-table.md` | 從「要改誰」反查標籤、class、id 或 `*` |
| 想確認 `selector` 符號和 HTML 怎麼對 | `concept-glossary.md` | 查 `.class`、`#id`、`*` 與 HTML attribute 的關係 |
| CSS 樣式沒有套到預期元素 | `decision-table.md` | 檢查 `selector` 是否真的匹配目標 HTML |
| 想排除 class、id、多類名的常見錯誤 | `common-mistakes.md` | 查少寫符號、重複 id、逗號分隔 class 等誤解 |
| 想用關鍵字回找本章概念 | `keyword-index.md` | 從 `p`、`.`、`#`、`*`、reset 或過度選取定位查表檔 |

## 核心總覽

| 核心項目 | 角色或作用範圍 | 適合情境 | 注意或限制 | 來源 |
|---|---|---|---|---|
| CSS `selector` | 在宣告區塊套用前，先決定要選到哪些 HTML 元素 | 排查樣式作用對象 | 只看 `property: value;` 不能判斷誰被改 | [先看問題](../../../notes/第06章_選擇器/01-基礎選擇器.md#先看問題css-第一件事是選到元素) |
| 標籤選擇器 | 以 HTML 標籤名稱選到同種元素 | 同一種標籤都要套用基礎樣式 | 不適合只改其中幾個同標籤元素 | [第一課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第一課想一次改同一種標籤) |
| 類選擇器 | 以 `.類名` 選到有相同 `class` 的元素 | 多個元素共用同一種樣式 | CSS 要寫 `.`；同一 class 可以重複使用 | [第二課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第二課想只改某幾個元素) |
| 多類名 | 同一元素用空格放多個 class，讓多條 class 規則同時選到 | 把共用樣式拆開重組 | 多個 class 用空格，不是逗號 | [第三課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第三課想把共用樣式拆開重組) |
| id 選擇器 | 以 `#id` 選到頁面中的唯一元素 | 主標題區塊或唯一定位需求 | 重複套用外觀通常更適合用 class | [第四課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第四課想選頁面中的唯一元素) |
| 通配符選擇器 | 以 `*` 選取頁面中所有元素 | 全頁基礎規則或 reset | 影響範圍非常大，不適合處理局部樣式 | [第五課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第五課想對所有元素套用基礎規則) |

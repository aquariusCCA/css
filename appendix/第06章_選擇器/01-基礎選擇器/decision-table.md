# 問題對應檢查表

來源筆記：`notes/第06章_選擇器/01-基礎選擇器.md`

用途：遇到選擇器選取需求或樣式失效時，快速判斷應優先檢查哪個 `selector` 或 HTML 對應。

| 畫面問題 | 優先檢查 | 建議檢查或語法 | 判斷理由 | 來源 |
|---|---|---|---|---|
| CSS 樣式沒有套到預期元素 | `selector` 是否真的匹配目標 HTML | `selector { property: value; }` | `selector` 決定作用對象，宣告區塊只決定樣式內容 | [先看問題](../../../notes/第06章_選擇器/01-基礎選擇器.md#先看問題css-第一件事是選到元素) |
| 想一次修改所有同種標籤 | 目標元素是否都是同一種 HTML 標籤 | `p { ... }` | 標籤選擇器會選到所有符合該標籤名稱的元素 | [第一課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第一課想一次改同一種標籤) |
| 只想修改幾個同為 `<li>` 的項目 | 目標項目是否有可共用的 `class` | `class="important"`、`.important` | 同一標籤中的少數元素需要用 class 區分 | [第二課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第二課想只改某幾個元素) |
| 多個不同標籤要套用同一種樣式 | 是否能加上相同 class | `.important` | 類選擇器可選到不同標籤上相同 class 的元素 | [第二課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第二課想只改某幾個元素) |
| 一個元素需要同時套用紅色與大字 | `class` 中多個名稱是否用空格分隔 | `class="red large-text"` | 同一元素可同時被 `.red` 與 `.large-text` 選到 | [第三課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第三課想把共用樣式拆開重組) |
| HTML 有 `class="important"`，CSS 卻沒生效 | CSS `selector` 是否少了 `.` | `.important { ... }` | 少了 `.` 會被當成標籤名稱，不是 class 名稱 | [第二課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第二課想只改某幾個元素) |
| 多類名寫了但其中一個樣式選不到 | 是否把 class 名稱用逗號分隔 | `class="red large-text"` | HTML 的 `class` 屬性中，多個 class 必須用空格隔開 | [第三課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第三課想把共用樣式拆開重組) |
| 想選頁面中的唯一主標題區塊 | `id` 是否用於唯一元素 | `id="main-title"`、`#main-title` | id 屬性提供唯一識別，CSS 用 `#` 選取 | [第四課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第四課想選頁面中的唯一元素) |
| 多個元素都需要同一種外觀 | 是否誤把 id 當成可重複樣式名稱 | 改用 `class` 與 `.類名` | 來源提醒重複使用樣式通常更適合用 class | [第四課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第四課想選頁面中的唯一元素) |
| 練習版面時出現未預期空白 | 是否真的要影響所有元素 | `* { margin: 0; padding: 0; box-sizing: border-box; }` | 通配符選擇器可用於全頁基礎規則或 reset | [第五課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第五課想對所有元素套用基礎規則) |
| 寫了 `*` 後太多元素都被改到 | 是否發生過度選取 | 改用標籤、class 或 id 選擇器 | `*` 會選取所有元素，不適合處理局部畫面 | [第五課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第五課想對所有元素套用基礎規則) |
| 不確定基礎選擇器與 HTML 怎麼對應 | 先查 `selector` 寫法與 HTML 對應欄 | `p`、`.important`、`#main-title`、`*` | 回查表已整理基礎選擇器、CSS 寫法與適合情境 | [回查表](../../../notes/第06章_選擇器/01-基礎選擇器.md#基礎選擇器回查表) |

# 名詞表

來源筆記：`notes/第06章_選擇器/01-基礎選擇器.md`

用途：快速查找本章重要 `selector` 名詞、HTML 對應方式與選取範圍。

| 名詞 | 簡短定義 | 初學者理解方式 | 相關屬性或概念 | 來源 |
|---|---|---|---|---|
| CSS `selector` | CSS 規則中負責選到 HTML 元素的部分 | 先回答「要改誰」，再看宣告區塊要改成什麼樣子 | `declaration block`、HTML 元素 | [前置知識](../../../notes/第06章_選擇器/01-基礎選擇器.md#前置知識) |
| `declaration block` | `selector` 後方用來放樣式宣告的區塊 | 回答「要改成什麼樣子」，不負責決定選到誰 | `property: value;`、`selector` | [前置知識](../../../notes/第06章_選擇器/01-基礎選擇器.md#前置知識) |
| 標籤選擇器 | 直接用 HTML 標籤名稱選到同種元素 | `p` 會選到頁面上所有 `<p>`，不是只選第一個 | `p`、`div`、同類元素 | [第一課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第一課想一次改同一種標籤) |
| 類選擇器 | 用 `.類名` 選到有相同 `class` 的元素 | 適合把同一種可重複樣式套到多個元素 | `.important`、`class`、reusable style | [第二課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第二課想只改某幾個元素) |
| `class` attribute | HTML 上讓 CSS 辨識特定元素群組的 attribute | 替需要共用樣式的元素取一個可被 CSS 找到的名字 | 類選擇器、多類名 | [第二課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第二課想只改某幾個元素) |
| 多類名 | 同一個 HTML 元素在 `class` 中放多個 class 名稱 | 一個元素可同時被多條 class 規則選到 | `class="red large-text"`、可組合樣式 | [第三課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第三課想把共用樣式拆開重組) |
| id 選擇器 | 用 `#id` 選到指定 `id` 的元素 | 用於頁面中唯一元素，不適合當重複樣式名稱 | `#main-title`、`id` | [第四課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第四課想選頁面中的唯一元素) |
| `id` attribute | HTML 中提供唯一識別的 attribute | 來源提醒同一份 HTML 文件中 id 應該只出現一次 | id 選擇器、JavaScript 定位 | [第四課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第四課想選頁面中的唯一元素) |
| 通配符選擇器 | 用 `*` 選取頁面中所有元素 | 適合全頁基礎設定，不適合當局部樣式捷徑 | reset、全頁基礎規則 | [第五課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第五課想對所有元素套用基礎規則) |
| 過度選取 | `selector` 選到超過需求範圍的元素 | 例如只想改局部，卻用 `li` 或 `*` 改到太多元素 | 標籤選擇器、通配符選擇器 | [常見誤解](../../../notes/第06章_選擇器/01-基礎選擇器.md#常見誤解與修正方式) |
| 可重複使用樣式 | 可被多個元素共用的樣式命名方式 | 來源把 class 視為適合 reusable style 的選擇器 | 類選擇器、多類名 | [第二課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第二課想只改某幾個元素) |
| reset | 全頁基礎設定的起手式情境 | 用 `*` 先處理所有元素的基礎空白或盒模型設定 | `margin`、`padding`、`box-sizing` | [第五課](../../../notes/第06章_選擇器/01-基礎選擇器.md#第五課想對所有元素套用基礎規則) |

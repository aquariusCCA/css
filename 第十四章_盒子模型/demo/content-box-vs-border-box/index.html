CSS Box Model Demo

content-box vs border-box

使用相同的 width、height、padding、border，觀察兩種 box-sizing 模式下的實際盒子尺寸差異。

共同 CSS 參數

content padding border outer size

預設模式

content-box

width = content
content 240 × 180
CSS width / height
240 × 180
content size
240 × 180
outer size
304 × 244

常用布局模式

border-box

width = border box
content 176 × 116
CSS width / height
240 × 180
content size
176 × 116
outer size
240 × 180

尺寸計算規則

content-box

width 只代表 content 寬度。

outer width = width + padding × 2 + border × 2

border-box

width 代表 content + padding + border 的總寬度。

content width = width - padding × 2 - border × 2

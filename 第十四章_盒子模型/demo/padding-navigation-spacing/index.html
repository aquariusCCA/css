How it works

為什麼導航列常用 padding？

1. 文字決定基礎寬度

導航項目通常是 a 標籤。文字越長，內容區域本來就越寬。

2. padding 增加內部留白

padding-inline 增加左右空間，padding-block 增加上下空間，點擊區域也會一起變大。

3. 比固定寬度更彈性

不同語系、不同文字長度、響應式版面，都比固定寬度更容易維護。

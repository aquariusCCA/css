Scenario 1

兩個 block 兄弟元素上下相鄰時，中間距離通常取較大的 margin，而不是相加。

觀察重點

  • 相鄰 block 兄弟元素的垂直 margin 不一定相加，正值通常取較大者。
  • 父元素沒有 padding、border、inline content、height/min-height 或 BFC 時，第一個子元素的 margin-top 可能與父層合併。
  • display: flow-root 是現代 CSS 中建立 BFC、阻止父子 margin 塌陷的清楚寫法。
  • overflow: hidden 也能建立 BFC，但可能裁切陰影、浮動內容或超出範圍的元素。
  • padding-top: 1px 與透明 border-top 都能阻隔塌陷，但會額外影響尺寸。

# 第14章 盒子模型

## 章節說明

本章整理 CSS 盒子模型相關概念，包含網頁布局的盒子視角、盒子模型的組成、內容區域、內邊距、邊框、外邊距、默認內外邊距清除、`box-sizing`、盒子陰影、外輪廓，以及行內元素在垂直方向上的內外邊距問題。

這一章的學習重點不是只背屬性名稱，而是理解瀏覽器如何把頁面元素當成一個個矩形盒子來計算尺寸、間距與排列。後續學背景、布局、對齊與響應式設計時，都會反覆用到這個基礎。

## 學習建議

學盒子模型時，要先分清楚「盒子自身尺寸」和「盒子之間的距離」。

- `content` 是內容區域，`width` 和 `height` 默認控制這一層。
- `padding` 是內容和邊框之間的距離，會增加盒子佔用空間。
- `border` 是盒子的邊界，會影響盒子尺寸計算。
- `margin` 是盒子外部的距離，用來拉開不同盒子。
- `box-sizing` 會改變寬高的計算方式，是排版尺寸是否符合預期的常見關鍵。
- `box-shadow` 和 `outline` 屬於視覺效果與輔助邊界，理解時不要和盒子模型四個基本部分混在一起。

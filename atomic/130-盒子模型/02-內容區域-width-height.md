# 內容區域 width height

在預設盒模型中，`width` 和 `height` 設定的是盒子的內容區域大小。

![盒子內容區域由 width 和 height 控制的示意圖](../../origin/130-盒子模型/assets/images/box-content-img-001-b074cc.png)

相關屬性：

- `width`
- `height`

常見取值是數字加上 `px` 單位。

```css
div {
  width: 200px;
  height: 200px;
  background-color: pink;
}
```

```html
<div>文字</div>
```

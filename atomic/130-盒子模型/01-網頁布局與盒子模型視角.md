# 網頁布局與盒子模型視角

網頁布局的核心本質，是利用 CSS 擺放盒子。

網頁布局通常可以理解成三個步驟：

1. 先準備好相關的網頁元素。網頁元素基本都可以視為盒子 Box。
2. 利用 CSS 設定盒子的樣式，然後把盒子擺放到相應的位置。
3. 往盒子裡面放入內容。

所謂盒子模型，就是把 HTML 頁面中的布局元素看作矩形盒子，也就是盛裝內容的容器。

頁面中的每一個標籤，都可以從盒子的角度觀察。瀏覽器在渲染網頁時，也會將網頁中的元素看作一個個矩形區域，因此用「盒子」來理解布局會更直觀。

CSS 規定每個盒子由四個部分構成：

- content：內容
- padding：內邊距
- border：邊框
- margin：外邊距

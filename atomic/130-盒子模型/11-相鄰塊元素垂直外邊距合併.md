# 相鄰塊元素垂直外邊距合併

使用 `margin` 定義塊元素的垂直外邊距時，可能會出現外邊距合併。

主要有兩種情況：

- 相鄰塊元素垂直外邊距的合併。
- 嵌套塊元素垂直外邊距的塌陷。

當上下相鄰的兩個塊元素是兄弟關係時，如果上面的元素有下外邊距 `margin-bottom`，下面的元素有上外邊距 `margin-top`，它們之間的垂直間距不是兩者相加，而是取兩個值中的較大者。

這種現象稱為相鄰塊元素垂直外邊距的合併。

![相鄰塊元素垂直外邊距合併前後對比圖](../../origin/130-盒子模型/assets/images/box-margin-img-002-ce44ee.png)

```css
.damao, .ermao {
  width: 200px;
  height: 200px;
  background-color: pink;
}

.damao {
  margin-bottom: 100px;
}

.ermao {
  margin-top: 200px;
}
```

```html
<div class="damao">大毛</div>
<div class="ermao">二毛</div>
```

解決方案：只給其中一個盒子添加 `margin` 值。

# 嵌套塊元素垂直外邊距塌陷

在相互嵌套的塊級元素中，子元素的 `margin-top` 可能會作用到父元素上。

解決方案：

1. 給父元素設定 `border-top` 或 `padding-top`，用來分隔父子元素的 `margin-top`。
2. 為父元素添加 `overflow: hidden`。
3. 將元素轉換為行內塊元素。
4. 設定浮動。

```css
.box-father {
  width: 200px;
  height: 200px;
  background-color: #b2b6b6;
}

.box-child {
  width: 100px;
  height: 100px;
  background-color: #7f9faf;
  margin-top: 100px;
}

.resolve {
  overflow: hidden;
  margin-top: 20px;
}
```

```html
<!-- 元素的 margin-top 作用在了父元素上 -->
<div class="box-father">
  <div class="box-child"></div>
</div>

<!-- 解決方案：給父元素設定 overflow:hidden; -->
<div class="box-father resolve">
  <div class="box-child"></div>
</div>
```

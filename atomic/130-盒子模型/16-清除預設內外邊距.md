# 清除預設內外邊距

很多網頁元素都帶有預設的內外邊距，而且不同瀏覽器的預設值也不一致。因此在布局前，通常要先清除網頁元素的內外邊距。

行內元素為了照顧兼容性，通常盡量只設定左右內外邊距，不要設定上下內外邊距。轉換為塊級或行內塊元素後，就可以設定上下內外邊距。

京東做法：

```css
* {
  margin: 0;
  padding: 0;
}
```

淘寶做法：

```css
blockquote,
body,
button,
dd,
dl,
dt,
fieldset,
form,
h1,
h2,
h3,
h4,
h5,
h6,
hr,
input,
legend,
li,
ol,
p,
pre,
td,
textarea,
th,
ul {
  margin: 0;
  padding: 0;
}
```

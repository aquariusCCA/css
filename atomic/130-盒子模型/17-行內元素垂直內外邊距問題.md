# 行內元素垂直內外邊距問題

如果想要透過 `margin` 或 `padding` 改變行內標籤的垂直位置，無法生效：

- 行內標籤的 `margin-top` 和 `margin-bottom` 不生效。
- 行內標籤的 `padding-top` 和 `padding-bottom` 不生效。

```css
span {
  /* 垂直位置，無法生效 */
  margin: 100px;
  /* padding: 100px; */
  /* line-height: 100px; */
}
```

```html
<span>span</span>
<span>span</span>
```

解決方式：加上 `line-height`。

# 行內元素垂直內外邊距問題

如果想要透過 `margin` 或 `padding` 改變行內標籤的垂直位置，需要注意：

- 行內標籤的 `margin-top` 和 `margin-bottom` 通常不影響行盒高度，也不會把上下內容推開。
- 行內標籤的 `padding-top` 和 `padding-bottom` 會套用到元素本身，但通常不會像區塊元素一樣推開上下行。

```css
span {
  /* 垂直位置，無法生效 */
  margin: 100px;
  /* padding: 100px; */
  /* line-height: 100px; */
}
```

```html
<span>span</span>
<span>span</span>
```

解決方式：使用 `line-height` 控制行高，或視需求把元素改成 `inline-block` / `block`。

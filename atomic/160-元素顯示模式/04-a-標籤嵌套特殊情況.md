# a 標籤嵌套特殊情況

連結裡面不能再放連結。

```html
<a href="http://www.baidu.com">
  <a href="#">空鏈接</a>
</a>
```

特殊情況下，連結 `<a>` 裡面可以放塊級元素，但是給 `<a>` 轉換成塊級模式最安全。

```css
a {
  display: block;
}
```

```html
<a class="a" href="">
  <div></div>
</a>
```

> 待後續內容審查確認：`a` 標籤可嵌套內容的說法需要確認 HTML 版本語境與互動內容限制。

# a 標籤嵌套特殊情況

連結裡面不能再放連結。

```html
<a href="http://www.baidu.com">
  <a href="#">空鏈接</a>
</a>
```

現代 HTML 中，連結 `<a>` 可以包住符合其內容模型的非互動內容，例如部分 flow content；但不能包含另一個 `<a>`、其他互動內容，或帶有 `tabindex` 的後代元素。

如果想讓整塊內容都成為可點擊區域，可以把 `<a>` 顯示成塊級盒子；`display: block` 只改變版面呈現，不會改變 HTML 的嵌套規則。

```css
a {
  display: block;
}
```

```html
<a class="a" href="">
  <div></div>
</a>
```

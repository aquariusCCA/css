# 元素顯示模式總結與嵌套規範

塊級、行內與行內塊元素的顯示模式可以透過比較表統整。

![塊級、行內與行內塊元素顯示模式比較表](../../origin/160-元素顯示模式/assets/images/element-display-mode-summary-img-001-498fc4.png)

HTML 嵌套規範注意點：

- 塊級元素一般作為大容器，可以嵌套文本、塊級元素、行內元素、行內塊元素。
- `<p>` 標籤中不要嵌套 `<div>`、`<p>`、`<h>` 等塊級元素。
- `<a>` 標籤內部可以嵌套任意內容。
- `<a>` 標籤不能嵌套 `<a>` 標籤。

> 待後續內容審查確認：`a` 標籤可嵌套任意內容的說法需要確認 HTML 版本語境與互動內容限制。

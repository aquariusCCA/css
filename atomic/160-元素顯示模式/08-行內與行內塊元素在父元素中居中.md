# 行內與行內塊元素在父元素中居中

行內元素、行內塊元素可以被父元素當作文本處理。

也就是說，可以像處理文本對齊一樣處理它們，例如使用：

- `text-align`
- `line-height`
- `text-indent`

如果子元素是行內元素或行內塊元素，要讓它在父元素中水平居中，可以給父元素加上：

```css
text-align: center;
```

如果要讓行內元素或行內塊元素在父元素中垂直居中，可以給父元素與子元素搭配以下設定：

- 讓父元素的 `height` 等於 `line-height`。
- 讓每個子元素都加上 `vertical-align: middle`。
- 如果想更精確地垂直居中，可以把父元素 `font-size` 設定為 `0`。

```css
.outer {
  width: 400px;
  height: 400px;
  background-color: gray;
  text-align: center;
  line-height: 400px;
  font-size: 0px;
}

input {
  vertical-align: middle;
}

span {
  font-size: 20px;
  vertical-align: middle;
  background-color: orange;
}
```

```html
<div class="outer">
  <span>出來玩啊</span>
  <input type="text">
</div>
```

水平居中示例：

```css
.header {
  width: 900px;
  height: 200px;
  background-color: pink;
  margin: 100px auto;
  text-align: center;
}
```

```html
<div class="header">
  <span>里面的文字</span>
</div>

<div class="header">
  <img src="./images/down.jpg" alt="">
</div>
```

# 行內元素特點

常見行內元素有 `<a>`、`<strong>`、`<b>`、`<em>`、`<i>`、`<del>`、`<s>`、`<ins>`、`<u>`、`<span>` 等。

其中 `<span>` 是最典型的行內元素。有的地方也會將行內元素稱為內聯元素。

行內元素的特點：

- 相鄰行內元素會在一行上，一行可以顯示多個。
- 直接設定高度、寬度是無效的。
- 寬度和高度預設由內容撐開。
- 行內元素只能容納文本或其他行內元素。

```css
span {
  background-color: hotpink;
}
```

```html
<span>pink老师你怎么穿着品如的衣服呢</span>
<strong>品如的衣服</strong>
<span>pink老师</span>
<strong>品如的衣服</strong>
```

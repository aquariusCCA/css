# 塊元素特點

常見塊元素有 `<h1>` 到 `<h6>`、`<p>`、`<div>`、`<ul>`、`<ol>`、`<li>` 等，其中 `<div>` 是最典型的塊元素。

塊級元素的特點：

- 比較霸道，自己獨占一行。
- 高度、寬度、外邊距以及內邊距都可以控制。
- 寬度預設為 `auto`；一般正常流中的塊級盒子會撐滿父容器的可用寬度。
- 高度預設由內容撐開。
- 是一個容器及盒子，裡面可以放行內元素或塊級元素。

注意事項：

- 文字類元素內不能使用塊級元素。
- `<p>` 標籤主要用於存放文字，因此 `<p>` 裡面不能放塊級元素，特別是不能放 `<div>`。
- 同理，`<h1>` 到 `<h6>` 都是文字類塊級標籤，裡面也不能存放其他塊級元素。

```css
div {
  width: 200px;
  background-color: pink;
}
```

```html
<div>比较霸道,自己独占一行</div>

瑟瑟发抖
```

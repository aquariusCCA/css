# 定位組成：定位模式與邊偏移

定位是將盒子定在某一個位置，因此定位也是在擺放盒子，只是按照定位的方式移動盒子。

定位由兩個部分組成：

- 定位模式。
- 邊偏移。

定位模式用於指定元素在文檔中的定位方式。

邊偏移決定元素的最終位置。

定位模式由 CSS 的 `position` 屬性設定。

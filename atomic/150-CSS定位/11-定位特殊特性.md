# 定位特殊特性

絕對定位和固定定位也和浮動類似。

特殊特性包含：

- 行內元素添加絕對定位或固定定位後，可以直接設定高度和寬度。
- 塊級元素添加絕對定位或固定定位後，如果不給寬度或高度，預設大小由內容決定。
- 行內盒子如果加了浮動、固定定位或絕對定位，不用轉換，就可以直接設定寬度和高度。
- 浮動元素、絕對定位元素、固定定位元素都不會觸發外邊距合併問題。
- 脫標的盒子不會觸發外邊距塌陷。

```css
span {
  position: absolute;
  top: 300px;
  width: 200px;
  height: 150px;
  background-color: pink;
}

div {
  position: absolute;
  background-color: skyblue;
}
```

```html
<body>
  <span>123</span>

  <div>abcd</div>
</body>
```

# 定位模式總結

定位模式包含 `static`、`relative`、`absolute`、`fixed`、`sticky`。

![static relative absolute fixed sticky 定位模式總結表](../../origin/150-CSS定位/assets/images/positioning-modes-summary-img-001-f984fa.png)

表中的 `sticky` 可先理解為相對定位與固定效果的混合；更精確地說，它會相對最近的滾動祖先與包含塊運作，不一定永遠以瀏覽器可視區為基準。

一定要記住相對定位、固定定位、絕對定位的兩個大特點：

- 是否佔有位置，也就是是否脫標。
- 以誰為基準點移動。

重點學會子絕父相：兒子使用絕對定位，父親使用相對定位。

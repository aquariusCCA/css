# 絕對定位與固定定位會完全壓住盒子

絕對定位、固定定位會壓住下面標準流中的所有內容。

```css
.box {
  /* 1. 浮動的元素不會壓住下面標準流的文字 */
  /* float: left; */

  /* 2. 絕對定位或固定定位會壓住下面標準流所有內容 */
  position: absolute;
  width: 150px;
  height: 150px;
  background-color: pink;
}
```

```html
<body>
  <div class="box"></div>
  <p>阁下何不同风起，扶摇直上九万里</p>
</body>
```

浮動之所以不會壓住文字，是因為浮動最初的目的就是做文字環繞效果。文字會圍繞浮動元素。

# 絕對定位與固定定位可能覆蓋普通流內容

當絕對定位或固定定位元素與普通流內容重疊時，通常不會像浮動那樣讓文字環繞，而可能覆蓋下方內容；實際前後關係仍會受層疊規則影響。

```css
.box {
  /* 1. 浮動的元素不會壓住下面標準流的文字 */
  /* float: left; */

  /* 2. 絕對定位或固定定位與普通流內容重疊時，可能覆蓋下方內容 */
  position: absolute;
  width: 150px;
  height: 150px;
  background-color: pink;
}
```

```html
<body>
  <div class="box"></div>
  <p>阁下何不同风起，扶摇直上九万里</p>
</body>
```

浮動之所以不會壓住文字，是因為浮動最初的目的就是做文字環繞效果。文字會圍繞浮動元素。

# 顏色與單色顯示特性

這組媒體特徵用來檢測設備的顏色能力，並根據顏色深度、顏色索引或是否單色顯示來調整樣式。

## color

`color` 用來檢測設備能夠顯示的顏色深度，也就是輸出設備每個顏色分量可以顯示的位元數。

- `color`：檢查設備每個顏色分量能夠顯示的位元數。
- `min-color` 和 `max-color`：檢查每個顏色分量位元數的最小或最大值。

```css
/* 當設備每個顏色分量能顯示 8 位或更多時，使用高清圖片 */
@media (min-color: 8) {
  body {
    background-image: url('high-quality-background.png');
  }
}

/* 當設備的顏色深度較低時，使用簡單圖片 */
@media (max-color: 4) {
  body {
    background-image: url('low-quality-background.png');
  }
}
```

`color` 可以根據設備顯示能力調整圖片或顏色效果，也可以在低色深設備上提供簡化樣式，以提升性能。

## color-index

`color-index` 用來檢查設備所支持的顏色索引數量，也就是顯示器使用的調色板中可用顏色總數。

```css
/* 如果設備支持256種顏色或更多，使用高品質圖片 */
@media (min-color-index: 256) {
  body {
    background-image: url('high-quality-background.png');
  }
}

/* 如果設備只支持16種顏色，使用簡單的圖片或單一顏色背景 */
@media (max-color-index: 16) {
  body {
    background-color: #333;
  }
}
```

原始筆記指出，`color-index` 在現代應用中較少使用，但在特定嵌入式系統或老舊設備上仍有使用場景。

## monochrome

`monochrome` 用於檢測設備是否使用黑白或灰度顯示模式。

- `monochrome: 1` 表示設備僅支持單色顯示。
- `monochrome: 0` 表示設備支持多種顏色。

```css
/* 當設備僅支持單色顯示時應用 */
@media (monochrome: 1) {
  body {
    color: black;
    background-color: white;
  }
}

/* 當設備支持多種顏色時應用 */
@media (monochrome: 0) {
  body {
    color: #333;
    background-color: #f0f0f0;
  }
}
```

`monochrome` 可用於針對單色顯示設備進行樣式優化，但現代設備中使用較少。設計時仍應注意顏色對比和可讀性。

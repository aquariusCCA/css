# 使用者偏好媒體特性

使用者偏好媒體特性可以根據用戶的系統或瀏覽器設定調整樣式，常見例子包括深色模式與減少動畫。

## prefers-color-scheme

`prefers-color-scheme` 用於檢測用戶的色彩模式偏好，讓你可以根據系統或瀏覽器設置自動應用深色模式或淺色模式。

- `no-preference`：表示系統沒有告知用戶使用的顏色方案。
- `light`：表示用戶偏好淺色主題。
- `dark`：表示用戶偏好深色主題。

```css
/* 當用戶偏好淺色主題時應用 */
@media (prefers-color-scheme: light) {
  body {
    background-color: white;
    color: black;
  }
}

/* 當用戶偏好深色主題時應用 */
@media (prefers-color-scheme: dark) {
  body {
    background-color: black;
    color: white;
  }
}
```

原始筆記也記錄了 JS 判斷深淺模式的方式：

```js
window.matchMedia("(prefers-color-scheme: dark)").matches;
```

## prefers-reduced-motion

`prefers-reduced-motion` 用於檢測用戶是否偏好減少動態效果，常用於對運動敏感用戶提供更友好的體驗。

- `reduce`：表示用戶偏好減少動態效果。
- `no-preference`：表示用戶沒有特別的減少動態效果偏好。

```css
/* 當用戶偏好減少動態效果時應用 */
@media (prefers-reduced-motion: reduce) {
  .animated-element {
    animation: none;
    transition: none;
  }
}

/* 當用戶沒有偏好時應用（這是默認情況） */
@media (prefers-reduced-motion: no-preference) {
  .animated-element {
    animation: slide-in 1s ease-in-out;
    transition: opacity 0.5s ease-in-out;
  }
}
```

使用 `prefers-reduced-motion` 可以尊重用戶的顯示設置，提供更舒適的瀏覽體驗，並符合無障礙設計的最佳實踐。

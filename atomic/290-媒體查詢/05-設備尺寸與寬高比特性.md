# 設備尺寸與寬高比特性

這組媒體特性都與尺寸或比例有關，但檢測對象不同：`aspect-ratio` 偏向目標顯示區域的寬高比，在一般網頁中主要對應視口寬高比；`device-*` 偏向設備顯示區域尺寸，但已不建議用於新專案。

## aspect-ratio

`aspect-ratio` 用來檢測目標顯示區域的寬高比例；在一般網頁中，通常就是視口寬高比。常用方式包括：

- `aspect-ratio`：檢測目標顯示區域的實際寬高比。
- `min-aspect-ratio` 和 `max-aspect-ratio`：檢測目標顯示區域的寬高比是否在指定範圍內。

```css
/* 當視口寬高比是 16:9 時，使用一種佈局 */
@media (aspect-ratio: 16/9) {
  .container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
  }
}

/* 當寬高比小於 1（縱向模式，如手機）時，改變顯示方式 */
@media (max-aspect-ratio: 1/1) {
  .container {
    flex-direction: column;
  }
}
```

`aspect-ratio` 可用於響應式佈局調整、多媒體設計、全屏應用，以及橫向與縱向模式切換時的佈局變化。

## device-aspect-ratio

`device-aspect-ratio` 曾用來檢測設備顯示區域本身的寬高比，而不是瀏覽器窗口或視口的寬高比。

`device-aspect-ratio` 已被列為 deprecated media feature，不建議在新專案中使用。現代響應式設計通常優先使用 `aspect-ratio`、`orientation`、`width`、`min-width` 等更靈活的媒體特徵。

```css
/* 當設備的寬高比是 16:9 時，使用一種佈局 */
@media (device-aspect-ratio: 16/9) {
  body {
    background-color: lightblue;
  }
}

/* 當設備的寬高比是 4:3 時，使用另一種佈局 */
@media (device-aspect-ratio: 4/3) {
  body {
    background-color: lightgreen;
  }
}
```

原始筆記中特別提醒：現代網頁開發更多依賴 `aspect-ratio`、`orientation`、`min-width` 等較靈活的媒體特徵，`device-aspect-ratio` 在現代開發中較少使用。

## device-height

`device-height` 用來檢測設備屏幕的物理高度，不是瀏覽器窗口或視口高度。

`device-height` 已被列為 deprecated media feature，不建議在新專案中使用。大部分現代響應式設計更常使用 `height`、`min-height`、`max-height`。

```css
/* 當設備高度大於等於 800px 時，應用特定樣式 */
@media (min-device-height: 800px) {
  body {
    background-color: lightblue;
    font-size: 16px;
  }
}

/* 當設備高度小於 600px 時，應用另一種樣式 */
@media (max-device-height: 600px) {
  body {
    background-color: lightgreen;
    font-size: 14px;
  }
}
```

這類查詢主要保留作為理解舊資料或舊程式碼的背景知識；新寫樣式時，應優先使用視口高度相關的媒體特徵。

## device-width

`device-width` 用來檢測設備屏幕的物理寬度，而非瀏覽器窗口或視口寬度。

`device-width` 已被列為 deprecated media feature，不建議在新專案中使用。現代響應式設計更常依賴視口寬度 `width`、`min-width`、`max-width`。

```css
/* 當設備寬度大於等於 1024px 時，應用桌面樣式 */
@media (min-device-width: 1024px) {
  body {
    font-size: 18px;
    margin: 0 auto;
    max-width: 1000px;
  }
}

/* 當設備寬度小於 768px 時，應用手機樣式 */
@media (max-device-width: 768px) {
  body {
    font-size: 14px;
    padding: 10px;
  }
}
```

原始筆記中特別提醒：`device-width` 針對的是設備實際屏幕寬度，無論瀏覽器窗口如何變化，它的值都不會改變。這也正是現代響應式設計通常改用視口寬度 `width` 的原因。

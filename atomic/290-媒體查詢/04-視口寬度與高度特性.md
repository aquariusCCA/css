# 視口寬度與高度特性

`width`、`min-width`、`max-width`、`height`、`min-height`、`max-height` 都是針對視口（瀏覽器窗口或顯示區域）尺寸的媒體特徵。

## 高度：height、min-height、max-height

CSS 中的 `height`、`min-height` 和 `max-height` 是媒體特徵，用來根據視口高度應用不同的樣式，特別適合用在垂直佈局調整。

- `height`：檢測視口的確切高度。
- `min-height`：當視口高度等於或大於某個指定值時觸發。
- `max-height`：當視口高度小於或等於某個指定值時觸發。

```css
/* 當視口高度等於 800px 或更高時應用 */
@media (min-height: 800px) {
  body {
    background-color: lightblue;
  }
}

/* 當視口高度小於或等於 500px 時應用 */
@media (max-height: 500px) {
  body {
    font-size: 12px;
  }
}

/* 當視口高度正好為 600px 時應用 */
@media (height: 600px) {
  body {
    background-color: lightgreen;
  }
}
```

高度媒體特徵可用於移動端設計、全屏應用、動態佈局調整。需要注意的是，它們針對的是視口高度，不是設備的物理高度。

## 寬度：width、min-width、max-width

CSS 中的媒體特徵 `width`、`min-width` 和 `max-width` 用來根據視口寬度應用不同的樣式，是響應式設計中的核心工具。

- `width`：檢測視口的具體寬度。
- `min-width`：當視口寬度等於或大於某個值時觸發，適用於針對較大設備進行設計。
- `max-width`：當視口寬度小於或等於某個值時觸發，適用於針對較小設備進行設計。

```css
/* 當視口寬度等於或大於 1200px 時，應用大屏幕樣式 */
@media (min-width: 1200px) {
  body {
    font-size: 18px;
    margin: 0 auto;
    max-width: 1100px;
  }
}

/* 當視口寬度小於或等於 768px 時，應用手機樣式 */
@media (max-width: 768px) {
  body {
    font-size: 14px;
    padding: 10px;
  }
}

/* 當視口寬度正好為 1024px 時應用 */
@media (width: 1024px) {
  body {
    background-color: lightblue;
  }
}
```

寬度媒體特徵常用於桌面與移動設備的佈局切換、自適應佈局、內容顯示與隱藏。當用戶調整瀏覽器窗口大小時，這些媒體查詢會實時觸發。

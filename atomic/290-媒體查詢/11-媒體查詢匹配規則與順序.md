# 媒體查詢匹配規則與順序

> 所有符合條件的 `@media` 規則都會生效並參與層疊；若選擇器權重相同且設定同一屬性，後出現的宣告會覆蓋前面的宣告。

## 示例 1

在下面的寫法中，如果視口寬度大於等於 `1024px`，三個媒體查詢都會匹配；因為三段規則選擇器相同、屬性相同，最後出現的 `background: blue` 會成為結果。

```html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    /* 规则1 */
    @media (min-width: 320px) {
      html {
        background: red;
      }
    }

    /* 规则2 */
    @media (min-width: 800px) {
      html {
        background: green;
      }
    }

    /* 规则3 */
    @media (min-width: 1024px) {
      html {
        background: blue;
      }
    }
  </style>
</head>

<body>

</body>

</html>
```

## 示例 2

在下面的寫法中，如果視口寬度大於等於 `1024px`，三個媒體查詢同樣都會匹配；但最後出現的是 `background: red`，因此紅色會覆蓋前面的藍色與綠色。

```html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    /* 规则1 */
    @media (min-width: 1024px) {
      html {
        background: blue;
      }
    }

    /* 规则2 */
    @media (min-width: 800px) {
      html {
        background: green;
      }
    }

    /* 规则3 */
    @media (min-width: 320px) {
      html {
        background: red;
      }
    }
  </style>
</head>

<body>

</body>

</html>
```

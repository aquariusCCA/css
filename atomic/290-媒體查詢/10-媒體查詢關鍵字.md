# 媒體查詢關鍵字

> 關鍵字將媒體類型或多個媒體特性連接到一起做為媒體查詢的條件。

## and

`and` 可以將多個媒體特性連接到一起，相當於「且」。

也就是說，一個媒體查詢中可以包含 0 到多個表達式，表達式又可以包含 0 到多個關鍵字，以及一種媒體類型。

```html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    * {
      margin: 0;
      padding: 0;
    }

    h1 {
      height: 200px;
      background-color: gray;
      text-align: center;
      line-height: 200px;
      font-size: 100px;
    }

    /* 且運算 */
    @media screen and (min-width: 700px) and (max-width: 800px) {
      h1 {
        background-color: orange;
      }
    }
  </style>
</head>

<body>
  <h1>你好啊!!</h1>
</body>

</html>
```

## or

對於所有的連接選項，只要匹配成功一個就能應用規則。

原始筆記說明：將多個媒體查詢以逗號分隔放在一起，只要其中任何一個為真，整個媒體語句就返回真，相當於 or 操作符。

```html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    * {
      margin: 0;
      padding: 0;
    }

    h1 {
      height: 200px;
      background-color: gray;
      text-align: center;
      line-height: 200px;
      font-size: 100px;
    }

    /* 或運算 */
    @media (max-width: 700px),
    (min-width: 800px) {
      h1 {
        background-color: orange;
      }
    }
  </style>
</head>

<body>
  <h1>你好啊!!</h1>
</body>

</html>
```

原始筆記中也保留了使用 `or` 的示例，後續內容審查階段建議再確認語法相容性與教學表述。

```css
@media (max-width: 700px) or (min-width: 800px) {
  h1 {
    background-color: orange;
  }
}
```

## not

`not` 操作符用來對一條媒體查詢的結果進行取反，相當於「非」。

> 注意：`not` 關鍵字僅能應用於整個查詢，而不能單獨應用於一個獨立的查詢。

```html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    * {
      margin: 0;
      padding: 0;
    }

    h1 {
      height: 200px;
      background-color: gray;
      text-align: center;
      line-height: 200px;
      font-size: 100px;
    }

    /* 否定運算符 */
    @media not screen {
      h1 {
        background-color: orange;
      }
    }
  </style>
</head>

<body>
  <h1>你好啊!!</h1>
</body>

</html>
```

## only

`only` 操作符表示僅在媒體查詢匹配成功時應用指定樣式。

主要可以通過它讓選中的樣式在老式瀏覽器中不被應用。

```css
/* 肯定運算符 */
@media only screen and (width: 800px){
  h1 {
    background-color: orange;
  }
}
```

上面這行代碼，在老式瀏覽器中被解析為 `media="only"`，因為沒有一個叫 only 的設備，所以實際上老式瀏覽器不會應用樣式。

```css
@media screen and (width: 800px){
  h1 {
    background-color: orange;
  }
}
```

上面這行代碼，在老式瀏覽器中被解析為 `media="screen"`，它把後面的邏輯表達式忽略了。所以，在使用媒體查詢時，原始筆記建議 `only` 最好不要忽略。

# 媒體查詢概念與基本語法

> 簡單來說：媒體查詢可以讓我們根據設備顯示器的特性（如視口寬度、屏幕比例、設備方向是橫向或縱向）為其設定 CSS 樣式。

隨著移動設備的快速普及，用戶不再只是通過傳統的電腦來瀏覽 Web 內容，越來越多的用戶使用各種尺寸的手機、電腦或者其它設備來瀏覽 Web 內容。為了確保應用在不同設備中呈現美觀的界面樣式，讓用戶擁有良好的體驗，就需要用到媒體查詢。

媒體查詢是 CSS 最重要的功能之一，所謂媒體查詢指的就是根據不同的媒體類型（設備類型）和條件來區分各種設備（例如：電腦、手機、平板電腦等），並為它們分別定制 CSS 樣式。

媒體查詢能讓 CSS 更精確地作用於不同的設備或同一設備的不同條件，讓所有用戶在不同設備中都能得到很好的用戶體驗。

## 基本語法

```css
@media 媒体类型 and|not|only ( 媒体特性 ) {
	/* CSS 样式 */
}
```

它由以下部分組成：

- 一個媒體類型，告訴瀏覽器這段代碼是用在什麼類型的媒體上的，例如印刷品或者屏幕。
- 一個媒體表達式，是一個被包含的 CSS 生效所需的規則或者測試。
- 一組 CSS 規則，會在測試通過且媒體類型正確時應用。

## 使用情境

- 媒體查詢（`Media Query`）是 `CSS3` 新語法。
- 使用 `@media` 查詢，可以針對不同的媒體類型定義不同的樣式。
- `@media` 可以針對不同的屏幕尺寸設置不同的樣式。
- 當重置瀏覽器大小的過程中，頁面也會根據瀏覽器的寬度和高度重新渲染頁面。
- 目前針對很多蘋果手機、`Android` 手機、平板等設備都用得到媒體查詢。

## 範例

```html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    /*  
      注意點 :
        - 媒体查询一般按照从大到小或者 从小到大的顺序来 
        - screen 还有 and 必须带上不能省略的
        - 我们的数字后面必须跟单位  970px   这个 px 不能省略的
    */

    /* 1. 小于 540px 页面的背景颜色变为蓝色 */
    @media screen and (max-width: 539px) {
      body {
        background-color: blue;
      }
    }

    /* 2. 540 ~ 970 我们的页面颜色改为 绿色 */
    /* @media screen and (min-width: 540px) and (max-width: 969px) {
      body {
          background-color: green;
      }
    } */

    @media screen and (min-width: 540px) {
      body {
        background-color: green;
      }
    }

    /* 3. 大于等于970 我们页面的颜色改为 红色 */
    @media screen and (min-width: 970px) {
      body {
        background-color: red;
      }
    }
  </style>
</head>

<body>

</body>

</html>
```

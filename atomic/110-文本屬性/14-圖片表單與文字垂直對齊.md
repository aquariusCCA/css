# 圖片表單與文字垂直對齊

圖片、表單都屬於行內塊元素，默認的 `vertical-align` 是基線對齊。

![圖片與文字預設基線對齊的示意圖](../../origin/110-文本屬性/assets/images/vertical-align-img-003-2907f2.png)

此時可以給圖片、表單這些行內塊元素的 `vertical-align` 屬性設置為 `middle`，就可以讓文字和圖片垂直居中對齊。

```css
img {
  /* 让图片和文字垂直居中 */
  vertical-align: middle;
}

textarea {
  vertical-align: middle;
}
```

```html
<img src="./images/img.jpg" alt="">  pink老师是刘德华

<br>
<textarea name="" id="" cols="30" rows="10"></textarea> 请您留言
```

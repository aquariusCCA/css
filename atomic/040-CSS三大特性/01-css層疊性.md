# CSS 層疊性

層疊就是疊加的意思，表示樣式可以一層一層地層疊覆蓋。

層疊性主要用來解決樣式衝突問題。當相同選擇器設定相同樣式時，一個樣式會覆蓋另一個衝突的樣式。

層疊性原則是：樣式衝突時遵循就近原則，也就是誰離標籤最近，就執行該樣式。

```css
div {
    color: red;
    font-size: 12px;
}

div {
    color: pink;
}
```

```html
<div>长江后浪推前浪,前浪死在沙滩上</div>
```

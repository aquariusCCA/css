# CSS 層疊性

層疊就是疊加的意思，表示樣式可以一層一層地層疊覆蓋。

層疊性主要用來解決樣式衝突問題。當多條 CSS 規則同時作用在同一個元素、同一個屬性時，瀏覽器會依照層疊規則決定最後採用哪一個值。

在同來源、同重要性、同選擇器優先級的規則發生衝突時，後宣告的規則會覆蓋前面的規則；如果選擇器優先級不同，會先比較選擇器權重。

```css
div {
    color: red;
    font-size: 12px;
}

div {
    color: pink;
}
```

```html
<div>长江后浪推前浪,前浪死在沙滩上</div>
```

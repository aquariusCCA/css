# 優先級注意點

優先級公式由左至右、由低到高：

```text
繼承 ← 通配符選擇器 ← 標籤選擇器 ← 類選擇器 ← id 選擇器 ← 行內選擇器 ← !important
```

可以理解為類選擇器永遠大於標籤選擇器，id 選擇器永遠大於類選擇器，依此類推。

等級判斷由左向右。如果某一位數值相同，則判斷下一位數值。

權重由 4 組數字組成，但是不會有進位。

```css
/* 父亲的权重是 100  */
#father {
    color: red !important;
}

/* 继承的权重为 0 */
p {
    color: pink;
}
```

```html
<div id="father">
  <p>你还是很好看</p>
</div>
```

繼承的權重是 0。如果該元素沒有直接被選中，不管父元素權重多高，子元素得到的權重都是 0。

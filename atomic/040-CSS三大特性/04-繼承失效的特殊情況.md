# 看起來沒有繼承的特殊情況

CSS 繼承機制本身沒有失效，但有些元素帶有瀏覽器預設樣式，因此看起來沒有直接沿用父層的樣式。

![CSS 瀏覽器預設樣式示意圖，標示 a 標籤 color 與 h 系列 font-size 看起來未沿用父層](../../origin/040-CSS三大特性/assets/images/css-inheritance-img-001-c2fc84.png)

- `a` 標籤通常有瀏覽器預設的連結顏色，因此看起來沒有沿用父層的 `color`。
- `h` 系列標籤通常有瀏覽器預設的字體大小，因此看起來沒有沿用父層的 `font-size`。

如果明確設定 `color: inherit` 或 `font-size: inherit`，仍可以讓這些元素使用父層繼承值。

```css
div {
  color: red;
  font-size: 12px;
}
```

```html
<div>
  <a href="#">超鏈接</a>
  <h1>一級標題</h1>
</div>
```

# 繼承失效的特殊情況

CSS 繼承存在失效的特殊情況。

![CSS 繼承失效示意圖，標示 a 標籤 color 與 h 系列 font-size 不繼承父層](../../origin/040-CSS三大特性/assets/images/css-inheritance-img-001-c2fc84.png)

- `a` 標籤的 `color` 會繼承失效。
- `h` 系列標籤的 `font-size` 會繼承失效。

```css
div {
  color: red;
  font-size: 12px;
}
```

```html
<div>
  <a href="#">超鏈接</a>
  <h1>一級標題</h1>
</div>
```

# 優先級練習題

權重相同時，執行層疊性；選擇器不同時，根據選擇器權重執行。

## 示例 1

```css
/* (行内，id, 类，标签) */
/* (0, 0, 0, 2) */
div div {
  color: skyblue;
}

/* (0, 0, 0, 1) */
div {
  color: red;
}
```

```html
<div>
  <div>
    <div>白日依山尽，黄河入海流。</div>
  </div>
</div>
```

## 示例 2

```css
/* (行内，id, 类，标签) */
/* (0, 0, 0, 7) */
div div div div div div div {
  color: red;
}

/* (0, 0, 1, 0) */
.one {
  color: green;
}
```

```html
<div>
  <div>
    <div>
      <div>
        <div>
          <div>
            <div class="one">白日依山尽，黄河入海流。</div>
          </div>
        </div>
      </div>
    </div>
  </div>
</div>
```

## 示例 3

```css
/* (行内，id, 类，标签) */
/* (0, 0, 2, 1) */
.c1 .c2 div {
  color: red;
}

/* (0, 1, 0, 1) */
div #box3 {
  color: green;
}

/* (0, 1, 0, 1) */
#box1 div {
  color: blue;
}
```

```html
<div id="box1" class="c1">
  <div id="box2" class="c2">
    <div id="box3" class="c3">白日依山尽，黄河入海流。</div>
  </div>
</div>
```

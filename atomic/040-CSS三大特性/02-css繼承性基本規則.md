# CSS 繼承性基本規則

有些 CSS 樣式會繼承。元素如果本身設定了某個樣式，就使用本身設定的樣式；如果本身沒有設定某個樣式，會從父元素開始一級一級地繼承。

繼承時會優先繼承離得近的祖先元素。

## 會繼承的 CSS 屬性

- 字體屬性
- 文本屬性，除了 `vertical-align`
- 文字顏色等

## 不會繼承的 CSS 屬性

- 邊框
- 背景
- 內邊距
- 外邊距
- 寬高
- 溢出方式等

CSS 中的繼承是指子標籤會繼承父標籤的某些樣式，例如文本顏色和字號。恰當使用繼承可以簡化代碼，降低 CSS 樣式的複雜性。

一個粗略規律是：能繼承的屬性通常偏向文字與排版呈現，較少直接控制盒子模型。不過這只是記憶方向，實際是否繼承仍要以各 CSS 屬性的定義為準。

```css
body {
    /* 1.5 的意思：讓裡面所有的子元素的行高是當前文字大小的 1.5 倍。 */
    font: 12px/1.5 Microsoft YaHei
}
```

子元素可以繼承父元素的部分樣式：

- `color` 可以繼承。
- 多數字體相關屬性可以繼承，例如 `font-size`、`font-family`。
- `line-height` 可以繼承。
- 部分文本排版屬性可以繼承，但不能只用 `text-`、`font-` 或 `line-` 這類前綴判斷。

```css
body {
  font: 12px/1.5 Microsoft YaHei
}

div {
  color: pink;
  font-size: 50px;
}
```

```html
<div>
  <p>龙生龙，凤生凤，老鼠生的孩子会打洞</p>
</div>
```

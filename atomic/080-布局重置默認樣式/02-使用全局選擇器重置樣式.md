# 使用全局選擇器重置樣式

使用全局選擇器 `*` 可以在簡單案例中重置所有元素的部分樣式。

```css
* {
  margin: 0;
  padding: 0;
  /* ..... */
}
```

這種方法在實際開發中通常不會使用，因為 `*` 選擇的是所有元素，但並不是所有元素都有需要重置的默認樣式。

此外，重置樣式時有時需要做特定處理，例如希望 `a` 元素文字是灰色，而其他元素文字是藍色。全局選擇器無法細緻區分這些需求。

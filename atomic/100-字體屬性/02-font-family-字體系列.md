# font-family 字體系列

CSS 使用 `font-family` 屬性定義文本的字體系列。

注意點：

1. 各種字體之間必須使用英文狀態下的逗號隔開。
2. 一般情況下，如果字體名稱由多個單詞組成，建議加引號。
3. 字體清單最後建議放一個通用字體族，例如 `serif`、`sans-serif` 或 `monospace`，讓瀏覽器在指定字體都不可用時仍有合理的後備字體。

渲染規則：

1. 瀏覽器會從左往右按照順序查找可用字體；如果某個字體不可用，或缺少特定字形，會繼續嘗試下一個字體。
2. 如果前面的具名字體都不適用，瀏覽器會使用最後指定的通用字體族所對應的系統後備字體。

```css
div {
  font-family: Arial, "Microsoft YaHei", "微软雅黑", sans-serif;
}
```

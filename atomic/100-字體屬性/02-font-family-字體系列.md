# font-family 字體系列

CSS 使用 `font-family` 屬性定義文本的字體系列。

注意點：

1. 各種字體之間必須使用英文狀態下的逗號隔開。
2. 一般情況下，如果有空格隔開的多個單詞組成的字體，要加引號。
3. 盡量使用系統默認自帶字體，保證在任何用戶的瀏覽器中都能正確顯示。

渲染規則：

1. 從左往右按照順序查找，如果電腦中未安裝該字體，則顯示下一個字體。
2. 如果都不支持，此時會根據操作系統，顯示最後字體系列的默認字體。

```css
div {
  font-family: Arial, "Microsoft Yahe", "微软雅黑";
}
```

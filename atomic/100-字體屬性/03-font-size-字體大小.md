# font-size 字體大小

CSS 使用 `font-size` 屬性定義字體大小。

重點：

1. 單位需要設置，否則無效。
2. `px` 大小是網頁最常用的單位。
3. 谷歌瀏覽器默認的文字大小為 `16px`。
4. 不同瀏覽器可能默認顯示的字號大小不一致，因此盡量給一個明確值，不要使用默認大小。

```css
body {
  font-size: 16px;
}

/* 標題標籤比較特殊，需要單獨指定文字大小 */
h2 {
  font-size: 16px;
}
```

```html
<h2>pink的秘密</h2>
<p>那一抹众人中最漂亮的颜色,</p>
<p>优雅,淡然,又那么心中清澈.</p>
<p>前端总是伴随着困难和犯错,</p>
<p>静心,坦然,攻克一个又一个.</p>
<p>拼死也要克服它,</p>
<p>这是pink的秘密也是老师最终的嘱托.</p>
```

可以給 `body` 指定整個頁面文字的大小。

```css
body {
  font-size: 16px;
}
```

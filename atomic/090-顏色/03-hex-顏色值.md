# HEX 顏色值

用 HEX 值的特定格式來表示顏色值，是顏色定義中最常見的方法。

HEX 值由符號 `#` 和十六進制組合而成，表示形式為 `#RRGGBB`。

其中，`R` 表示紅色，`G` 表示綠色，`B` 表示藍色，其值介於 `00` 到 `ff` 之間，用於指定顏色的強度。

![HEX 色值範例，紅色背景顯示 #ff0b00](../../origin/090-顏色/assets/images/colors-img-003-8da288.png)

> 注意：當 `#RRGGBB` 滿足兩個 `R`、兩個 `G` 和兩個 `B` 同時相等時，可以簡寫為 `#RGB`。例如 `#aabbcc` 可以簡寫為 `#abc`。

# 類選擇器

類選擇器會選中具有指定 `class` 屬性的元素。CSS 中以英文句點 `.` 開頭，HTML 中則透過 `class` 屬性標記。

```css
.red {
    color: red;
}
```

```html
<div class="red">這個 div 會套用 red 類別樣式</div>
```

類名的常見命名建議：

- 使用有語意的名稱，例如 `.nav`、`.header`、`.footer`。
- 可以用連字號連接多個單字，例如 `.star-sing`。
- 避免使用純數字、中文或特殊符號作為類名。

類選擇器比標籤選擇器更精準，也更適合在不同標籤之間重複使用同一組樣式。

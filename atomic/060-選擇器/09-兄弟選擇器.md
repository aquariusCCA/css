# 兄弟選擇器

兄弟選擇器用來選中與某個元素位於同一父層底下的後方兄弟元素。

相鄰兄弟選擇器 `+` 只會選中緊接在前一個元素後面的第一個兄弟元素：

```css
h1 + p {
    color: red;
}
```

```html
<h1>標題</h1>
<p>這個段落會被選中</p>
<p>這個段落不會被 h1 + p 選中</p>
```

也可以用於連續元素：

```css
li + li {
    border-top: 1px solid #ccc;
}
```

一般兄弟選擇器 `~` 會選中指定元素之後所有符合條件的兄弟元素：

```css
h1 ~ p {
    color: blue;
}
```

```html
<h1>標題</h1>
<p>第一個後方兄弟段落</p>
<p>第二個後方兄弟段落</p>
```

`+` 強調「下一個」，`~` 強調「後方所有同層符合條件的元素」。

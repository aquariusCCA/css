# 選擇器的作用與分類

CSS 選擇器用來根據需求選出指定的 HTML 標籤，讓後續樣式只套用在目標元素上。

可以把 CSS 規則理解成兩個步驟：

1. 找到標籤：選擇器負責「找對人」。
2. 設定樣式：宣告區塊負責「做對事」。

![CSS 規則示意圖，標示 h1 選擇器與 color、font-size 宣告](../../origin/060-選擇器/assets/images/selectors-purpose-img-001-3ac98e.png)

選擇器可以分成兩大類：

- 基礎選擇器：由單一選擇器組成，例如標籤選擇器、類選擇器、id 選擇器與通配符選擇器。
- 複合選擇器：在基礎選擇器之上組合而成，用來表達更精準的元素關係或狀態。

# 鏈接偽類選擇器

偽類選擇器用冒號 `:` 表示，用來選中元素的特定狀態或位置，例如滑過、被選中、第一個子元素等。

鏈接偽類選擇器常用於 `<a>` 的不同狀態：

```css
a:link {
    color: #333;
}

a:visited {
    color: purple;
}

a:hover {
    color: red;
}

a:active {
    color: orange;
}
```

四個常見狀態分別是：

- `:link`：尚未訪問的連結。
- `:visited`：已訪問的連結。
- `:hover`：滑鼠移到連結上時。
- `:active`：連結被按下但尚未放開時。

書寫時通常遵守 LVHA 順序：`:link`、`:visited`、`:hover`、`:active`。這樣可以避免相同權重下，後寫的狀態覆蓋前面狀態造成效果不如預期。

```html
<a href="#">站內示範連結</a>
<a href="http://www.xxxxxxxx.com">外部示範連結</a>
```

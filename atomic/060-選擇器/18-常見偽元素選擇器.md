# 常見偽元素選擇器

偽元素選擇器可以用 CSS 建立或選取元素的某個局部表現。這些內容並不是原本 HTML 文件中實際寫出的元素，因此稱為偽元素。

常見偽元素：

```css
p::first-letter {
    font-size: 32px;
    color: red;
}
```

`::first-letter` 會選中第一個字母或文字。

```css
p::first-line {
    color: blue;
}
```

`::first-line` 會選中第一行文字。

```css
p::selection {
    background-color: pink;
    color: #fff;
}
```

`::selection` 會控制使用者選取文字時的樣式。

偽元素選擇器的權重通常與標籤選擇器相同，權重值記為 1。

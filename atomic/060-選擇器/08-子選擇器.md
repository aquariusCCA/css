# 子選擇器

子選擇器使用大於符號 `>` 表示，只會選中某個元素的直接子元素。

```css
元素1 > 元素2 {
    樣式宣告
}
```

例如：

```css
.nav > a {
    color: red;
}
```

```html
<div class="nav">
    <a href="#">直接子元素連結</a>
    <p>
        <a href="#">不是直接子元素的連結</a>
    </p>
</div>
```

在這個例子中，`.nav > a` 只會選中 `.nav` 的直接子元素 `<a>`，不會選中包在 `<p>` 裡面的 `<a>`。

子選擇器比後代選擇器更嚴格，適合用在元素層級關係需要明確限定的情境。

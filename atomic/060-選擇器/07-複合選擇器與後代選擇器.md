# 複合選擇器與後代選擇器

複合選擇器是建立在基礎選擇器之上的組合寫法，通常由兩個或更多基礎選擇器組成，用來更精準地選中元素。

常見的複合選擇器包含：

- 後代選擇器
- 子選擇器
- 並集選擇器
- 交集選擇器
- 偽類選擇器

後代選擇器使用空格表示元素之間的祖先與後代關係：

```css
元素1 元素2 {
    樣式宣告
}
```

它會選中 `元素1` 裡面所有符合 `元素2` 的後代元素，不限於直接子元素。

```css
ol li {
    color: pink;
}

ol li a {
    color: red;
}

.nav li a {
    color: yellow;
}
```

```html
<ul class="nav">
    <li><a href="#">連結一</a></li>
    <li><a href="#">連結二</a></li>
</ul>
```

後代選擇器的關鍵是中間的空格：空格表示「在某個祖先元素裡面的所有指定後代」。

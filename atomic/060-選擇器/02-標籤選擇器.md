# 標籤選擇器

標籤選擇器使用 HTML 標籤名稱作為選擇器，會選中頁面中所有同名標籤。

基本語法：

```css
標籤名 {
    屬性名: 屬性值;
}
```

例如，以下 CSS 會讓所有 `<p>` 變成綠色，所有 `<div>` 變成粉色：

```css
p {
    color: green;
}

div {
    color: pink;
}
```

標籤選擇器適合設定同一類元素的共同樣式。它的影響範圍較大，因此若只想選中某幾個特定元素，通常會改用類選擇器或其他更精準的選擇器。

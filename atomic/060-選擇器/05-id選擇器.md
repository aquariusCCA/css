# id 選擇器

id 選擇器會選中具有指定 `id` 屬性的元素。CSS 中以井字號 `#` 開頭，HTML 中透過 `id` 屬性指定。

```css
#pink {
    color: pink;
}
```

```html
<div id="pink">這個 div 會套用 pink id 樣式</div>
```

在同一份 HTML 文件中，`id` 應該是唯一的。也就是說，同一個 `id` 名稱不應重複套用在多個元素上。

類選擇器與 id 選擇器的常見差異：

- 類選擇器可以重複使用，像人的名字，可以很多人同名。
- id 選擇器應保持唯一，像身分證字號，通常只對應一個元素。

實務上，`id` 也常用於 JavaScript 精準取得特定元素。

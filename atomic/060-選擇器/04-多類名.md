# 多類名

同一個 HTML 元素可以同時擁有多個類名。多個類名寫在同一個 `class` 屬性中，彼此用空格分隔。

```css
.red {
    color: red;
}

.font35 {
    font-size: 35px;
}
```

```html
<div class="red font35">Kevin</div>
```

多類名常用來拆分樣式職責：

- 公共樣式：多個元素都會使用的樣式。
- 個別樣式：只針對某些元素補上的差異。

這種寫法可以減少重複 CSS，也能讓樣式組合更彈性。

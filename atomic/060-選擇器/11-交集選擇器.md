# 交集選擇器

交集選擇器會選中同時符合多個條件的元素。多個選擇器之間不加空格，也不加其他符號。

```css
p.box {
    color: red;
}
```

```html
<p class="box">這個 p 會被選中</p>
<div class="box">這個 div 不會被 p.box 選中</div>
```

`p.box` 的意思是：選中「同時是 `<p>`，且 class 包含 `box`」的元素。

如果交集選擇器中包含標籤選擇器，標籤選擇器通常要寫在最前面：

```css
標籤名.類名 {
    樣式宣告
}
```

交集選擇器常用於在已有類別的基礎上，再限定元素類型或其他條件。

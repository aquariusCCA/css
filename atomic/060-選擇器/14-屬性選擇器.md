# 屬性選擇器

屬性選擇器可以根據元素是否具有某個屬性，或某個屬性的值是否符合條件來選中元素。

![CSS 屬性選擇器語法表，展示 presence、等於、開頭與結尾匹配等寫法](../../origin/060-選擇器/assets/images/compound-selectors-img-001-546d82.png)

常見寫法：

```css
input[value] {
    color: red;
}
```

選中帶有 `value` 屬性的 `<input>`。

```css
input[type="text"] {
    color: pink;
}
```

選中 `type` 屬性值等於 `text` 的 `<input>`。

```css
div[class^="icon"] {
    color: red;
}
```

選中 `class` 屬性值以 `icon` 開頭的 `<div>`。

```css
section[class$="data"] {
    color: blue;
}
```

選中 `class` 屬性值以 `data` 結尾的 `<section>`。

屬性選擇器的權重與類選擇器、偽類選擇器相同，權重值通常記為 10。

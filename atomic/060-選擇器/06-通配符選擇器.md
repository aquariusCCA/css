# 通配符選擇器

通配符選擇器使用 `*` 表示，會選中頁面中的所有元素。

```css
* {
    margin: 0;
    padding: 0;
}
```

通配符選擇器不需要在 HTML 中額外呼叫，因為它會自動匹配所有標籤。

常見用途是做全域初始化，例如清除瀏覽器預設的外距與內距：

```css
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
```

由於影響範圍是整個頁面，使用時要注意不要意外覆蓋過多元素的預設行為。

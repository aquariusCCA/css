# 並集選擇器

並集選擇器使用逗號 `,` 分隔多個選擇器，表示同時選中多組元素並套用同一組樣式。

```css
div,
p {
    color: red;
}
```

上例會同時選中所有 `<div>` 與所有 `<p>`。

也可以混合不同種類的選擇器：

```css
div,
p,
.pig li {
    color: pink;
}
```

並集選擇器適合處理多個元素需要相同樣式的情境，可以避免重複書寫相同宣告。

為了提升可讀性，實務上常把每個選擇器獨立寫在一行，最後一個選擇器後面不加逗號。

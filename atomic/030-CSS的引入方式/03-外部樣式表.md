# 外部樣式表

實際開發通常會使用外部樣式表，適合於樣式比較多的情況。

外部樣式表的核心，是將樣式單獨寫到 CSS 文件中，之後再把 CSS 文件引入 HTML 頁面中使用。

引入外部樣式表分為兩步：

- Step 1：新建一個後綴名為 `.css` 的樣式文件，並把所有 CSS 代碼都放入此文件中。
- Step 2：在 HTML 頁面中，使用 `<link>` 標籤引入這個文件。

```css
/* 這 style.css 文件裡面只有樣式沒有標籤 */
div {
    color:  pink;
}
```

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <link rel="stylesheet" href="./style.css">
</head>
<body>
  <div>來呀~快活呀,反正有大把時間...</div>
</body>
</html>
```

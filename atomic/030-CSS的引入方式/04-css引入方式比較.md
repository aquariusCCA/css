# CSS 引入方式比較

CSS 常見的三種引入方式是外部樣式、內部樣式和行內樣式。它們的書寫位置、作用範圍與使用場景不同。

![CSS 三種引入方式比較表，列出書寫位置、作用範圍與使用場景](../../origin/030-CSS的引入方式/assets/images/ways-to-include-css-img-001-e7905f.png)

## 外部樣式

作用範圍是多個頁面。

缺點：

- 需要引入才能使用。

優點：

- 樣式可多個頁面複用。
- 代碼結構清晰。
- 可觸發瀏覽器的緩存機制。
- 結構與樣式徹底分離。

## 內部樣式

作用範圍是當前頁面。

缺點：

- 結構與樣式未徹底分離。
- 樣式不能多個頁面複用。

優點：

- 樣式可複用。
- 代碼結構清晰。

## 行內樣式

作用範圍是當前標籤。

缺點：

- 結構與樣式未分離。
- 代碼結構混亂。
- 樣式不能複用。

優點：

- 優先級最高。

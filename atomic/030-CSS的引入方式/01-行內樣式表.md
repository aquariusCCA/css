# 行內樣式表

行內樣式表是在元素標籤內部的 `style` 屬性中設定 CSS 樣式，適合用來修改簡單樣式。

行內樣式表的重點如下：

- `style` 其實就是標籤的屬性。
- 在雙引號中間，寫法要符合 CSS 規範。
- 行內樣式表可以控制當前標籤的樣式。

```html
<p style="color: pink; font-size: 20px;">給我一個粉紅的回憶</p>
```

由於行內樣式表書寫繁瑣，並且沒有體現出結構與樣式相分離的思想，所以不推薦大量使用。只有在需要對當前元素添加簡單樣式時，才可以考慮使用。

# img 標籤和背景圖片的區別

`img`：

1. 不設置高寬也會默認顯示。
2. 重要突出的圖使用 `img`。

`background-image`：

1. 需要設置元素尺寸。
2. 裝飾性圖片使用背景圖。

如果需要在網頁中展示一張圖片的效果，有兩種方法：

1. 直接寫上 `img` 標籤。
   `img` 標籤是一個標籤，不設置寬高默認會以原尺寸顯示。
2. 使用 `div` 標籤加背景圖片。
   需要設置 `div` 的寬高，因為背景圖片只是裝飾的 CSS 樣式，不能撐開 `div` 標籤。

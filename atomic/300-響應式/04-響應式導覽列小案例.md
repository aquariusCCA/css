```html

<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  <style>
    * {
      margin: 0;
      padding: 0;
    }

      margin: 0 auto;
    }

      width: 93.75px;

    }

        width: 100%;
      }

        height: 33.33%;
      }
    }
  </style>
</head>
```

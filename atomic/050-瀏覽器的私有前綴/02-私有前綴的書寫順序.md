# 私有前綴的書寫順序

如果因支援舊瀏覽器而必須保留私有前綴，通常先寫帶有私有前綴的屬性，再寫標準屬性。

以下為歷史相容寫法示例；現代 `border-radius` 通常不需要手動加前綴。

```css
-moz-border-radius: 10px;

-webkit-border-radius: 10px;

-o-border-radius: 10px;

border-radius: 10px;
```

這種順序會讓支援標準屬性的瀏覽器優先採用最後的標準寫法。

# 什麼是瀏覽器私有前綴

W3C 標準提出某個 CSS 特性後，在瀏覽器正式支持之前，瀏覽器廠商會根據自己的瀏覽器內核，使用私有前綴來測試該 CSS 特性。

當瀏覽器正式支持該 CSS 特性之後，就不需要再使用私有前綴。

瀏覽器的私有前綴（Private Prefix）是指瀏覽器廠商在實現 HTML、CSS 和 JavaScript 等標準規範時，使用自己的前綴字母作為標識的一種方式。

私有前綴主要用來實驗和測試新功能。這些功能還未被標準化，或尚未被所有瀏覽器支援。私有前綴可以區分實驗性功能和標準功能，使開發人員避免誤用尚未正式納入標準的功能，降低兼容性問題。

常見私有前綴如下：

- `moz-`：代表 Firefox 瀏覽器的私有屬性。
- `ms-`：代表 IE 瀏覽器的私有屬性。
- `webkit-`：代表 Safari、Chrome 瀏覽器的私有屬性。
- `o-`：代表 Opera 瀏覽器的私有屬性。

瀏覽器私有前綴是為了兼容老版本的寫法，比較新版本的瀏覽器無須添加。

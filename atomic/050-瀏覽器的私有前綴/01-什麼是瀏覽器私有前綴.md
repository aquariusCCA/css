# 什麼是瀏覽器私有前綴

過去當某個 CSS 特性尚未穩定或尚未標準化時，瀏覽器廠商可能會根據自己的瀏覽器內核，使用私有前綴來提供實驗性實作。

現代瀏覽器通常會把實驗性功能放在旗標或設定後面，不再把私有前綴當作主要的實驗機制；但閱讀舊程式碼時，仍常會遇到這類前綴。

當瀏覽器與規格都穩定支持該 CSS 特性後，通常應優先使用不帶前綴的標準寫法。

瀏覽器的私有前綴（Vendor Prefix）是指瀏覽器廠商在實作 CSS 等 Web 標準或非標準功能時，使用自己的前綴字串作為識別的一種方式。

私有前綴主要用來標示實驗性、非標準或曾經不完全穩定的功能，讓開發者能針對特定瀏覽器實作做相容處理；實務上不應把只帶某一家前綴的寫法當成跨瀏覽器標準功能。

常見私有前綴如下：

- `-webkit-`：代表 WebKit/Blink 系瀏覽器常見的前綴，例如 Safari、Chrome、Chromium 系瀏覽器與 iOS 瀏覽器。
- `-moz-`：代表 Firefox 瀏覽器的私有前綴。
- `-ms-`：代表 IE 與舊版 Edge 的私有前綴。
- `-o-`：代表舊版 Presto Opera 的私有前綴。

瀏覽器私有前綴是為了兼容老版本的寫法，比較新版本的瀏覽器無須添加。

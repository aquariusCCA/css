# 浮動元素脫離標準流

浮動元素不能透過 `text-align: center` 或 `margin: 0 auto` 實現水平居中。

浮動元素會脫離標準流。

![浮動元素脫離標準流並漂浮在普通流上方的示意圖](../../origin/140-CSS浮動/assets/images/float-characteristics-img-001-73cc54.png)

脫離標準流的控制，移動到指定位置，俗稱脫標。

浮動的盒子不再保留原先的位置。

```css
.box1 {
  float: left;
  width: 200px;
  height: 200px;
  background-color: pink;
}

.box2 {
  width: 300px;
  height: 300px;
  background-color: rgb(0, 153, 255);
}
```

```html
<div class="box1">浮动的盒子</div>
<div class="box2">标准流的盒子</div>
```

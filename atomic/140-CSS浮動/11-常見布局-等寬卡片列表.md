# 常見布局：等寬卡片列表

等寬卡片列表可以使用父級固定寬度，子級 `li` 浮動排列。

最後一個卡片通常需要清除右側間距；範例中使用 `.box .last`，是為了讓選擇器權重足夠覆蓋前面的 `.box li`。

```html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    * {
      margin: 0;
      padding: 0;
    }

    li {
      list-style: none;
    }

    .box {
      width: 1226px;
      height: 285px;
      background-color: pink;
      margin: 0 auto;
    }

    .box li {
      width: 296px;
      height: 285px;
      background-color: purple;
      float: left;
      margin-right: 14px;
    }

    /* 這裡必須寫 .box .last，要注意權重問題 */
    .box .last {
      margin-right: 0;
    }
  </style>
</head>

<body>
  <ul class="box">
    <li>1</li>
    <li>2</li>
    <li>3</li>
    <li class="last">4</li>
  </ul>
</body>

</html>
```

# 為什麼需要浮動：塊級元素水平排列

浮動最典型的應用，是讓多個塊級元素在一行內排列顯示。

在傳統浮動布局的語境中，常見做法是：

- 多個塊級元素縱向排列時，找標準流。
- 多個塊級元素橫向排列時，可以使用浮動。

很多布局效果，標準流沒有辦法完成，此時可以利用浮動完成布局。因為浮動可以改變元素標籤預設的排列方式。現代頁面布局中，橫向排列通常也會優先考慮 Flexbox 或 Grid。

問題：如何讓多個塊級盒子 `div` 水平排列成一行？

![三個 div 透過浮動排列在同一行的示意圖](../../origin/140-CSS浮動/assets/images/why-do-we-need-float-img-001-432417.png)

把元素轉換成行內塊雖然可以實現一行顯示，但元素之間會出現較大的空白縫隙，很難控制。

```css
div {
  display: inline-block;
  /* float: left; */
  width: 150px;
  height: 200px;
  background-color: pink;
}
```

```html
<div>1</div>
<div>2</div>
<div>3</div>
```

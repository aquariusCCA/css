# 浮動元素具有行內塊特性

任何元素都可以浮動。不管原先是什麼顯示模式，添加浮動後都具有類似行內塊元素的特性。

浮動元素的特性包含：

- 如果塊級盒子沒有設定寬度，預設寬度和父級一樣寬；但添加浮動後，它的大小會根據內容決定。
- 如果行內元素添加了浮動，則不需要轉換成塊級或行內塊元素，就可以直接設定高度和寬度。
- 浮動盒子中間沒有縫隙，會緊挨在一起。
- 浮動元素不會產生 margin 合併，也不會產生 margin 塌陷，可以設定四個方向的 margin 和 padding。

```css
/* 任何元素都可以浮動。添加浮動後，具有類似行內塊元素的特性。 */
span,
div {
  float: left;
  width: 200px;
  height: 100px;
  background-color: pink;
}

/* 行內元素有了浮動，就可以直接給高度和寬度 */
a {
  float: right;
  width: 200px;
  height: 200px;
  background-color: purple;
}
```

```html
<span>1</span>
<span>2</span>
<div>div</div>
<a href="">aaaaa</a>
```

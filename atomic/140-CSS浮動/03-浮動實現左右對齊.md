# 浮動實現左右對齊

浮動可以讓兩個盒子分別貼齊父盒子的左右兩側。

![左浮動和右浮動讓兩個盒子分別貼齊父盒子兩側的示意圖](../../origin/140-CSS浮動/assets/images/why-do-we-need-float-img-002-7020ef.png)

```css
.left,
.right {
  width: 200px;
  height: 200px;
  background-color: pink;
}

.left {
  float: left;
}

.right {
  float: right;
}
```

```html
<div class="left">左青龙</div>
<div class="right">右白虎</div>
```

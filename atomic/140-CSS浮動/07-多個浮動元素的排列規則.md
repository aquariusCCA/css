# 多個浮動元素的排列規則

如果多個盒子都設定了浮動，它們會按照屬性值在一行內顯示，並且頂端對齊排列。

![多個浮動盒子在同一行頂端對齊排列的示意圖](../../origin/140-CSS浮動/assets/images/float-characteristics-img-002-afa13a.png)

浮動元素會相互貼靠在一起，不會有瀏覽器自動產生的空白縫隙。如果父級寬度裝不下這些浮動盒子，後續浮動盒會移到能放下的位置，通常會換到下一行；實際位置仍會受前面浮動盒的高度與剩餘空間影響。

```css
div {
  float: left;
  width: 200px;
  height: 200px;
  background-color: pink;
}

.two {
  background-color: purple;
  height: 249px;
}

.four {
  background-color: skyblue;
}
```

```html
<div>1</div>
<div class="two">2</div>
<div>3</div>
<div class="four">4</div>
```

# 傳統布局方式與標準流

網頁布局的本質，是用 CSS 來擺放盒子，把盒子放到相應的位置。

CSS 提供三種傳統布局方式：

- 普通流，也稱為標準流。
- 浮動。
- 定位。

標準流是標籤按照預設規則排列的方式。

塊級元素會獨占一行，從上到下順序排列。常用塊級元素包含 `div`、`hr`、`p`、`h1` 到 `h6`、`ul`、`ol`、`dl`、`form`、`table`。

行內元素會按照順序，從左到右排列，碰到父元素邊緣時會自動換行。常用行內元素包含 `span`、`a`、`i`、`em`。

標準流是最基本的布局方式。三種布局方式都是用來擺放盒子，盒子擺放到合適位置，布局就自然完成。

實際開發中，一個頁面通常會同時包含標準流、浮動與定位這三種布局方式。

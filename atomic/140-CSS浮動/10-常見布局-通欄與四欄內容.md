# 常見布局：通欄與四欄內容

常見網頁布局可以由 top、banner、內容區與 footer 組成。

![包含 top banner main footer 的常見網頁布局示意圖](../../origin/140-CSS浮動/assets/images/common-web-layouts-img-001-673e45.png)

![包含 top banner 多欄內容區與 footer 的常見網頁布局示意圖](../../origin/140-CSS浮動/assets/images/common-web-layouts-img-002-5b71b7.png)

以下範例使用標準流控制上下結構，並在內容區使用浮動排列四欄。

```html
<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    * {
      margin: 0;
      padding: 0;
    }

    li {
      list-style: none;
    }

    .top {
      height: 50px;
      background-color: gray;
    }

    .banner {
      width: 980px;
      height: 150px;
      background-color: gray;
      margin: 10px auto;
    }

    .box {
      width: 980px;
      margin: 0 auto;
      height: 300px;
      background-color: pink;
    }

    .box li {
      float: left;
      width: 237px;
      height: 300px;
      background-color: gray;
      margin-right: 10px;
    }

    .box .last {
      margin-right: 0;
    }

    /* 通欄盒子和瀏覽器一樣寬，不需要指定寬度 */
    .footer {
      height: 200px;
      background-color: gray;
      margin-top: 10px;
    }
  </style>
</head>

<body>
  <div class="top">top</div>
  <div class="banner">banner</div>
  <div class="box">
    <ul>
      <li>1</li>
      <li>2</li>
      <li>3</li>
      <li class="last">4</li>
    </ul>
  </div>
  <div class="footer">footer</div>
</body>

</html>
```

# CSS 語法規範

使用 HTML 時需要遵守一定的規範，CSS 也是如此。要熟練地使用 CSS 修飾網頁，首先需要了解 CSS 的樣式規則。

CSS 規則主要由兩個部分構成：

- 選擇器
- 一條或多條聲明

![CSS 語法規則示意圖，標示 h1 選擇器、宣告區塊與屬性名和值](../../origin/010-CSS簡介/assets/images/css-introduction-img-001-3ac98e.png)

```css
选择器 {
    属性名: 属性值
    ...
}
```

CSS 聲明的重點如下：

- 屬性和屬性值以「鍵值對」的形式出現。
- 屬性是對指定對象設定的樣式，例如字體大小、文本顏色等。
- 屬性和屬性值之間使用英文冒號 `:` 分開。
- 多個鍵值對之間使用英文分號 `;` 區分。

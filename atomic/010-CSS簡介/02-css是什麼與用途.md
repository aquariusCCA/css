# CSS 是什麼與用途

> CSS 也是一種標記語言。

CSS 是層疊樣式表的簡稱，也常被稱為 CSS 樣式表或級聯樣式表。

CSS 主要用來設定 HTML 頁面中的文字內容、圖片外型、版面布局，以及整體外觀顯示樣式。

簡單來說，CSS 可以美化 HTML，讓網頁更豐富、更漂亮，也讓版面布局更靈活。

CSS 的最大價值，是讓 HTML 專注於結構呈現，讓樣式交給 CSS 管理，也就是把結構與樣式分離。

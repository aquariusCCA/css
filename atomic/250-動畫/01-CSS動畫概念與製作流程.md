# CSS 動畫概念與製作流程

> 動畫 animation 是 CSS3 中具有顛覆性的特徵之一，可通過設置多個節點來精確控制一個或一組動畫，常用來實現複雜的動畫效果；相比較過渡，動畫可以實現更多變化、更多控制、連續自動播放等效果。

## 過渡 V.S 動畫

- 過渡：用於兩個狀態變化過程。
- 動畫：實現多個狀態間的變化過程，動畫過程可控。

## 動畫

- 動畫的本質：快速切換大量圖片時在人腦中形成的具有連續性的畫面。
- 構成動畫的最小單元：幀或動畫幀。

## 製作動畫的兩個步驟

- 先定義動畫。
- 再使用（調用）動畫。

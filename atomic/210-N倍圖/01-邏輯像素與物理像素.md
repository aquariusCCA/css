# 邏輯像素與物理像素

- 參考文章
    - [谈谈web中多倍图](https://juejin.cn/post/7087441121541881892)

其實早期是沒有這個概念的，起源於蘋果。

當我們寫下這行程式碼：

```css
div {
  width:200px;    
}
```

這裡的 `200px` 是**邏輯**意義上的像素，瀏覽器會轉化成相應的**物理**意義的真實像素點到一個 `div` 容器上。

寫在 `css` 程式碼裡的像素稱為**邏輯像素**，而最終繪製出來的像素點稱為**物理像素**。在 PC 端或早期普通手機螢幕中，常見情境是：

```text
1CSS像素 = 1 物理像素
```

但我們開發時候的 `1px` 不是一定等於 1 個物理像素，尤其在行動端就不盡相同。

**物理像素點**指的是螢幕顯示的最小顆粒，是物理真實存在的。這是廠商在出廠時就設置好了，比如蘋果 6 / 7 / 8 是 `750 * 1334`。

# 二倍圖的製作與顯示方式

舉例：當一個 50 × 50 像素（CSS 像素）的圖片直接放到 iPhone 6/7/8 設備中顯示時，圖片會被放大，變成原來的兩倍。

iPhone 6/7/8 的設備像素比為 2，也就是 100 × 100。

為了避免圖片放大後模糊，我們可以預先製作一張 100 × 100 的圖片，然後在網頁中手動設置這個圖片的樣式，將圖片的寬和高都設為 50 像素。

這樣，這張圖片就會以它原本的像素（100 × 100）來顯示，保證圖片的原有清晰度，這樣就不會造成圖片模糊。

我們準備的圖片，比我們實際需要的大小大 2 倍，這種方式就是二倍圖。

## 範例程式碼

```css
img:nth-child(2) {
	width: 50px;
	height: 50px;
}
```

```html
<body>
	<!-- 原图 -->
	<img src="./assets/images/n-times-image-img-002-702660.jpg" alt="綠底白色 Apple 標誌原圖">

	<!-- 二倍图 -->
	<img src="./assets/images/n-times-image-img-003-e2d385.jpg" alt="綠底白色 Apple 標誌二倍圖">
</body>
```

# 設備像素比與 Retina 螢幕

蘋果最受歡迎的業務恐怕是手機了。相比 PC 和平板，手機的螢幕是比較小的，如何在有限的物理尺寸下讓螢幕更加炫彩奪目呢？

蘋果想到了一招：增加單位面積內的像素數，也就是在螢幕尺寸不變的前提下增加可容納的物理像素數。像素點越密集，圖像的細節自然就越多，圖像也就更清晰、更絢麗。

我們熟悉的 Mac-pro、iPhoneXR 的物理像素增加了一倍。

例如，一個 `100*100` 尺寸的圖片在 Mac-pro 下的真實像素是 `200*200`，如果使用了實際像素為 `100*100` 的圖像，相當於少了一半的圖像細節，自然會比較模糊。

因此，為了圖像的清晰展示，需要製作相應的 n 倍圖。

一個 `px` 能顯示的物理像素點的個數，稱為**物理像素比**或**螢幕像素比**。

- 當設備像素比很大時，圖片會被放大，而放大會讓圖片看起來模糊。
- Retina（視網膜螢幕）是一種顯示技術，可以將更多的物理像素點壓縮至一塊螢幕裡，從而達到更高的解析度，並提高螢幕顯示的細膩程度。
- 由於 Retina 的出現，對於一張 50px * 50px 的圖片，在手機 Retina 螢幕中打開，按照物理像素比會放大倍數，這樣會造成圖片模糊。

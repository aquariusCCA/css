# 平面轉換與二維座標

`轉換(transform)`：可以簡單理解為變形，有 2D 和 3D 之分。

2D 轉換是改變標籤在二維平面上的位置和形狀的一種技術，可以實現元素的位移、旋轉、縮放等效果。

- 移動：`translate`
- 旋轉：`rotate`
- 縮放：`scale`

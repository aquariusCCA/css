# 第四章 CSS 三大特性

本章整理 CSS 的核心特性，幫助你理解瀏覽器在面對多條樣式規則時，最後是怎麼決定畫面結果的。

目前這一章已整理的內容包含「層疊性」「繼承性」與「優先級」，重點在建立樣式衝突、樣式傳遞與規則比較三個基礎觀念。

## 本章和整體學習地圖的關係

- 這一章承接前面的 CSS 基礎語法，開始進入 CSS 規則如何真正生效的核心觀念。
- 先理解層疊性、繼承性與優先級，之後讀選擇器、權重或覆蓋關係時會更容易。
- 這一章適合作為從「會寫 CSS」進一步走向「看得懂 CSS 為什麼這樣生效」的起點。

# 文本溢出

> 返回章節首頁：[README.md](./README.md)
>
> `text-overflow` 用於處理文本溢出容器時的顯示方式。

## 導讀
- `text-overflow` 常和 `overflow`、`white-space` 一起使用。
- 單行文本溢出時，最常見的是顯示省略號。
- 多行文本省略號屬於常見技巧，但依賴特定瀏覽器能力。

## 關鍵字
- text-overflow
- 文本溢出
- 省略號
- overflow
- white-space

## 30 秒複習入口
- `text-overflow` 用來設定文本溢出時怎麼顯示
- `clip` 代表裁剪
- `ellipsis` 代表省略號
- 單行省略號通常要配合 `overflow: hidden` 與 `white-space: nowrap`
- 多行省略號常用 `-webkit-line-clamp`

## 速查區

| 值 | 說明 |
| --- | --- |
| `clip` | 裁剪溢出的文本 |
| `ellipsis` | 用省略號表示溢出的文本 |

## 正文
`text-overflow` 屬性用來規定當文本溢出包含元素時，應該如何顯示。

常見值如下：

1. `clip`：裁剪文本，這是默認值。
2. `ellipsis`：使用省略號表示被修剪的文本。

```css
.a {
  width: 100px;
  height: 20px;
  font: 8px/20px arial;
  border: 1px solid black;

  /* 超出元素框部分隐藏 */
  overflow: hidden;

  /* 超出的文本被裁剪 */
  text-overflow: clip;

  /* 强制文本在同一行显示 */
  white-space: nowrap;
}

.b {
  width: 100px;
  height: 20px;
  font: 8px/20px arial;
  border: 1px solid black;

  /* 超出元素框部分隐藏 */
  overflow: hidden;

  /* 超出的文本用省略号表示 */
  text-overflow: ellipsis;

  /* 强制文本在同一行显示 */
  white-space: nowrap;
}
```

```html
<div class="a">我是大聪明我是大聪明我是大聪明我是大聪明</div>
<br><br>
<div class="b">我是大聪明我是大聪明我是大聪明我是大聪明</div>
```

## 單行文本溢出省略號
要讓 `text-overflow` 生效，通常需要同時滿足三個條件：

```css
/* 1. 先強制一行內顯示文本 */
white-space: nowrap;

/* 2. 超出的部分隱藏 */
overflow: hidden;

/* 3. 文字用省略號替代超出的部分 */
text-overflow: ellipsis;
```

```css
div {
  width: 150px;
  height: 80px;
  background-color: pink;
  margin: 100px auto;

  /* 如果文字顯示不開就強制一行顯示 */
  white-space: nowrap;

  /* 溢出的部分隱藏起來 */
  overflow: hidden;

  /* 文字溢出時用省略號顯示 */
  text-overflow: ellipsis;
}
```

```html
<div>
  啥也不說，此處省略一萬字
</div>
```

## 多行文本溢出省略號
多行文本溢出省略號有較大的兼容性問題，通常適合 WebKit 瀏覽器或移動端。

```css
overflow: hidden;
text-overflow: ellipsis;
display: -webkit-box;
-webkit-line-clamp: 2;
-webkit-box-orient: vertical;
```

```css
div {
  width: 150px;
  height: 65px;
  background-color: pink;
  margin: 100px auto;
  overflow: hidden;
  text-overflow: ellipsis;

  /* 彈性伸縮盒子模型顯示 */
  display: -webkit-box;

  /* 限制在一個塊元素顯示的文本行數 */
  -webkit-line-clamp: 3;

  /* 設置或檢索伸縮盒對象的子元素排列方式 */
  -webkit-box-orient: vertical;
}
```

```html
<div>
  啥也不說，此處省略一萬字,啥也不說，此處省略一萬字此處省略一萬字
</div>
```

## 一句話總結
`text-overflow` 用來處理文本溢出顯示，單行常見省略號，多行則常用 `-webkit-line-clamp`。

## 導讀
- `text-align` 主要控制文字的水平對齊。
- 要讓文字居中，通常是設在文本所在元素的父元素上。
- 常見值是 `left`、`center`、`right`。

## 速查區

| 寫法 | 說明 |
| --- | --- |
| `text-align: left;` | 靠左對齊 |
| `text-align: center;` | 置中對齊 |
| `text-align: right;` | 靠右對齊 |

## 正文
`text-align` 屬性用於設置元素內文本內容的水平對齊方式。

如果需要讓文本水平居中，通常要把 `text-align` 設定在文本所在標籤，也就是文本的父元素上。

```css
h1 {
  /* 本質是讓 h1 盒子裡面的文字水平居中對齊 */
  text-align: center;
}
```

```html
<h1>居中對齊的標題</h1>
```

## 一句話總結
`text-align` 用來控制文字的水平對齊，最常見的用法是把父元素設成 `center`。

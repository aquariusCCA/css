# 第12章 文本屬性

## 章節說明
本章整理 CSS 的文本相關屬性，目前收錄文本顏色 `color`、文本對齊 `text-align`、文本裝飾 `text-decoration`、文本縮進 `text-indent`、行間距 `line-height`、文本換行 `white-space`、文本溢出 `text-overflow`、垂直對齊 `vertical-align` 與文字陰影 `text-shadow`。

## 回查重點
- 想確認文字顏色怎麼寫時，看 `color`
- 想快速對照常見顏色寫法時，看筆記中的速查區
- 想確認文字如何水平對齊時，看 `text-align`
- 想確認文字裝飾怎麼寫時，看 `text-decoration`
- 想確認首行縮排怎麼寫時，看 `text-indent`
- 想確認行距與單行垂直居中時，看 `line-height`
- 想確認文本如何保留空白或控制換行時，看 `white-space`
- 想確認文本溢出時的顯示方式時，看 `text-overflow`
- 想確認行內元素的垂直對齊時，看 `vertical-align`
- 想確認文字陰影怎麼寫時，看 `text-shadow`

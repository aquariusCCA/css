## 學習建議

學習本章時，不建議只背屬性和值，而是要特別注意每個屬性的「控制對象」。

例如：

- `text-align` 控制的是元素內部的文字或行內內容，不是元素本身的位置。
- `text-indent` 控制的是第一行，不是整段文字。
- `line-height` 控制的是行盒高度，不只是行與行之間的空白。
- `white-space` 同時控制空白、換行與自動換行，不只是「要不要換行」。
- `text-overflow` 只控制溢出後的顯示方式，不負責製造溢出條件。
- `vertical-align` 不是通用垂直置中工具，主要用於行內排版與表格單元格。
- `text-shadow` 是視覺效果，不會改變元素本身的盒模型大小。

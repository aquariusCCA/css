# 垂直對齊

> 返回章節首頁：[README.md](./README.md)
>
> `vertical-align` 用來設定行內元素、行內塊元素或表格單元格內容的垂直對齊方式。

## 導讀
- `vertical-align` 不能控制塊級元素本身的垂直對齊。
- 它最常見的用途，是讓圖片、表單和文字對齊。
- 另一個常見場景，是處理圖片底側多出來的空白縫隙。

## 關鍵字
- vertical-align
- baseline
- middle
- 行內元素
- 行內塊元素

## 30 秒複習入口
- `vertical-align` 主要控制行內元素與行內塊元素的垂直對齊
- 常見值有 `baseline`、`top`、`middle`、`bottom`
- 圖片和表單常用 `middle` 來和文字對齊
- 圖片底側空白縫隙，可用 `vertical-align` 或 `display: block` 解決

## 速查區

| 值 | 說明 |
| --- | --- |
| `baseline` | 基線對齊，預設值 |
| `top` | 頂部對齊 |
| `middle` | 中線對齊 |
| `bottom` | 底部對齊 |

## 正文
`vertical-align` 用於設置一個元素的垂直對齊方式，但它只對行內元素或行內塊元素有效，不能控制塊元素。

它常用於圖片、表單等行內塊元素和文字的垂直對齊。

```css
vertical-align: baseline | top | middle | bottom;
```

## 圖片、表單和文字對齊
圖片和表單都屬於行內塊元素，預設的 `vertical-align` 是基線對齊。

![圖片表單和文字對齊](./images/image_1776236389856_5216aa1d.png)

如果希望圖片或表單和文字垂直居中對齊，可以把它們的 `vertical-align` 設為 `middle`。

```css
img {
  /* 讓圖片和文字垂直居中 */
  vertical-align: middle;
}

textarea {
  vertical-align: middle;
}
```

```html
<img src="./images/img.jpg" alt=""> pink老師是劉德華

<br>
<textarea name="" id="" cols="30" rows="10"></textarea> 請您留言
```

## 圖片底側空白縫隙解決
圖片底側出現空白縫隙，通常是因為行內塊元素會和文字的基線對齊。加上邊框後，這個縫隙會更容易看出來。

![圖片底側空白縫隙解決](./images/image_1776236389856_e2db6952.png)

主要有兩種解法：

- 給圖片添加 `vertical-align: middle | top | bottom` 等。
- 把圖片改成塊級元素 `display: block;`，因為塊級元素不受 `vertical-align` 影響。

```css
div {
  border: 2px solid red;
}

img {
  /* vertical-align: middle; */
  display: block;
}
```

```html
<div>
  <img src="images/img.jpg" alt="">
</div>
```

## 一句話總結
`vertical-align` 用來調整行內元素和行內塊元素的垂直對齊，最常見的是圖片、表單與文字對齊。

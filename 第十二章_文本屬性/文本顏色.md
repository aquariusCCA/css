## 導讀
- `color` 是最常見的文本顏色屬性。
- 開發時通常會優先使用 HEX，因為最常見也最方便維護。
- 命名色與 RGB 也能用，只是使用情境不同。

## 關鍵字
- color
- 文本顏色
- 命名色
- HEX
- RGB

## 30 秒複習入口
- `color` 用來設定文字顏色
- `red` 是命名色
- `#cc00ff` 是 HEX 寫法
- `rgb(255, 0, 255)` 是 RGB 寫法

## 速查區

| 寫法 | 範例 | 說明 |
| --- | --- | --- |
| 命名色 | `color: red;` | 直接用英文顏色名稱 |
| HEX | `color: #cc00ff;` | 最常見的十六進位顏色寫法 |
| RGB | `color: rgb(255, 0, 255);` | 用紅綠藍三個數值表示 |

## 正文
`color` 屬性用於定義文本的顏色。
開發中最常用的是 HEX 寫法，但命名色與 RGB 也很常見。

```css
div {
  color: red;
  /* color: #cc00ff; */
  /* color: rgb(255, 0, 255); */
}
```
